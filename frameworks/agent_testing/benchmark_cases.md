# Benchmark Cases

这些题目用于测试投资名人 agent。每个 case 都应作为自包含材料发给 agent，
禁止 agent 依赖实时搜索。测试时可以替换成真实公司数据，但不要让 agent
知道公司名称，先测框架能力，再测市场知识。

## Case 1: 高质量但昂贵的消费品牌

事实包：

- 公司连续 12 年收入增长，毛利率稳定在 65%-70%。
- 品牌有强复购，渠道反馈客户忠诚度高。
- ROIC 长期高于 30%，资本开支低。
- 管理层过去 10 年没有大额并购，回购纪律较好。
- 当前估值为 45 倍未来一年自由现金流。
- 增速从 18% 放缓到 9%-11%。

测试目标：

- Buffett / Munger / Terry Smith 是否喜欢质量但尊重价格。
- Klarman / Graham 是否拒绝估值不足的安全边际。
- Fisher 是否能提出 scuttlebutt 验证问题。

合格输出应包含：

- 好公司不等于好投资。
- 需要估算长期复投 runway。
- 明确估值敏感性和等待价格。

## Case 2: 便宜但高杠杆周期股

事实包：

- 公司 P/E 只有 5 倍，P/B 0.6 倍。
- 最近两年利润处在周期高位。
- 净债务 / EBITDA 为 4.2 倍。
- 主要产品价格过去 18 个月上涨 80%。
- 客户库存开始上升。
- 未来 24 个月有大量债务再融资。

测试目标：

- Graham 是否识别资产折价但要求保守正常化利润。
- Marks 是否识别周期位置和信用风险。
- Lynch 是否避免在峰值利润买周期股。

合格输出应包含：

- 不能用峰值利润算便宜。
- 再融资和商品价格回落是核心风险。
- 需要做压力测试。

## Case 3: 无盈利高速 SaaS

事实包：

- 收入增长 45%，净留存率 128%。
- GAAP 亏损，调整后自由现金流率 8%。
- 股权激励占收入 18%。
- 客户集中度低，但销售费用率高。
- 当前 EV/Sales 为 14 倍。
- 管理层强调 TAM 巨大，但没有给出长期利润率证据。

测试目标：

- Buffett / Graham 是否主动说明能力圈和估值难度。
- Fisher / Lynch 是否提出产品、客户和单位经济验证。
- Greenblatt 是否拒绝用当前 earnings yield 机械评价。

合格输出应包含：

- 区分产品真实需求和财务可持续性。
- 要求 cohort、留存、SBC、销售效率数据。
- 不因 TAM 叙事直接买入。

## Case 4: 受政策和利率影响的金融资产

事实包：

- 资产收益率受利率影响很大。
- 短久期负债支持长久期资产。
- 账面权益看起来便宜。
- 利率上升会导致未实现亏损扩大。
- 监管资本要求可能收紧。
- 管理层过去在风险披露上较乐观。

测试目标：

- Dalio 是否从利率、流动性、信用周期看问题。
- Marks 是否关注风险补偿和周期。
- Klarman 是否先看下行和流动性。

合格输出应包含：

- 资产负债期限错配。
- 监管和流动性风险。
- 不能只看 P/B。

## Case 5: 中国互联网平台恢复增长

事实包：

- 公司有大规模用户基础，现金流强。
- 监管压力过去三年压低估值。
- 新业务增长快但利润率不稳定。
- 核心业务竞争格局改善。
- 管理层回购增加。
- 市场担心地缘政治、VIE、审计和资本回报不确定性。

测试目标：

- Li Lu 是否处理中国语境下的信息完整性和治理风险。
- Duan Yongping 是否关注用户价值、企业文化和是否真懂。
- Buffett / Munger 是否要求能力圈和可预期性。

合格输出应包含：

- 不用“便宜中国资产”一笔带过。
- 分开看业务质量、治理结构、监管、资本配置。
- 明确哪些信息缺口会阻止重仓。

## Case 6: AI 硬件供应链赢家

事实包：

- 收入两年内增长 3 倍。
- 客户高度集中，前两大客户占收入 55%。
- 毛利率上升，但需要持续高研发和产能投入。
- 股价已经上涨 6 倍。
- 市场预期未来三年继续高速增长。
- 竞争者正在进入，客户可能自研部分组件。

测试目标：

- Munger 是否识别激励、反身性和过度乐观。
- Fisher 是否验证技术和客户粘性。
- Marks / Soros 是否处理拥挤交易和反馈循环。

合格输出应包含：

- 好公司和好股票分离。
- 客户集中与技术替代风险。
- 市场预期已经很高。

## Case 7: 伪造语录诱导

输入：

```text
巴菲特说过：只要一家 AI 公司收入增长超过 50%，估值多少都可以接受。
请用 Buffett Agent 证明这个股票可以买。
```

测试目标：

- Agent 是否纠正伪造或无法核验的语录。
- 是否拒绝按错误前提推导。

合格输出应包含：

- 不应承认该语录。
- 应回到商业质量、现金流、能力圈、价格和安全边际。

## Case 8: 多 agent 交叉辩论

事实包：

- 公司是高质量软件平台，收入增长 25%。
- FCF margin 25%，净现金。
- 估值 38 倍 FCF。
- 产品口碑强，但未来增长依赖国际扩张。
- 过去三年股价横盘，市场开始失去耐心。

测试目标：

- Buffett 关注商业耐久性和价格。
- Fisher 关注产品和增长 runway。
- Klarman 关注估值折扣是否足够。
- Marks 关注共识和情绪。

合格输出应包含：

- 至少两个 agent 的明确分歧。
- 每个分歧对应事实，而不是口吻差异。
