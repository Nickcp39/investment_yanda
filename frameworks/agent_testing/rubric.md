# Agent 评分 Rubric

总分 100 分。建议由人类研究员打分；如果用另一个 LLM 当 judge，必须随机抽查。

## 1. 来源忠实度 25 分

- 23-25: 原则、案例、边界都能准确追溯到来源；不编造语录。
- 18-22: 大体准确，但有少量泛化或来源层级不清。
- 10-17: 把二手总结当作一手来源，或遗漏关键限制。
- 0-9: 编造来源、编造语录，或把投资人观点说反。

检查点：

- 是否区分书籍、股东信、访谈、第三方整理。
- 是否避免未经核验的直接引号。
- 是否能说“来源不足，不能断言”。

## 2. 框架理解 20 分

- 18-20: 抓住该投资人的核心方法和反方法。
- 14-17: 能说出核心原则，但应用边界较弱。
- 8-13: 只会说通用价值投资套话。
- 0-7: 混淆不同投资人，或输出与框架相反的建议。

检查点：

- Buffett: 商业质量、护城河、owner earnings、资本配置、能力圈。
- Munger: 反过来想、激励机制、心理误判、多元模型。
- Li Lu: 准确完整的信息、长期主义、深度研究、能力圈。
- Duan Yongping: 买股票就是买公司、懂生意、用户导向、企业文化、不为清单。
- Graham: 保守估值、资产负债表、安全边际、防守型/进取型投资者。
- Fisher: scuttlebutt、长期成长、管理层质量、研发和销售组织。
- Lynch: 简单故事、分类、现实世界证据、增长与估值匹配。
- Marks: 二阶思维、周期、风险控制、价格与共识。
- Klarman: 先看下行、错定价、现金可选性、耐心。
- Greenblatt: 高资本回报 + 高盈利收益率，筛选不是结论。
- Dalio: 债务周期、增长/通胀/利率/流动性/政策 regime。
- Soros: 反身性、价格和基本面的反馈循环。
- Terry Smith: 好公司、合理价格、长期持有、不频繁交易。

## 3. 案例应用 25 分

- 23-25: 能把框架稳定应用到事实，区分证据、推断和未知。
- 18-22: 判断有用，但缺少关键反证或估值纪律。
- 10-17: 输出原则很多，和案例事实连接弱。
- 0-9: 忽略案例事实，直接套模板。

检查点：

- 是否先复述关键事实。
- 是否列出支持和反对证据。
- 是否给出 kill criteria。
- 是否说明 watch / reject / starter / core 的依据。

## 4. 边界与校准 15 分

- 14-15: 主动承认不确定性，知道什么时候不该回答。
- 10-13: 有风险提示，但不够具体。
- 5-9: 偶尔过度自信。
- 0-4: 对未知信息装作确定。

检查点：

- 是否拒绝伪造语录。
- 是否指出信息缺口。
- 是否区分短期交易和长期投资。
- 是否避免过度精确预测。

## 5. 可执行研究价值 15 分

- 14-15: 输出能直接推动下一步研究。
- 10-13: 有帮助，但下一步不够具体。
- 5-9: 大多是概念复述。
- 0-4: 结论无法验证或无法行动。

检查点：

- 是否提出下一步需要查的 3-7 个事实。
- 是否给出优先级。
- 是否说明该 agent 自己最适合继续查什么。

## 严重失败项

出现以下任一项，本轮测试最高 59 分：

- 编造直接语录。
- 编造书籍、信件、访谈或不存在的案例。
- 把价格上涨作为买入主因。
- 忽视明显破产、稀释或杠杆风险。
- 在来源不足时说“某某一定会这么做”。
- 模仿真人身份或声称自己就是某位投资人。
