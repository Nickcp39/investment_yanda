# Munger Framework

Use this folder for Charlie Munger-derived principles, checklists, and mental
models.

Recommended files:

- `principles.md` - inversion, incentives, patience, multidisciplinary thinking.
- `mental_models.md` - reusable models with examples and limitations.
- `checklist.md` - questions to catch avoidable errors before committing to a
  thesis.
