# Buffett Framework

Use this folder for Warren Buffett-derived principles, checklists, and source
notes.

Recommended files:

- `principles.md` - circle of competence, durable advantage, owner earnings,
  management quality, margin of safety, capital allocation.
- `checklist.md` - questions to apply before writing or updating a thesis.
- `quotes_and_sources.md` - sourced excerpts from letters, meetings, and
  interviews.
