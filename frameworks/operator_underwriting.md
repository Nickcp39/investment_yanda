# Founder And Operator Underwriting

创建: 2026-06-15

用途: 研究创始人、实际经营者、控制人、董事会和企业文化。投资不是只买资产，也是在把资本交给一群人长期配置。

---

## 1. Core Principle

不要做未经证据支持的心理诊断。我们研究的是**可观察行为、激励结构、历史决策、控制权、危机反应、人才文化**。

---

## 2. People Map

先分清楚谁真正影响公司:

| Role | Person/entity | Power source | What to study |
|---|---|---|---|
| Founder/controller | | voting power, board, culture, product origin | values, ambition, restraint, control risk |
| CEO/operator | | formal authority, product/org control | execution, capital allocation, crisis handling |
| CFO/CIO/capital allocator | | budget, capex, buybacks, M&A | discipline, incentives, transparency |
| Board/chair | | oversight, compensation, risk | independence, courage, expertise |

## 3. Founder/Operator Checklist

### Background

- 出身和早期经历塑造了什么偏好？节俭、冒险、工程、销售、金融、学术、政治？
- 是否有 founder-market fit？
- 是否经历过真正逆境？逆境中怎么处理人、钱、产品？

### Temperament

- 是长期主义还是短期股价导向？
- 是工程/产品第一，还是资本市场第一？
- 是否愿意承认错误并改变？
- 是否过度自信、过度承诺、频繁改变叙事？

### Capital Allocation

- 回购、并购、capex、研发、裁员、SBC、分红的历史是否一致？
- 是否把股东的钱当自己的钱？
- 是否用复杂结构隐藏真实经济性？

### Culture

- 公司奖励什么行为？
- 最强人才为什么留下？
- 内部是否允许坏消息上行？
- 是否存在政治化、官僚化、短期 KPI 化？

### Control And Governance

- 创始人控制权保护长期主义，还是让外部股东无法约束错误？
- 董事会是否独立、有能力、有胆量？
- 薪酬是否和 owner earnings、ROIC、长期股东回报一致？

### Red Flags

- 过度个人崇拜。
- 反复大幅承诺但不复盘。
- 关联交易和复杂融资。
- 频繁高价并购。
- 用股权稀释掩盖经营问题。
- 核心高管连续离开。
- 董事会像橡皮图章。
- CEO 个人风险可能变成公司风险。

## 4. Evidence Sources

优先级:

1. Proxy / DEF 14A: 持股、薪酬、董事会、关联交易。
2. 10-K/20-F: risk factors、capital allocation、segment reporting。
3. Founder letters / shareholder letters / official interviews。
4. Earnings calls: 危机期回答比顺风期更重要。
5. Court/regulatory documents。
6. Employee reviews / insider comments: 只能作为 C/D 线索，需要交叉验证。
7. Biographies and media profiles: 只作背景，不作最终证据。

## 5. Operator Scorecard

| Dimension | Score | Evidence | Red flag |
|---|---:|---|---|
| Founder-market fit | 1-5 | | |
| Long-term orientation | 1-5 | | |
| Product/technical judgment | 1-5 | | |
| Capital allocation | 1-5 | | |
| Candor and error correction | 1-5 | | |
| Culture/talent magnetism | 1-5 | | |
| Governance and control risk | 1-5 | | |
| Personal/key-man risk | 1-5 | | |

Interpretation:

- 32-40: Operator can be a positive underwriting factor.
- 24-31: Neutral; rely more on business economics.
- 16-23: Needs discount / margin of safety.
- <16: Avoid unless asset value/downside protection is overwhelming.

## 6. Google People Map

As of Alphabet's 2026 materials:

- Larry Page and Sergey Brin are directors; Alphabet lists them on the board.
- Sundar Pichai is CEO of Alphabet and Google and also a director.
- Alphabet's Executive Committee consists of Larry Page as Chair, Sergey Brin, and Sundar Pichai.
- Alphabet's Class B shares carry ten votes per share; 2026 proxy materials show Page and Brin together retain very large voting power through Class B ownership.

Research implications:

- This is not a pure founder-run company like Tesla under Musk.
- It is also not a normal one-share-one-vote professional-manager company.
- It is a hybrid: founder control/culture + professional CEO execution + unusually large AI/capex decisions.

Google operator questions:

- Page/Brin: Are they still providing technical ambition and long-term control, or are they mostly passive controllers?
- Pichai: Is his temperament a strength for scaled execution and regulatory management, or a weakness in an AI platform shift that rewards speed and aggression?
- Ruth Porat / Anat Ashkenazi / finance leadership: Is AI capex being underwritten with discipline?
- Board: Does oversight truly constrain empire building and dilution?
- Culture: Does Google still attract and retain frontier AI/product talent, or is bureaucracy a real tax?
