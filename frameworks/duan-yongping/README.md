# Duan Yongping Framework

Use this folder for Duan Yongping-derived principles, checklists, and source
notes.

Recommended files:

- `principles.md` - business model, user value, differentiation, culture, and
  long-termism.
- `checklist.md` - questions to evaluate companies through this lens.
- `quotes_and_sources.md` - sourced excerpts from interviews, posts, and Q&A.
