# Layer 1: 规则与监管

## 1.1 工作场景

- 审评医疗器械 / 药品 / 临床试验申请
- 制定监管指南 / 报销规则
- 检查工厂 / 临床现场（FDA inspections）
- 反欺诈调查（OIG）
- 制定支付规则（CMS）
- 资助研究（NIH）
- 起诉违规（DOJ healthcare team）
- 州级保险监管 + 医生执照管理

## 1.2 地点集中度

| 地点 | 雇主 |
|---|---|
| Silver Spring, MD（FDA 白橡 campus） | FDA 总部 |
| Baltimore, MD（CMS 总部 + 周边） | CMS |
| Bethesda, MD（NIH 大本营） | NIH + NCI + NHLBI etc. |
| Atlanta, GA | CDC |
| Washington, DC | HHS + FTC + DOJ + 行业协会 |
| Rockville, MD | FDA 部分中心 + HRSA |
| 各州首府 | 州保险监管 + Medicaid |

**注**：联邦监管 80% 集中在 DC 周边（Maryland + Virginia）。

## 1.3 工作单位

### 联邦
- **FDA**（~18,000 员工） - 跟你最相关的是 CDRH（器械中心）
- **CMS**（~6,000 员工）
- **NIH**（~18,000 员工 + ~5,000 in 27 个研究所）
- **CDC**（~13,000 员工）
- HHS 中央 / OIG / HRSA / AHRQ / ONC（小很多）

### 州
- 50 个州各有保险委员会 + Medicaid agency + medical board

### 私营（"卖给监管"的人）
- **RA 咨询公司**：NDA Partners, Parexel, Lachman Consultants, Greenleaf Health
- **大药企/械企的 in-house RA 团队**（每家 100-500 人）
- **CRO 的 RA 部门**：IQVIA RA, ICON RA
- **法律事务所 healthcare 组**：King & Spalding, Hogan Lovells, Ropes & Gray
- **政策智库**：Avalere, Manatt Health

## 1.4 工作类型 + 数量

| 角色 | 联邦/州 数量 | 薪酬（联邦 GS） | 进入门槛 |
|---|---|---|---|
| Medical Officer (FDA) | ~2,000 | GS-14/15: $130-$200K | MD/DO + 临床经验 |
| Lead Reviewer (FDA SaMD) | 几百 | GS-14: $130-$170K | 工程/CS + RA 经验 |
| Policy Analyst (CMS) | 几千 | GS-13/14: $100-$170K | MPH/MPP/JD |
| Inspector (FDA) | ~3,000 | GS-12/13: $80-$140K | 科学背景 + 出差耐受 |
| Epidemiologist (CDC) | 几千 | GS-12-15: $100-$180K | MPH/PhD |
| Statistician (FDA/CMS/NIH) | 几千 | GS-13/14: $100-$170K | 统计 PhD |
| Researcher (NIH 内部) | ~6,000 | tenure-like: $100-$250K | PhD + tenure track |
| Healthcare Attorney (DOJ/HHS-OIG) | 几千 | GS-14/15: $130-$200K | JD |

私营 RA 岗位（更高薪）：
| 角色 | 数量（美国全市场） | 薪酬 |
|---|---|---|
| RA Specialist (junior, in-house) | ~50,000 | $90-$140K |
| RA Manager / Director | ~20,000 | $180-$350K |
| VP RA / Chief Reg Officer | ~2,000 | $400-$800K |
| Healthcare Consultant (Avalere/ZS 等) | ~10,000 | $200-$700K |
| Healthcare Attorney | ~30,000 | $250-$1M |

## 1.5 进入门槛 + 障碍

| 障碍 | 详情 |
|---|---|
| **国籍** | 多数联邦岗位要求美国公民（FDA reviewer 要求 US citizen，绿卡也不行）⚠️ **对你卡死** |
| 政审 | 安全许可 clearance 长流程 |
| 临床经验 | FDA Medical Officer 普遍要 5+ 年临床 |
| 监管经验 | 私营 RA 高薪岗一般要 5-10 年 in-house 经验 |

## 1.6 新时代 (AI) 工作机会

| 机会 | 解释 | 谁在招 |
|---|---|---|
| AI-Assisted Review Tools | FDA / CMS 自己开发 AI 工具帮审评 | FDA OPEN 项目, CMS Modernization |
| Regulatory Submission AI | 帮药企/械企自动起草 IND/NDA/510(k) | 早期初创：Norm.ai, RegASK, Saama Tech |
| RWE Analysis for FDA | 真实世界证据分析（FDA 越来越依赖） | Aetion, Flatiron Health, Verana Health |
| AI Compliance Monitoring | 持续监控生产/临床的 deviation | Verge Health, Compliance.ai |
| AI Pharmacovigilance | 自动检测不良事件信号 | Saama, IQVIA, Veeva Vault Safety |
| AI for FDA SaMD reviewer training | 帮 FDA 培训 AI/SaMD 审评员 | （新兴需求，机会窗口） |

## 1.7 你的匹配度

**★ (低)**

理由：
- 联邦岗位国籍卡死（你美籍 PhD 但不是 US citizen，应该是？需要核实 —— 如果你已入籍那门路打开很多）
- 私营 RA 需要 5-10 年行业经验，你 0 年
- 但你的 PhD + 美国博士培训 + 双语 = 走"美国 RA 咨询公司的中国/亚太业务"路径有戏

**潜在切入点**：
- Avalere China Practice / IQVIA China RA 团队
- Parexel China RA
- 美国 RA 咨询公司想拓展中国市场的方向

## 2.5 进入门槛 + 障碍

| Member Engagement AI | 给会员智能 chatbot + 服务 | Hyro, Glia, ada Health |
| Provider Network AI | 优化网络 design | Healthsparq, Quantum Health |
| Risk Adjustment AI | MA 风险调整模型（V28 改革后核心痛点） | Apixio (Edifecs), Indegene |
| Value-Based Care 分析 | ACO/MSSP 数据分析 | Aledade, Privia Health, Innovaccer |
| AI Underwriting | 自动核保 | Lemonade Health（早期）, Bind |

**这一层 AI 是真痛点驱动**：Prior auth 反人类是政治问题（医生集体抱怨），CMS 在压保险公司 standardize → AI 是出路。

## 2.7 你的匹配度

**★★★ (中)**

理由：
- AI / ML 岗位密度高，薪酬最高（UNH Optum $200-$500K 是常态）
- 不需要医疗执照
- 你的 LLM 经验 + 医学背景 = prior auth AI 这种岗位完美 fit
- 但跟"回国"冲突 → 这一层基本都是 US 本土岗

**潜在切入点**：
- UNH Optum 中国数据中心（有，在贵州）
- Centene 没有中国分部
- 大保险公司没有 China R&D = **基本没回国路径**

→ 这一层主要是"如果你改主意留美"的备选
