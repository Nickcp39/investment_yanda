# Part 1: L2 的定义与边界

## 1.1 L2 的定义

L2 = **美国医疗"付钱"的所有实体与机制**。

包括：付钱给医疗服务的政府项目（CMS 操作面）、私营保险公司、PBM 中间层、雇主自保计划，以及围绕这套付钱机制运转的所有玩家。

## 1.2 包含什么（L2 范围内）

- **政府付费项目**：Medicare（含 MA）、Medicaid、CHIP、VA、TRICARE、IHS、FEHB、ACA Marketplace
- **私营保险公司**：UNH、ELV、CVS/Aetna、CI、HUM、CNC、KP、BCBS 系统
- **PBM（Pharmacy Benefit Managers）**：CVS Caremark、Express Scripts、OptumRx 三巨头 + Prime Therapeutics + Humana Pharmacy Solutions + MedImpact 等
- **雇主自保 + ASO**：大公司 ERISA 计划 + 第三方管理（多被大保险公司做 ASO）
- **Insurtech 初创**：Devoted、Oscar、Bright（困境）、Clover、Alignment、Apree 等
- **保险经纪 + Benefit consulting**：Mercer、Aon、WTW、Lockton
- **健康储蓄账户 (HSA) 银行**：Optum Bank、HealthEquity、Fidelity HSA
- **PBM 替代方案 / 中小 PBM**：Capital Rx、Navitus、Costplus Drugs（Mark Cuban）
- **TPA（Third Party Administrators）**：HealthSCOPE Benefits、Imagine360 等

## 1.3 不包含什么（在其他 Layer）

- CMS 制定规则的部分 → **L1**（Center for Medicare 政策面 + CCIIO + CMMI）
- 医院本身（即使垂直整合医生集团 part of insurer） → **L3**
- 药店本身 → **L7**（CVS Pharmacy 零售归 L7，CVS Caremark PBM 归 L2，CVS/Aetna 保险归 L2）
- 保险监管的州 Insurance Commissioners → **L1**
- HSA bank 的银行业务 → 跨 L2/金融

## 1.4 边界争议（实际操作中模糊地带）

| 项目 | 归 L2 还是其他 | 处理 |
|---|---|---|
| **CMS** | L1 政策面 + L2 操作面 | 一家公司两层都列 |
| **CVS Health** | 跨 L2（PBM + Aetna 保险）+ L3（MinuteClinic + Oak Street）+ L7（CVS Pharmacy 零售） | 同公司多层 |
| **UnitedHealth** | 跨 L2（保险 + OptumRx PBM + Optum Bank）+ L3（Optum Health 医生集团）+ L6（Optum Insight 数据） | 同公司多层 |
| **Centene** | 主 L2（Medicaid managed care + Marketplace），但 Magellan Health (now Centene) 跨健康 IT | 主归 L2 |
| **Kaiser Permanente** | 跨 L2（保险）+ L3（医院 + 医生集团） | 主归 L2 |
| **Humana** | 跨 L2（MA 保险）+ L3（CenterWell 居家护理）+ L7（CenterWell Pharmacy 药店） | 主归 L2 |

**核心观察**：L2 的玩家越来越垂直整合，单层归属变得模糊。但本调研主轴聚焦"付钱"功能。

# Part 2: L2 的完整 entity inventory

## 2.1 政府付费项目（联邦层操作面）

```
HHS / CMS（操作面，跟 L1 政策面分离）
│
├── Medicare ($1,118B in 2024, 63M enrollees)
│   ├── Part A: 医院保险（自动覆盖 65+ 老人）
│   ├── Part B: 医生 + 门诊保险（自选 + 月费 ~$175）
│   ├── Part C: Medicare Advantage（私营代管，~50.7% 渗透率）
│   ├── Part D: 处方药（私营代管）
│   └── MACs (Medicare Administrative Contractors) - 区域承包商
│       例：Noridian, Palmetto GBA, CGS, WPS
│
├── Medicaid ($932-963B in 2024, 69M enrollees)
│   ├── 联邦 + 州 合管，州主控操作
│   ├── Managed Care: ~75% 受益人通过私营 MCO（Centene/UNH/ELV/Molina/CVS）
│   └── FFS Medicaid: 越来越少
│
├── CHIP - Children's Health Insurance Program
├── Federal Employees Health Benefits (FEHB)
│   - 由 OPM 管，覆盖 ~8M 联邦员工 + 退休 + 家属
├── ACA Marketplace ($113B in 2024)
│   - 24M Marketplace 参保人
│   - 私营保险公司在 healthcare.gov / state exchanges 卖 plans
└── 其他政府项目
    - 340B（HRSA）
    - Disproportionate Share Hospital (DSH)
```

# Part 3: L2 的工作类别全集

按工作内容分类（不按机构）。每类列：内容、要求、薪酬、规模。

## 3.1 精算（Actuarial）

### 3.1.1 Health Actuary
- **内容**：定保费、估储备金、做风险调整模型
- **要求**：精算师证书（SOA ASA → FSA, 8-10 年考试）+ 数学背景
- **薪酬**：$130-$400K（high tier 可超 $500K）
- **规模**：~30,000（健康精算师）
- **主要雇主**：UNH、ELV、CVS/Aetna、CI、HUM、Milliman（咨询）、Oliver Wyman（咨询）

### 3.1.2 Pricing Analyst（精算入门）
- **内容**：分析产品定价、做 rate filing
- **要求**：BS 数学/精算 + 1-2 年
- **薪酬**：$80-$130K
- **规模**：~10,000

## 3.2 数据科学 / AI（增长最快）⭐

### 3.2.1 Data Scientist（保险/PBM）
- **内容**：会员行为分析、claims 模式、价值医疗模型
- **要求**：MS Stats/CS
- **薪酬**：$150-$300K
- **规模**：~30,000

### 3.2.2 ML Engineer
- **内容**：训练 AI 模型用于 prior auth / 欺诈检测 / 风险调整 / Medicare Advantage Star Ratings
- **要求**：MS/PhD CS
- **薪酬**：$200-$500K（Big Tech 类比）
- **规模**：~10,000

### 3.2.3 Healthcare AI Research Scientist
- **内容**：研究 + 部署 healthcare LLM、predictive models
- **要求**：PhD CS/ML + medical experience
- **薪酬**：$300-$700K（UNH Optum、Anthem 等大公司）
- **规模**：~2,000

### 3.2.4 Risk Adjustment Analyst (MA Star Ratings)
- **内容**：分析 MA 风险调整数据，优化 V28 模型
- **要求**：MS Biostatistics
- **薪酬**：$150-$300K
- **规模**：~15,000

## 3.3 Claims 操作

### 3.3.1 Claims Processor / Adjudicator
- **内容**：审 claims，决定支付
- **要求**：高中/AS degree + 培训
- **薪酬**：$45-$75K
- **规模**：~300,000（最大单一岗）

### 3.3.2 Claims Manager
- **内容**：管理 claims team，处理升级 case
- **要求**：BS + 5+ 年
- **薪酬**：$80-$150K
- **规模**：~30,000

### 3.3.3 Medical Director (Insurance)
- **内容**：审医疗必要性、决定 prior auth、appeals
- **要求**：MD + 临床经验
- **薪酬**：$300-$500K
- **规模**：~10,000

## 3.4 Provider Network 管理

### 3.4.1 Network Contracting Manager
- **内容**：跟医院/医生集团谈合同（费率、网络分级）
- **要求**：BS + 5+ 年 healthcare
- **薪酬**：$90-$200K
- **规模**：~30,000

### 3.4.2 Network Strategy Director
- **内容**：网络设计 + 高级谈判（如 OON 处理）
- **要求**：MBA/MHA + 10+ 年
- **薪酬**：$200-$400K
- **规模**：~5,000

### 3.4.3 Value-Based Care 合同设计
- **内容**：跟 ACO/医院谈 capitation / shared savings 合同
- **要求**：MBA + VBC 经验
- **薪酬**：$200-$450K
- **规模**：~5,000

## 3.5 Pharmacy Benefit / PBM 操作

### 3.5.1 Pharmacy Director (in-house at insurer)
- **内容**：管理药品福利、formulary 决策
- **要求**：PharmD + 5+ 年
- **薪酬**：$200-$350K
- **规模**：~5,000

### 3.5.2 PBM Account Manager
- **内容**：服务大客户（雇主、保险）
- **要求**：BS + 5+ 年 PBM 经验
- **薪酬**：$120-$250K
- **规模**：~10,000

### 3.5.3 Rebate Negotiator
- **内容**：跟药企谈 rebate（PBM 的核心 monetization）
- **要求**：MBA/PharmD + Big PBM 经验
- **薪酬**：$200-$500K
- **规模**：~3,000

### 3.5.4 Formulary Pharmacist
- **内容**：决定哪些药进 formulary、tier 设置
- **要求**：PharmD
- **薪酬**：$130-$220K
- **规模**：~10,000

## 3.6 Medicare / Medicaid 业务

### 3.6.1 MA Stars Ratings Manager
- **内容**：优化 MA Star Ratings（CMS 评分，影响 bonus payment）
- **要求**：MS + healthcare operations
- **薪酬**：$150-$300K

- **规模**：~2,000

# Part 4: L2 的运行机制

## 4.1 钱怎么流（核心）

```
保费 (~$1.6T 私营) + 税款 (~$2T 政府) + OOP (~$556B)
   │
   ▼
保险公司 / 政府付费方
   │
   ├── 12-15% 留作行政 + 利润 (Medical Loss Ratio 规则要求 ≥85% 商保, ≥80% 个人/小团体)
   │
   ▼
药费 → PBM 处理（PBM 拿 rebate + spread + admin fee = ~$50-100B 利润池）
   ↓
医院 / 医生 / 药店（按 fee schedule 付）
```

**关键机制：Medical Loss Ratio (MLR) 规则**
- ACA 要求大团体保险 MLR ≥ 85%
- 个人 / 小团体 ≥ 80%
- 不达标要给会员退款
- 但 ASO（self-insured）不受 MLR 约束 → 雇主自保是绕规则的方式

## 4.2 PBM 的"暗物质"经济学

PBM 的钱从哪儿来：

```
1. Rebate（药企返点）—— 占 PBM 收入 50-70%
   药企标价 $1000 → PBM 谈到 $300 rebate → 给保险/雇主 $200，自己留 $100

2. Spread Pricing —— 占 20-30%
   PBM 跟药店谈价 $50 → 给保险公司报价 $80 → 自己留 $30

3. Admin Fees —— 占 10-20%
   按 claims 量收 admin fees

4. 自有 mail-order pharmacy 利润 —— 越来越大
   PBM 自己开邮购药店，把客户引去自家
```

**FY2025 PBM 利润池估算**：
- CVS Caremark: ~$8B
- Express Scripts: ~$7B
- OptumRx: ~$6B
- 三家合计 ~$21B

**FTC 2024 + 2025 报告攻击**：
- "PBM Rebates Inflate Drug Prices"
- "PBMs Steered Patients to Affiliated Pharmacies"
- 国会持续推动 PBM 改革立法

## 4.3 垂直整合 — L2 最大故事

```
UnitedHealth Group (UNH) - 市值 $330B
├── UnitedHealthcare（保险） — $300B 收入
├── OptumRx（PBM）— $130B 收入
├── Optum Health（医生集团 + ASC）— $100B 收入
│     └── 雇佣 ~70,000 医生（最大私营雇主）
├── Optum Insight（数据 + 分析 + Change Healthcare）
└── Optum Bank（HSA 银行）

CVS Health (CVS) - 市值 $90B
├── CVS Pharmacy（零售药店）— $120B 收入
├── CVS Caremark（PBM）— $130B 收入
├── Aetna（保险）— $100B 收入
├── MinuteClinic（诊所）
├── Oak Street Health（初级医疗，收购 2023）
└── Signify Health（家访 risk adjustment，收购 2023）

Cigna (CI) - 市值 $90B
├── Cigna Healthcare（保险）— $50B
├── Evernorth（PBM + 专业药房 Accredo）— $190B
└── 2024 把 MA 业务卖给 HCSC（专注 commercial）

Humana (HUM) - 市值 $40B
├── Insurance Segment（MA 龙头）— $107B
├── CenterWell Pharmacy
├── CenterWell Senior Primary Care
└── CenterWell Home Health（Kindred at Home）

Elevance Health (ELV) - 市值 $110B
├── Health Benefits（保险）— $160B
├── Carelon Health（服务）
├── Carelon Rx（PBM）
└── Carelon Behavioral Health
```

**垂直整合的目的**：
1. **利润转移**：保险 margin 低（3-6%），但 PBM/医生集团/数据 margin 高 → 通过内部 transfer pricing 把利润从保险转出去
2. **会员留存**：通过整合服务降低 churn
3. **数据闭环**：从保险 claims → 处方 → 临床 → 优化
4. **政治防御**：太大不能拆

## 4.4 信息怎么流

```
医疗服务发生 → claim 提交
   ↓
PBM/保险 adjudication（实时或 batch）
   ↓
fraud detection + 医疗必要性审核
   ↓
prior auth 决定
   ↓
支付 + 拒赔
   ↓
appeals process（如有）
   ↓
数据归 Optum Insight / Change Healthcare / IQVIA 等
   ↓
卖给药企（real-world evidence）
   ↓
帮药企做 Market Access 决策
```

**关键瓶颈：Prior Authorization**
- 90% 医生抱怨 PA 反人类
- 平均处理 1.3 天
- CMS 2026 强制 e-PA 标准
- 多个 startup（Cohere Health、Anterior）做 AI prior auth

## 4.5 权力怎么流

```
CMS（联邦付费方）
   │
   ├── 每年公布 MA Rate Notice（决定 MA 公司利润）
   ├── 每年公布 Medicare Physician Fee Schedule（决定医生收入）
   ├── 每年公布 Inpatient Prospective Payment System（决定医院收入）
   └── 每年公布 Part D bid（决定 PBM 经济）

州 Insurance Commissioner
   │
   └── Rate Filing 审核（保险费率不能任意涨）

雇主（ERISA preempts state law）
   │
   └── 可绕开州规则自己设计 plan（这是 ASO 流行的原因）

保险公司 + PBM
   │
   ├── 网络谈判：决定哪些医院/医生进网
   ├── Formulary：决定哪些药进
   ├── Prior Auth：决定哪些服务批
   └── Rebate：影响药企净价
```

## 4.6 旋转门（L2 版）

L1 的旋转门是 gov ↔ private RA。L2 的旋转门是：

```
CMS 政府工作（10 年）
   ↓
MA 保险公司 VP（薪酬 5-10x）
   ↓
Healthcare PE / VC 高管
   ↓
回 CMS 做 CMMI Director（公关 + 进一步专业积累）
```

**典型案例**：
- Don Berwick：Obama 时期 CMS Administrator → 私营咨询 → 回学术
- Andy Slavitt：Optum → CMS Acting Administrator → 私营 VC（Town Hall Ventures）
- Liz Fowler：Wellpoint VP → 参议院 staff → CMMI Director（写了 ACA 法案）

## 4.7 国际协调（少）

L2 几乎没有国际机制 —— 美国医疗付费高度孤立。
- 唯一例外：OECD 比较卫生支出
- WHO 全民健康覆盖讨论（美国基本不参与实际）

# Part 5: L2 的政治经济学

## 5.1 谁受益于现行制度

| 受益方 | 为什么 |
|---|---|
| 大保险 + PBM | 垂直整合 + 监管壁垒 |
| 雇主（在某种意义上） | ERISA preemption 给 flexibility |
| 大型综合医院 | 谈判力量（vs 保险） |
| 药企 | rebate 黑箱保留高标价 |
| 保险经纪 | 复杂性=人服务需求 |

## 5.2 谁吃亏

| 受益方 | 为什么 |
|---|---|
| 患者 | OOP 高 + prior auth 折磨 + 网络限制 |
| 小医院 / 独立医生 | 谈判力小，被压价 |
| 雇主（另一面） | 健康成本年涨 6-8% 持续吃利润 |
| 仿制药 + 透明定价 PBM | 主流 PBM 推回扣模式排斥 |

## 5.3 主要博弈格局

| 博弈 | 双方 | 现状 |
|---|---|---|
| **PBM 改革** | FTC + 国会 vs 三大 PBM | 2024 FTC 起诉 PBMs；2025 立法推进未通过 |
| **Medicare Advantage 监管** | CMS RADV 审计 vs UNH/HUM | UNH 被起诉 $3.7B fraud（DOJ）2025 |
| **IRA 药品议价** | CMS vs PhRMA | 第一批 10 药 2026 生效，第二批 15 药 2027 |
| **ACA 命运** | 民主党 vs 共和党 | 2024-2026 持续拉锯 |
| **Medicaid Redetermination** | 各州 vs Centene/Molina | 2023-2024 大量 disenrollment，CNC/MOH 利润压力 |
| **Site-Neutral Payment** | CMS vs 大医院 | CMS 推平价，医院反对 |
| **MA V28 风险调整改革** | CMS vs MA 保险 | 三年逐步实施，HUM 受冲击最大 |
| **No Surprises Act** | 联邦 vs OON 提供方 | 已生效，影响 ASC / 急诊收入 |

## 5.4 党派立场

| 议题 | 共和党 | 民主党 |
|---|---|---|
| Medicare 议价 | 反对 | 支持（IRA） |
| ACA | 削弱 / 替代 | 强化 + 扩张补贴 |
| Medicaid 扩张 | 各州自决 / 削减 | 全国扩张 |
| MA 报销率 | 中立偏支持 | 中立 |
| 单一付费方 / Medicare for All | 强烈反对 | 进步派支持 |
| PBM 改革 | 支持（民粹角度） | 支持 |
| 雇主 mandate | 反对 | 支持 |

|---|---|---|

# Part 6: L2 的增长与演化

## 6.1 历史扩张

| 年代 | 关键节点 | L2 规模变化 |
|---|---|---|
| 1929 | Blue Cross 第一个医院预付计划（Baylor U Hospital） | $0 → 私保起源 |
| 1934 | Blue Shield（医生服务） | |
| 1942-1945 | 二战期间工资管制 → 雇主提供医疗成为 fringe benefit | 雇主险开始 |
| 1954 | 雇主医疗福利免税（IRS Code Section 106） | 雇主险大规模化 |
| 1965 | Medicare + Medicaid 立法 | 政府付费方诞生 |
| 1974 | ERISA → 雇主自保框架 + preemption | self-insured 起 |
| 1985 | COBRA（离职后 continued coverage） | |
| 1996 | HIPAA + Mental Health Parity | 隐私 + 心理健康平等 |
| 1997 | Balanced Budget Act → Medicare+Choice（MA 前身） | MA 开始 |
| 2003 | Medicare Part D + MMA → MA 重命名 + 私营化加速 | MA 扩张 |
| 2006 | Massachusetts Health Care Reform（Romneycare） | ACA 蓝图 |
| 2010 | **ACA** | 行业大改：MLR、Marketplace、Medicaid 扩张、保险准入规则 |
| 2018 | Trump 削弱 ACA individual mandate | |
| 2018-2024 | 垂直整合大潮：CVS-Aetna、Cigna-ESI、ELV-Carelon | 三巨头形成 |
| 2022 | IRA → Medicare 第一次议价权 | 药价管制起 |
| 2023 | Medicaid Redetermination（COVID-era 续保结束） | Centene/Molina 失会员 |
| 2024 | MA V28 + RADV 审计加强 + PBM 起诉 | MA 利润压力 |

## 6.2 当前增长驱动（2024-2026）

| 驱动 | 增长率 |
|---|---|
| Medicare Advantage 渗透率上升 | +7%/年 enrollment, 渗透 50.7% → 60%+ 长期 |
| Medicaid Managed Care | +5-6%/年（除 2023-24 redetermination 短期负） |
| 雇主自保扩张 | +2%/年（小公司也开始） |
| AI 在 claims/PA 应用 | +30%/年（仍早期） |
| Value-Based Care 合同 | +15%/年 |
| 老龄化 | 持续，Medicare pool 每年增 +1.5M |
| GLP-1 报销争议 | 不确定（CMS 部分覆盖心血管适应症） |

## 6.3 未来 5-10 年趋势

| 趋势 | 影响 |
|---|---|
| **MA 渗透 → 70%+** | UNH/HUM/CVS 主导，HUM 因纯 MA exposure 压力大 |
| **PBM 强制透明化** | 部分立法可能通过，rebate 模型转向 pass-through |
| **AI prior auth 标准化** | CMS 推 e-PA，2026 起强制 |
| **Risk-based contracting 主流** | ACO REACH + MA 风险分担扩大 |
| **大保险 → "医疗服务公司"** | UNH Optum Health 已 60% 收入来自 non-insurance |
| **Insurtech 退潮** | Bright 关闭、Oscar 挣扎，垂直整合赢家通吃 |
| **跨境医疗保险 minimal** | 美国保险国际化基本不会 |
| **气候 + 健康保险** | 自然灾害健康影响开始进保险定价 |

## 6.4 颠覆风险

| 风险 | 谁被颠覆 |
|---|---|
| LLM 自动 prior auth | Prior auth 团队（~50,000 人） |
| AI claims adjudication | Claims processor（~300,000 人，最大职业） |
| 透明定价 PBM 模式扩散 | 传统 PBM rebate 经济 |
| 直接合同（雇主跟医院） | 保险中介角色 |
| 单一付费方（Medicare for All） | 全行业（低概率） |
| Mark Cuban Cost Plus Drugs 模式 | 仿制药 PBM 段 |
| Amazon Pharmacy 直接 D2C | 零售药店 + 部分 PBM |

# Part 7: 国际比较视角

## 7.1 美国 vs 单一付费方国家

| 维度 | 美国 | 加拿大 | 英国 NHS | 法国 |
|---|---|---|---|---|
| 付费方数量 | 多方（数百） | 单一（省政府） | 单一（政府税收） | 主单一 + 私补充 |
| 行政成本占比 | ~25% | ~15% | ~5% | ~10% |
| 人均支出 | $14,800 | $6,300 | $5,500 | $6,500 |
| 占 GDP | 18% | 12% | 11% | 12% |
| 健康结果（预期寿命） | 76.4 岁 | 81.8 岁 | 80.7 岁 | 82.4 岁 |
| 等待时间 | 中等 | 长 | 长 | 短 |
| 创新药可及性 | 高 + 贵 | 中等 + 公保 | 高（NICE 评估后） | 高 |

## 7.2 美国 vs 多付费方非美

| 维度 | 美国 | 德国 | 瑞士 | 荷兰 |
|---|---|---|---|---|
| 私营保险数量 | ~900+ | ~100（疾病基金） | ~60 | ~10 |
| Mandate | 现已无 | 有（社会保险） | 有 | 有 |
| Risk Adjustment | 简单 + 后调 | 复杂 + 政府管 | 复杂 + 政府管 | 极复杂 |
| 利润 cap | 无（MLR 规则） | 非营利为主 | 严格 cap | 适度 cap |

## 7.3 美国 vs 中国

| 维度 | 美国 | 中国 |
|---|---|---|
| 付费方结构 | 多方（政府 + 私营 + 雇主） | 主单一（医保基金） + 商保补充 |
| 总盘子 | $5.3T | ¥9.5T (~$1.3T) |
| 占 GDP | 18% | 7% |
| 集中度 | 多元 | 政府主导 |
| 议价机制 | PBM 黑箱 rebate | 国谈 / 集采（公开） |
| 创新药速度 | 快上市（FDA） + 贵 | 进医保 = 大降价，否则自费 |
| 商保规模 | 私保 $1.6T | 商保 ~¥900B（~$130B） |
| PBM 角色 | 三巨头 80% 市场 | 没有 PBM 物种 |

## 7.4 美国 L2 的"美国例外性"

- **唯一发达国家无全民覆盖** → 28M 没保险
- **行政成本极高** → 25% vs OECD 平均 12-15%
- **价格不透明** → 同一服务跨保险价差 5-10 倍
- **垂直整合极端** → UNH/CVS/CI 模式他国没有
- **PBM 存在** → 中介层独特发明
- **雇主主导** → 通过税收优惠绑定就业
