# Part 1: PBM 是什么 + 历史

## 1.1 一句话定义

PBM = 处方药的中间商，介于**药企（生产）→ 药店（分发）→ 保险/雇主（付款）→ 患者（消费）**之间，**做四件事**：

1. **谈药价**（跟药企谈 rebate）
2. **管 formulary**（决定哪些药能报销 + 在哪个 tier）
3. **处理 claims**（药店配药时实时审批）
4. **运营自有 mail-order + specialty pharmacy**（自己卖药 + 高利润）

## 1.2 历史演化

| 年代 | 节点 |
|---|---|
| 1960s | 第一代 PBM 诞生（PCS Health Systems 1968）= 单纯 claims processor，给保险做 IT |
| 1970-1980s | 添加 formulary 管理 + drug utilization review |
| 1990s | 开始谈 rebate（早期跟药企谈量价折扣） |
| 1992 | **Medco** 上市 → PBM 进入资本市场 |
| 1993 | Merck 收购 Medco（药企 + PBM 第一次整合，后失败拆分） |
| 2003 | **Medicare Part D 立法** → PBM 进入 Medicare，市场翻倍 |
| 2007 | Express Scripts 收购 Caremark → 大型 PBM 整合开始 |
| 2012 | Express Scripts 收购 Medco → 行业整合加速 |
| 2015 | OptumRx (UNH) + Catamaran 合并 → 三巨头雏形 |
| 2018 | **CVS 收购 Aetna**（PBM + 保险垂直整合） |
| 2018 | **Cigna 收购 Express Scripts**（保险 + PBM 垂直整合） |
| 2019 | Anthem 自建 IngenioRx（脱离 Express Scripts） |
| 2022 | **Mark Cuban Cost Plus Drugs** 创立 → 透明定价模式挑战 |
| 2024-07 | **FTC 第一份 PBM Interim Report** → 政治压力起 |
| 2025-01 | **FTC 第二份 PBM Interim Report** → 揭露 $7.3B specialty 加价 |
| **2026-02-03** | **Trump 签署 Consolidated Appropriations Act 2026 → PBM 改革立法生效**（Medicare Part D delink + 100% rebate pass-through） |

## 1.3 PBM 的产品

### 1.3.1 核心服务（卖给保险/雇主）
- Claims processing（实时 adjudication）
- Formulary management（药品清单）
- Rebate negotiation（跟药企谈返点）
- Utilization management（prior auth、step therapy、quantity limits）
- Drug Utilization Review（DUR）
- Plan design 咨询

### 1.3.2 自有 pharmacy（自己赚的）
- **Mail-order pharmacy**（邮购，主要 90-day supply）
- **Specialty pharmacy**（高价药，平均 $5K-$50K/月）
- **Retail pharmacy**（仅 CVS Caremark 有 CVS 实体店）

### 1.3.3 增值服务
- Disease management programs
- Adherence programs
- Specialty drug coordination

# Part 3: PBM 的钱怎么赚（详细经济学）

## 3.1 PBM 4 大收入来源

```
PBM 总收入 (~$571B 2024 行业)
│
├── A. Rebate (~50-70% PBM 利润来源)
│     药企标价 $1000
│       │
│       ▼ PBM 谈
│     Net price $300（PBM 拿到 $700 rebate）
│       │
│       ▼ PBM 分给客户
│     给保险/雇主：$500-600（说"我帮你省钱"）
│     PBM 自己留：$100-200
│
├── B. Spread Pricing (~10-20% 利润)
│     PBM 跟药店谈采购价 $50
│     PBM 跟保险/雇主谈服务价 $80
│     PBM 自己留差价 $30
│
├── C. Administrative Fees (~10-20%)
│     按 claims 量收 fee（通常 $0.50-$2/claim）
│     大客户合同里 PBM 主要靠 admin
│
└── D. 自有 mail-order + specialty pharmacy 利润 (~30-40%, 增长最快)
      Specialty drugs（HIV、癌症、风湿、罕见病等）
      标价 $10,000 → 药店采购成本 $2,000
      PBM 用自家 specialty pharmacy 卖 → margin 极高
```

## 3.2 Specialty Pharmacy = PBM 的真正金矿

**Specialty pharmacy 2024 数据**：
- 总市场 **$265B**（占药品总支出 ~50%！但只占处方数 2%）
- **3 PBM 旗下 specialty pharmacies 占 2/3 = ~$175B**
- CVS Specialty: $73.3B（2023）
- Accredo (Cigna): $59.5B
- Optum Specialty: ~$40B+

**为什么 Specialty 这么赚**：
- 单药价格高（$5K-$50K/月）
- 患者无选择（医生指定 + 药企指定 specialty pharmacy）
- PBM 自动 steer 到 affiliated pharmacy（FTC 2025 报告核心攻击点）
- 患者依从性强（不吃就死/恶化）
- Rebate 比例最高

**FTC 2025 第二份报告 揭露**：
- "**Big 3 PBMs marked up specialty generic drugs by THOUSANDS of percent**"
- 从 2017-2022 仅 specialty generic 一项 = **$7.3B 超额收入**
- "Big 3 PBMs reimbursed affiliated pharmacies higher than unaffiliated on nearly every drug examined"

**Specialty generic markup 实例**（FTC 报告）：
| 药 | 用途 | PBM 报销 affiliated vs 不 affiliated |
|---|---|---|
| Tadalafil（前列腺癌仿制药） | 前列腺癌 | affiliated 7,736% markup |
| Abiraterone 仿制药 | 前列腺癌 | affiliated 4,000%+ markup |
| Imatinib（白血病仿制药） | 慢性白血病 | affiliated 1,000%+ markup |

## 3.3 Rebate 经济学详解（最重要）

### Rebate 流向

```
药企的 Gross-to-Net Bubble
─────────────────────────
WAC（标价）$1,000
   │
   ├── PBM Rebate $400      → PBM/保险/雇主分
   ├── 340B 折扣 $100         → 给安全网医院
   ├── Medicaid Best Price $200 → 强制给 Medicaid
   ├── Patient Assistance $50   → 给低收入患者
   └── 其他折扣 $50
   ──────────────────────────
   Net Price $200（药企真实收到）
```

**说人话**：药企标价 $1000，药企真实收 $200，差价 $800 流给中间商。但**消费者付款仍然按 $1000 算 coinsurance**（这是 PBM 反人类的核心 — 患者付 percentage of WAC, not net）。

### Rebate vs Fee 的微妙差别

新立法（2026 PBM Reform Act）核心是把 PBM **从"rebate 抽成"转成"flat fee"**：

```
旧模式（rebate-based）：
  PBM 收入 = % of rebate
  → 激励 PBM 选高标价药（rebate 大）
  → 患者付高 coinsurance（按 WAC）
  → 药企不敢降标价（怕影响 rebate）

新模式（pass-through + flat fee）：
  PBM 收入 = $X per claim
  → 激励中性
  → 患者付按 net price
  → 药企可以正常降价
```

## 3.4 DIR Fees（仿制药商家的痛点）

DIR (Direct and Indirect Remuneration) = PBM 在配药后 retroactively 找药店要钱：
- 药店本来收到 $40
- 3 个月后 PBM 说"你 quality measure 没达标，退我 $5"
- 这成了 PBM 又一个收入来源
- 独立药店倒闭潮的主因之一
- CMS 2024 起部分限制 DIR

## 3.5 340B 套利

- 340B = HRSA 项目，强制药企给安全网医院折扣（~25-50%）
- PBM/医院/药店有时把 340B 药卖给非 eligible 患者，赚差价
- 2024-2025 药企开始拒绝向 contract pharmacy 提供 340B 药 → 法律战

# Part 4: 五方博弈（药企/PBM/保险/药店/患者）

## 4.1 利益博弈矩阵

| 角色 | 想要 | 让步给谁 | 跟谁对立 |
|---|---|---|---|
| **药企** | 高标价 + 低净价 + 进 formulary | PBM (rebate) | 仿制药 + 集采 + 医保议价 |
| **PBM** | 高 rebate + 高 spread + steer 患者 | 大客户压价 | FTC + 独立药店 + 患者 |
| **保险公司** | 低成本 + 高 MLR pass-through | PBM 暗箱接受 | 医院网络 + 药企 |
| **雇主（自保）** | 透明 + 低成本 | 保险/PBM 接受 ASO 包 | PBM 模糊性 |
| **药店** | 高 reimbursement + 低 DIR | PBM 压价 | PBM affiliated pharmacy |
| **患者** | 低 OOP + 选择权 | 所有人 | 全员（结构性弱者） |

## 4.2 信息不对称地图

```
高度透明                     不透明
─────────────────────────────────
药店采购价                   PBM 净 rebate
（药店知道）                  （仅 PBM 知道）

WAC（标价）                  PBM 给保险 spread
（公开）                     （PBM 报告，未审计）

患者 copay                   药企真实 net price
（在药店看到）               （行业秘密）
```

**患者悖论**：患者一切按 WAC（最高价）付 coinsurance，但 PBM/保险/药企内部按 net price 算账。一支 insulin 标价 $300，patient OOP 可能 $100，但药企真实收到 $50。

## 4.3 "Hospital → Pharmacy → PBM" vs "PBM → Pharmacy" 整合

**传统**：药企生产 → 分销商 → 独立药店 → 患者付钱（自付/保险），PBM 在保险公司一侧
**整合后**：药企 → CVS/OptumRx specialty → 直接配药 → 患者，PBM 自己就是药店

这个整合是 PBM 利润金矿，但也是 FTC 攻击重点。

# Part 5: 监管 + 2026 立法已通过 ⭐

## 5.1 FTC 调查时间线

| 时间 | 事件 |
|---|---|
| 2022-06 | FTC 启动 6(b) 调查（PBM 行业反垄断） |
| 2024-07 | **FTC 第一份 Interim Report** |
| 2024-09 | FTC 起诉 Big 3 PBMs（insulin rebate practices） |
| 2025-01 | **FTC 第二份 Interim Report**（聚焦 specialty generic markups） |
| 2025-11 | Mintz 法律分析：FTC PBM 调查持续推进 |

## 5.2 FTC 报告关键 finding

### 第一份报告（2024-07）
- PBM 三巨头 + 雇主联盟 + GPO 共同形成垂直整合，挤压独立药店
- Rebate-based 模式提高了 list prices
- Mail-order + specialty steering 是 self-dealing

### 第二份报告（2025-01）
- "**Big 3 PBMs marked up specialty generic drugs by THOUSANDS of percent**"
- 5 年（2017-2022）= $7.3B 超额 specialty 收入
- $1.4B spread pricing 利润（specialty 外）
- 估算这些 practices 让 employer + consumer 多付 $73B（过去十年）

## 5.3 立法演化

### 2023-2024：多个 standalone bill 都没过
- Pharmacy Benefits Manager Reform Act
- Modernizing and Ensuring PBM Accountability (MEPA) Act
- Drug Price Transparency in Medicaid Act
- 都死在 House 或 Senate 拉锯

### 2025：bipartisan momentum 起
- 2025-05：House 通过 reconciliation bill 含 PBM 措施（禁 Medicaid spread pricing + Part D 报告要求）
- 2025-07：**PBM Reform Act of 2025 (HR 4317)** 跨党派引入
- 2025-11：S.3345 PBM Price Transparency and Accountability Act 引入

### 2026-02-03 ⭐ PBM 改革立法通过！
**Consolidated Appropriations Act of 2026** Trump 签字生效。
包含以下 PBM reform 条款：

| 条款 | 详情 | 生效时间 |
|---|---|---|
| **Medicare Part D Delinking** | PBM 在 Part D 的报酬必须跟 list price 和 rebates 脱钩，只能收 flat administrative fees | 2027 plan year |
| **100% Rebate Pass-through** | PBM 必须把所有 rebates、fees、其他 remuneration 100% pass through 给 payer | 2027 plan year |
| **Semiannual Reporting to Employers** | PBM 必须给 employer plan sponsors 半年报告净药品支出、rebates、spread pricing | 立即 |
| **Steering Disclosure** | PBM 必须披露引导患者去 affiliated pharmacy 的 benefit design | 立即 |
| **CMS Monetary Penalties** | 不合规可处罚 | 立即 |
| **Reconciliation 中部分**（May 2025 House 通过）| 禁 Medicaid spread pricing + Part D 报告要求 | 视参议院进展 |

## 5.4 立法的实际影响（估算）

| 影响 | 估算 |
|---|---|
| PBM Medicare Part D rebate 利润损失 | ~$5-8B/年（三巨头合计） |
| Specialty pharmacy spread 利润损失 | ~$2-3B/年 |
| **三巨头 PBM 业务总利润下降** | ~$8-15B/年（占当前 ~$21B 的 40-70%） |
| 时间窗口 | 2026-02 立法 → 2027 plan year 生效 |

**对 CVS/CI/UNH 股价的潜在影响**：
- CVS 最受冲击（PBM + 零售药店组合 60%+ EBITDA）
- CI Evernorth 60% EBITDA 来自 PBM
- UNH OptumRx 占 Optum 业务一部分，比例小

## 5.5 其他监管动作

- **DOJ 起诉 UNH $3.7B MA fraud (2025)** — 跟 PBM 不直接相关但影响估值
- **HHS-OIG 调查 PBM 多次**
- **各州 PBM 立法**：50+ 州都通过过各种 PBM 透明化 / 报告要求 / 反 steering 法律
- **ERISA preemption 限制州法对 self-insured 计划的力度**

# Part 6: 颠覆者

## 6.1 Mark Cuban Cost Plus Drugs（最大威胁）

**模式**：成本 + 15% markup + $5 pharmacy fee + $5 shipping
**规模 2024-2025**：
- 2M+ 客户账号（13 个月达到）
- 4,000+ 药品 catalog（2022 launch → 2025 中）
- Team Cuban Card 网络：~7,000 家药店
- 2024 启动 wholesale marketplace（B2B，给医院/诊所）
- 2024 在 Dallas 开设 $11M 自有制造厂（开始产 sterile injectables）
- 2025 中：与 25+ pass-through PBM 合作
- 2025：跟 Humana 探索 D2E 项目

**对 PBM 的实际影响**：
- 价格压力：Express Scripts + CVS Caremark 2024 都被迫推出 "cost-plus style" 透明产品
- 但量还小：MCC 销售额估 ~$200-300M/年（vs 整个 retail Rx 市场 ~$400B）
- 影响最大不是 cost，是**让"PBM 模式不透明"成为大众认知**

**学术研究**：2024 PharmacoEconomics 研究估算如果 Medicare Part D 全用 MCC 90-day 价格，可省 **$8.6B**

## 6.2 透明定价 PBM（Capital Rx, Navitus, EmsanaRx）

- 模式：NADAC（National Average Drug Acquisition Cost）+ flat admin fee
- 客户增长 30-50%/年（来自 PBM 不满的雇主）
- 但市场份额仍 < 5%

## 6.3 Amazon Pharmacy + RxPass

- RxPass：$5/月会员，无限仿制药
- 2024 扩展到 Prime 会员
- 2025-08 启动 same-day delivery（多个城市）
- 直接挑战 mail-order PBM 体验

## 6.4 雇主直接合同（Direct Contracting）

- 大雇主（Walmart、Disney、Tesla 等）开始绕过 PBM 直接跟药企 / 制造商谈
- 雇主联盟（PBGH、HTA）集体议价
- 还小但增长

## 6.5 Hims & Hers + Ro 等 D2C 模式

- 直接卖仿制药给消费者
- GLP-1 compounded 版本一度爆发（2025-04 后 FDA 限制）
- 颠覆传统 PBM-pharmacy-patient 链

# Part 7: 未来 5-10 年

## 7.1 五个 plausible scenarios

### Scenario A：渐进改革（最可能，~50%）
- 2026 立法 Medicare Part D delink 实施
- 商业市场仍用 rebate（雇主可选透明）
- 三巨头利润 -30% 但仍稳定
- 透明 PBM 份额到 10-15%

### Scenario B：激进改革（~20%）
- 后续立法扩展到商业市场
- 强制 PBM 拆分（CVS/CI/UNH 必须把 PBM 卖掉）
- 三巨头估值 cut 50%+
- 中型 PBM 大爆发

### Scenario C：政治反转（~15%）
- 共和党加速去监管（Trump 2026-2028）
- PBM reform 部分回退
- 三巨头利润恢复

### Scenario D：技术颠覆（~10%）
- AI + D2C + 透明定价 + 雇主联盟 同步发生
- PBM 老模式 5 年内空心化
- Mark Cuban 模式成主流

### Scenario E：全面单一付费方（~5%）
- Medicare for All 或类似立法
- PBM 大幅简化（政府直接谈价 + 公共 formulary）
- 三巨头 PBM 业务大幅消失

## 7.2 关键观察点（监控 list）

| 时间 | 看什么 |
|---|---|
| 2026 Q2-Q3 | CMS 起草 Part D delinking 细则（Notice of Proposed Rulemaking） |
| 2027 plan year start | 立法实际生效 → 三巨头 PBM EPS 影响 |
| 2027 国会选举 | 决定下一轮 PBM 立法的政治可能性 |
| 2027-2028 | Mark Cuban 制造产能扩 + 是否 IPO |
| 2028 总统选举 | Medicare for All 议题强度 |
| 2030 | 透明 PBM 份额能否到 20% |

## 7.3 哪些岗位会消失，哪些会增长

| 岗位 | 趋势 |
|---|---|
| Rebate Negotiator (传统) | ↓ 减少（rebate 模式被监管） |
| Pass-through Pricing Specialist | ↑ 大增 |
| Spread Pricing Analyst | ↓ 模式淘汰 |
| AI Prior Auth Engineer | ↑ 持续 |
| Pharmacy Benefits Transparency Consultant | ↑ 新岗位 |
| 雇主 Direct Contracting Lead | ↑ 大增 |
| Specialty Pharmacy Clinician | ↑ 持续（药物本身没减） |
| State PBM Regulator | ↑ 增 |

# Part 8: 投资角度

## 8.1 PBM 股价影响因子

```
+ 利好
- 国会僵局（2025-12 共和党拉锯让原立法风险预期低于实际）
- Trump 自己签字（出乎多数预期）
- 透明改革反而让 PBM 简化 → 短期可控
- AI 应用降低 claims 成本

- 利空
- 2026-02 立法是真实的 ⭐ 影响 2027+
- Specialty pharmacy markup 模型受攻击
- 利润池 -30-50% 长期
- DOJ 持续起诉 UNH 等
- Mark Cuban + 雇主联盟逐渐侵蚀
```

## 8.2 三巨头股价暴露度

| 公司 | PBM 占总 EBITDA | 立法风险敞口 |
|---|---|---|
| CVS Health | ~50% (Caremark + 零售药店组合) | **高** |
| Cigna (CI) | ~60% (Evernorth = Express Scripts 主导) | **极高** |
| UnitedHealth (UNH) | ~15% (OptumRx 占 Optum ~25%) | 中等 |

**Sanity check**：CI 是最 PBM-exposed 的大保险，PBM 改革对 CI 估值压力最大。

## 8.3 估值影响（简单 DCF）

如果立法落地后 PBM 利润 -40%：
- CVS Caremark 利润 $8B → $4.8B → CVS 整体 EBITDA -10%
- Express Scripts 利润 $7B → $4.2B → CI 整体 EBITDA -15%
- OptumRx 利润 $6B → $3.6B → UNH 整体 EBITDA -3%

按 12x EBITDA 估值：
- CVS 市值压力 -10-15%
- CI 市值压力 -15-25%
- UNH 市值压力 -3-5%

## 8.4 受益方

| 受益方 | 怎么受益 |
|---|---|
| Cost Plus Drugs | 直接受益（合规市场扩大） |
| Capital Rx / Navitus / 透明 PBM | 大受益 |
| 仿制药企（Teva、Viatris） | 标价压力小 |
| 患者（个人） | 处方药 OOP 下降 5-15%（长期） |
| 雇主自保 plan | 行政成本下降 5-10% |
| 独立药店 | DIR fee + 反 steering 利好 |

## 8.5 投资策略选项

| 策略 | 操作 |
|---|---|
| Short PBM-heavy 三巨头 | Short CI 或 CVS（CI 更纯） |
| Long 受益方 | Long Cost Plus Drugs（私）、Long 仿制药 |
| Long 监管套利 | Long 透明 PBM（多数私） |
| 不参与 | 等立法细则明朗（2026 Q3 之后） |
