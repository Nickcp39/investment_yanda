# Part 1: L1 的定义与边界

## 1.1 L1 的定义

L1 = **美国医疗领域所有"制定/解释/执行/服务于规则"的实体与活动的总和**。

不仅包含监管机构本身，还包括所有围绕监管运转的生态参与者。

## 1.2 包含什么（L1 范围内）

- 联邦政府医疗监管机构
- 州政府医疗监管机构
- 联邦跨部委参与（FTC、DOJ、USPTO 等）
- 国际监管协调机构（美国参与的）
- 行业自律组织（accreditation bodies）
- 行业协会与游说组织（trade associations）
- 私营 RA 服务企业（CRO RA 部门、boutique 咨询）
- Healthcare 律所
- 政策咨询公司
- 监管科技（RegTech）+ AI 监管初创
- 监管智库
- 监管专业媒体
- 跨境监管服务（focused on US ↔ EU/China/Japan）

## 1.3 不包含什么（在其他 Layer）

- CMS 作为付费方的操作（Medicare/Medicaid 报销执行） → **L2**
- 医院执照后的运营 → **L3**
- CRO 的临床试验执行（非 RA 部分） → **L4/L5 配套**
- HIE/EHR 标准的技术实现 → **L6**
- 健康险产品本身 → **L2**
- 药品分销监管的执行（VAWD、DSCSA） → **L7**

## 1.4 边界争议（实际操作中模糊地带）

| 项目 | 归 L1 还是其他层 | 处理方式 |
|---|---|---|
| **CMS** | L1 政策面 + L2 操作面 | 同一机构两层都列，分部门归属 |
| **CRO** | L1（RA 部门）+ L4/L5（临床执行） | 按子部门拆 |
| **Veeva Vault Safety/RIM** | L1（RA SaaS）+ L6（IT 软件） | 主归 L1，但跟 L6 关联 |
| **FDA inspection 第三方** | L1 | 跟 inspection 相关都归 L1 |
| **专利律师** | L1 + 跨层 | 归 L1（USPTO 相关） |
| **医院评审（Joint Commission）** | L1 自律组织 | 归 L1 |
| **保险评级（NCQA HEDIS）** | L1 自律组织 | 归 L1 |
| **真实世界研究公司（Aetion 等）** | L1（服务 FDA）+ L6（数据基础设施） | 主归 L1 |

# Part 4: L1 的运行机制（social mechanism）

## 4.1 钱怎么流

```
税款 (~$50B/year)
   │
   ▼
联邦 FDA/CMS/NIH/CDC 等机构（运营费）
   │
   ├── FDA 监管 → 药企/械企必须合规
   │             │
   │             ▼
   │           内部 RA 团队 ($30B/year 行业成本)
   │             │
   │             ▼
   │           外部 RA 咨询 + 律所 ($30B/year)
   │             │
   │             ▼
   │           CRO RA 服务 + Veeva 等 SaaS ($15B)
   │
   ├── NIH grants → 大学/研究所 (~$40B/year 直接拨款)
   │             │
   │             ▼
   │           大学间接成本 (overhead ~30%)
   │
   └── CMS regulations → 医院/保险/医生必须合规
                       │
                       ▼
                     RCM 软件 + 法律咨询 (~$50B)
```

**关键**：政府支出 $50B 监管行业，**触发** $200-300B 的合规相关支出（"监管的杠杆效应"）。

## 4.2 用户付费 + FDA 自筹（PDUFA / MDUFA）

FDA 不全靠税收，~50% 预算靠 user fees：
- **PDUFA**（Prescription Drug User Fee Act, 1992 起）—— 药企交申报费养 FDA reviewer
- **MDUFA**（Medical Device User Fee Act, 2002 起）—— 械企交申报费
- **GDUFA**（Generic Drug User Fee Act, 2012 起）
- **BsUFA**（Biosimilar User Fee Act, 2012 起）

| Act | FY2026 申报费用（生效 2025-10-01 → 2026-09-30） |
|---|---|
| **NDA (New Drug Application) w/ clinical data** | **$4,682,003** |
| NDA w/o clinical data | $2,341,002 |
| BLA (Biologics License Application) | $4,682,003（同 NDA） |
| **510(k) clearance** | **$19,870**（比 FY2025 涨 55.9% ⚠️） |
| **De Novo (新分类器械)** | **$132,464** |
| **PMA (Premarket Approval, 高风险器械)** | **~$441,556**（510(k) = 4.5% of PMA 反推） |
| ANDA (仿制药 FY2026) | ~$280,000 估 |
| Annual establishment registration (设备) | $6,493 |
| FY2026 PDUFA 总目标 program fee | $1,244,830,400 |

**含义**：
- 药企每年付 FDA $1.2B+ 仅 PDUFA 一项；加上 MDUFA/GDUFA/BsUFA 合计 $3B+ user fees
- **FY2025 user fees 占 PDUFA 项目总成本 77%**，占 FDA 总预算 ~49%
- 这强化了 FDA "服务于行业" 的属性

## 4.3 信息怎么流

```
科学研究（学术 + NIH）
   │
   ▼
FDA Guidance Documents（指导原则，~50+ /年新发）
   │
   ▼
行业（药企 + 械企）按 guidance 设计申报
   │
   ▼
申报 → FDA review → 批准/不批准 → 公开 docket
   │
   ▼
RA 咨询行业积累知识 → 卖给下一轮申报方
   │
   ▼
专业媒体（Pink Sheet/Inside Health Policy/RAPS Focus）报道
   │
   ▼
形成行业共识 → 影响下一轮 guidance（反馈循环）
```

**关键**：FDA Guidance 是 L1 信息流的核心枢纽。一份新 guidance 可能让 50 家公司年度 RA 成本变 10 倍。

## 4.4 权力怎么流

```
国会 (legislative)
   │
   ├── 立法（FDA Modernization Act、IRA、Cures Act 等）
   │     │
   │     ▼
   │   总统签字 / 否决
   │
   └── 拨款（决定 FDA/CMS/NIH 预算）

行政（HHS + FDA + CMS + NIH）
   │
   ├── 制定 regulations（具体规则）
   ├── 执行 enforcement
   └── 发 guidance documents

司法
   │
   ├── 法院审查 regulations 合法性（Chevron 推翻后愈发活跃）
   ├── 受理 False Claims Act 案件
   └── 受理 patent 案件
```

## 4.5 旋转门（Revolving Door）⭐

L1 的核心人才流动机制：

```
联邦工作（FDA reviewer/CMS analyst）
   │
   ▼ 工作 5-15 年后
   │
   ├── 跳到 私营 RA 咨询（薪酬翻 2-3 倍）
   ├── 跳到 大律所 healthcare practice（薪酬翻 3-5 倍）
   ├── 跳到 in-house RA（药企/械企 VP RA）
   ├── 跳到 智库 senior fellow
   └── 创办 boutique 咨询公司
```

**反向**：私营回联邦
- Trump/Biden 任命的 FDA Commissioner 多半 ex-private
- Acting Commissioner 常从内部升

**Cooling-off period**：
- 联邦员工离职后 1-2 年不能直接接触原部门（lobbying restrictions）
- 但 "consult on" 老规则的客户合法

**社会含义**：
- L1 私营薪酬支撑（boutique 老板能赚 $1M+ 因为他懂 FDA 内部）
- L1 知识传递主要靠人 不是文档
- L1 创新主要来自 ex-FDA 创办的公司

## 4.6 ICH / IMDRF 国际机制

美国 FDA 不是孤立的：
- ICH 制定全球药品监管协调标准（FDA + EMA + PMDA + Health Canada + Swissmedic + MHLW + NMPA[观察员] 等）
- IMDRF 同上但器械（FDA CDRH + EU EMA + 中国 NMPA + 日本 PMDA + 加拿大 + 巴西 + 澳大利亚 + WHO）

**含义**：FDA 的 guidance 越来越要兼容 ICH，倒逼 L1 业内人士 必须 understand 多国监管。

# Part 5: L1 的政治经济学

## 5.1 谁受益于"强监管"

| 受益方 | 为什么 |
|---|---|
| 大药企 | 高门槛挡小竞争者 |
| Big Law / 大 RA 咨询 | 合规需求 |
| 已上市的医疗器械公司 | 新进入者难 |
| 内部 RA 部门 | 工作有保障 |
| 病人 (理论上) | 安全保护 |
| FDA 自己 (人力扩张) | 增预算 |

## 5.2 谁受益于"弱监管"

| 受益方 | 为什么 |
|---|---|
| 初创公司 | 上市快 |
| 仿制药企 | 进入门槛低 |
| 自由市场派 think tanks (AEI/Heritage) | 意识形态 |
| 保险公司 (有时) | 价格压力小 |
| 病人 (一些情况) | 创新药快、可负担 |

## 5.3 主要博弈格局

| 博弈 | 双方 | 现状 |
|---|---|---|
| **PBM 改革** | FTC + 国会 vs UNH/CVS/CI | 2024-2026 持续推动，未通过 |
| **药品价格透明** | CMS + 国会 vs PhRMA | IRA 第一次实现部分议价 |
| **AI 医疗器械审评速度** | 行业 vs FDA CDRH | FDA 2024 推 predetermined change control |
| **互联网医疗跨州执照** | telehealth 公司 vs 州医疗委员会 | 部分州互认 |
| **生物相似药入市障碍** | 仿制药 vs 大药企 | 大药企持续 patent thickets |
| **临床试验多元化** | FDA + advocacy vs 行业 | 2022 FDA Modernization Act 明确要求 |

## 5.4 监管捕获（Regulatory Capture）现象

**FDA 是被业界部分 "捕获" 的**：
- ~50% 预算靠业界 user fees → FDA 有动力维持业界关系
- 旋转门让 FDA 主管 ↔ 业界高管 频繁互转
- Advisory Committees 成员 60%+ 有 industry ties

**但也有制衡**：
- DOJ/OIG 反欺诈持续起诉
- ProPublica/STAT News 等记者监督
- Public Citizen 等 NGO advocacy
- 国会 oversight hearings

## 5.5 党派影响

| 议题 | 共和党立场 | 民主党立场 |
|---|---|---|
| Medicare 议价 | 反对 | 支持（IRA） |
| ACA | 削弱 | 强化 |
| Medicaid 扩张 | 各州自决 / 削减 | 全国扩张 |
| FDA 加速审批 | 支持（21st Century Cures） | 支持但谨慎 |
| 药价控制 | 部分支持（民粹角度） | 强烈支持 |
| 处方药 importation | 部分支持 | 部分支持 |

**结论**：药品价格 + Medicare 是两党都关注的议题，但 ACA / Medicaid 是党派分歧大的。

# Part 7: 国际比较视角

## 7.1 美国 FDA vs 欧盟 EMA

| 维度 | FDA | EMA |
|---|---|---|
| 治理 | 联邦独立机构 | EU 委员会下属 |
| 审评模式 | 中心化（FDA 总部） | 分散（EU 27 国 NCA） |
| 速度 | 创新药 6-10 月 priority | 创新药 ~14 月 |
| User fee | 高（PDUFA $4M/NDA） | 低（EMA $250K/MAA） |
| 透明度 | Docket 公开 | EPAR 公开但简略 |

## 7.2 美国 FDA vs 中国 NMPA

| 维度 | FDA | NMPA |
|---|---|---|
| 治理 | 独立机构 | 国家市场监管总局下属 |
| 审评速度 | 6-12 月 | 6-18 月（近年加速） |
| AI 医疗器械框架 | 成熟，1,000+ 批准 | 2022 起完善，300+ 批准 |
| 跨境互认 | 部分（IMDRF） | 部分（部分港澳互认） |
| 临床数据要求 | 严格 | 部分接受境外数据 |

## 7.3 美国 vs 日本 PMDA

| 维度 | FDA | PMDA |
|---|---|---|
| 治理 | 独立 | 厚生劳动省下属 |
| Sakigake 制度 | 类似 Breakthrough | 优先审评 |
| 临床要求 | 全球数据 | 部分要求日本数据（bridging study） |

## 7.4 多国申报协调

```
全球申报 sponsor
   │
   ├── 美国 FDA → 6-10 月
   ├── 欧盟 EMA → 14 月
   ├── 中国 NMPA → 6-18 月
   ├── 日本 PMDA → 12 月
   └── 其他

ICH 协调标准（M4 CTD 格式）使一份核心 dossier 可用多国
但每国都要本地化适配（local agent、language、临床差异）
```

**含义**：精通多国 RA 的人 = 全球稀缺资产。中国 BD 出海大潮（百济、信达、康方）创造了 US ↔ China 双语 RA 人才的窗口。

## 重要新发现
