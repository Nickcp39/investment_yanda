# Part 1: 行业结构（从大到小）

## 1.1 宏观盘子（多大）

| 指标 | 数据（2024 估） | 备注 |
|---|---|---|
| 美国医疗总支出 NHE | **~$5.0 万亿 / 年** | CMS NHE 数据，2023 是 $4.9T |
| 占 GDP 比例 | **~17.8%** | 全球最高 |
| 人均医疗支出 | **~$14,800** | OECD 平均 ~$5,000 |
| 年增长率 | **~5-6% 名义 / 2-3% 实际** | 长期超 GDP 增速 |
| 全球医疗市场份额 | **~42%** | 美国 + 中国 ~55% |
| 雇佣人数 | **~2200 万人**（占美国就业 ~14%） | 单一最大就业行业 |

**对照参考**：
- 全美 GDP ~$28T
- 美国国防开支 ~$900B（医疗是国防的 5 倍多）
- 中国医疗总支出 ~¥9T（折美元 ~$1.25T，是美国 1/4）

**为什么这么大**：
- 老龄化（65+ 人口 ~6000 万，到 2030 → 7400 万）
- 慢病慢性化（糖尿病 + 心血管 + 肥胖 = 60%+ 总支出驱动）
- 技术升级（昂贵的新疗法：CAR-T、基因疗法、GLP-1）
- 行政成本（占总支出 ~25%，全球最高）

## 1.2 钱从哪儿来（付费方层级）

### 大盘切分

```
$5.0T 总支出
├── 政府付费 ~52%（$2.6T）
│   ├── Medicare（联邦，65+/残障）       ~$1.0T   21%
│   ├── Medicaid（联邦+州，低收入）       ~$0.9T   18%
│   ├── 公共卫生项目（VA/TRICARE/IHS/CHIP/FEHB） ~$0.5T 11%
│   └── 政府公共健康活动                 ~$0.2T   2%
│
├── 私营保险 ~30%（$1.5T）
│   ├── 雇主自保 (ERISA)                ~$0.9T   18%
│   ├── 个人 / 团体保险                  ~$0.5T   10%
│   └── 高端补充险                      ~$0.1T   2%
│
├── 个人自费 OOP ~10%（$0.5T）
│   └── Copay / Deductible / 自费项目
│
└── 其他 ~8%（$0.4T）
    └── 行政成本 + 医疗研究投资 + 结构投资
```

### 关键玩家（按规模）

| 付费方 | 体量 | 增长 | 玩家 |
|---|---|---|---|
| Medicare | ~$1.0T | 7-9%/年 (老龄化驱动) | CMS 直管 + Medicare Advantage 私营代管（UNH/HUM/ELV/CVS） |
| Medicaid | ~$0.9T | 4-6%/年 | 各州 + Managed Medicaid（CNC/ELV/CVS/UNH） |
| 商业保险 | ~$1.5T | 3-5%/年 | UNH/ELV/CVS/CI/HUM/KP/BCBS |
| Out-of-pocket | ~$0.5T | 持平 | 个人 |

**Medicare Advantage 趋势** ⭐：传统 Medicare → MA 的迁移是 2018-2026 最大故事。MA 覆盖率从 33% → 54%（2024），UNH/HUM/ELV 因此爆发。

## 1.3 钱花在哪儿（消费侧）

```
$5.0T 总支出按类别拆分
├── 医院服务                  $1.5T   30%   ⭐ 最大单一类别
├── 医生 + 临床服务            $1.0T   20%
├── 处方药                    $0.45T  9%
├── 长期护理（养老院+居家）     $0.35T  7%
├── 牙科                      $0.18T  4%
├── 医疗设备 + 耐用品          $0.25T  5%
├── 政府行政 + 公共卫生        $0.35T  7%
├── 健康研究投资               $0.05T  1%
├── 结构投资                  $0.15T  3%
└── 其他个人医疗               $0.20T  4%
```

### 关键观察

**钱最多的地方 ≠ 利润最厚的地方**：

| 类别 | 总盘子 | 净利率 | 净利润池估算 |
|---|---|---|---|
| 医院 | $1.5T | 3-7% | ~$60-100B |
| 医生 / 临床 | $1.0T | 5-10% | ~$60-100B |
| 处方药 | $0.45T | **18-25%** | **~$90-110B** ⭐ |
| 保险 (商保) | $1.5T | 3-6% | ~$60-80B |
| 医疗器械 | $0.25T | **15-25%** | **~$50-60B** ⭐ |
| 健康 IT | ~$0.1T | 15-30% | ~$20-25B |
| AI 医疗软件 | ~$0.02T | 高但盈亏未明 | 待定 |

**结论**：医院体量大但利薄；药企/械企体量小但利厚；AI 软件体量最小但潜在 margin 最高。

### 隐藏的"中间商税"

- **PBM rebate 池**：~$200B/年的药企返点流过 PBM，三巨头（CVS Caremark / Express Scripts / OptumRx）captures ~$50B+
- **保险行政成本**：占保费 ~15-20%（单一付费制 ~5%）
- **医院收账成本**：医院花 ~$200B 在收账上（RCM 行业 = 这个的卖铲子）
- **医生执业管理**：医生平均 30-40% 时间花在文书 → AI scribe 的市场基础

# Part 2: 机会分析

## 2.1 从钱出发（follow the money）

### 2.1.1 哪里钱最多（absolute dollars）

按"利润池"而非"收入"看，Top 5：
1. **大药企净利润池**：~$100B/年（PFE/LLY/MRK/JNJ/ABBV 五家加起来）
2. **PBM + 综合付费方利润池**：~$70B/年（UNH/CVS/CI 三家）
3. **医疗器械净利润池**：~$50B/年（ABT/MDT/SYK/ISRG 等）
4. **医院净利润池**：~$60-80B/年（公立非营利 + 营利合计）
5. **健康 IT + 数据**：~$25B/年（Epic + Veeva + Optum Insight 等）

**对你的启示**：钱最多的地方（药企、PBM）招高薪 AI/数据人才最多，但门槛也最高（FDA 监管 + 行业经验）。

### 2.1.2 哪里 margin 最高（盈利能力）

```
按 gross margin（高→低）：
1. SaaS 健康 IT（Veeva）              80-85%
2. AI 医疗软件                       70-90% (理论)
3. 大药企（创新药）                   75-85%
4. 医疗器械（高值耗材）                65-75%
5. CDMO / CRO                       30-40%
6. PBM                              5-8% 但绝对值高
7. 医院                             变动，3-7% net
8. 保险                             3-6% net
```

**对你的启示**：margin 最高的位置都是"软件 + 数据"，跟你的技能 100% 匹配。

### 2.1.3 哪里增速最快（5 年 CAGR）

| 子段 | 5Y CAGR | 驱动 |
|---|---|---|
| **AI 医疗软件** | **30-50%** | 0 到 1 阶段 |
| GLP-1 / 减重药 | 40-50% | LLY/NVO 爆炸 |
| Medicare Advantage | 8-10% | 老龄化 + 政策 |
| AI 药物发现 | 25-30% | 大药企砸钱 |
| 居家健康 | 10-15% | 医院外迁 |
| 行为健康 (Mental Health) | 15-20% | 后疫情需求 |
| 真实世界数据 | 20-25% | 监管要求 |
| Telehealth | 8-12% | 后疫情稳定 |

**对你的启示**：你的核心赛道（AI 医疗软件）就是增速第一。

### 2.1.4 中间商被砍 = 新增量

监管 + 市场都在攻击中间商，被砍出来的钱会流向新的位置：

| 旧中间商 | 攻击者 | 新流向（你能切的位置） |
|---|---|---|
| PBM rebate 黑箱 | FTC / IRA Medicare 议价 | 直接面向药企的"透明定价"工具公司 |
