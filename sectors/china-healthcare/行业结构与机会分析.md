# Part 1: 行业结构（从大到小）

## 1.1 宏观盘子（多大）

> data_as_of: 2024 年度数据，发布于 2025-12-01
> 主源：国家卫健委《2024 年我国卫生健康事业发展统计公报》

### 总盘

| 指标 | 数据（2024） | 备注 |
|---|---|---|
| 中国卫生总费用 NHE | **¥9.09 万亿 / 年**（90,895.5 亿元）[初步核算] | NHC 2024 公报 |
| 占 GDP 比例 | **6.7%** | NHC 数据；WB 口径略高 ~7.2%（2023） |
| 人均卫生总费用 | **¥6,454.4**（≈ $900 USD） | NHC 2024 |
| 年增长率 | **~7-9% 名义** | 2023 → 2024：9.06 → 9.09 万亿（口径略调；趋势线 7-8%）|
| 折美元（@7.2 RMB/USD）| **~$1.26 万亿** | 约美国 NHE 的 **1/4** |
| 占全球医疗市场份额 | **~10-12%** | 美国 ~42%；中美合计 ~55% |
| 卫生人员总数 | **1,578 万人**（占全国就业 ~2.1%） | NHC 2024。卫生技术人员 1,302 万 |

### NHE 三大支出口径拆分（2024）

```
¥9.09 万亿
├── 政府卫生支出           ¥2.26 万亿   24.9%   财政拨款 + 公卫
├── 社会卫生支出           ¥4.33 万亿   47.6%   ⭐ 最大块（基本医保 + 商保 + 单位福利）
└── 个人卫生支出 OOP        ¥2.50 万亿   27.5%   自付（已从 2010s 的 ~36% 降到 27.5%）
```

**关键观察**：
- "社会卫生支出"占比最大，反映**基本医保已是中国医疗筹资的主轴**（区别于美国"政府 52% + 商保 30%"格局）
- OOP 27.5% 仍远高于发达国家（OECD ~20%；US ~10%），是医改长期工作目标
- 增量主要来自医保扩面 + 老龄化驱动的服务需求

### 跟美国对比（一张表）

| 指标 | 中国 2024 | 美国 2024 | 倍数 |
|---|---|---|---|
| NHE 总额 | $1.26T | $5.0T | 美国是中国 **4x** |
| 占 GDP | 6.7% | 17.8% | 美国 **2.7x** |
| 人均 | $900 | $14,800 | 美国 **16x** |
| GDP 总量 | ~$18T | ~$28T | 美国 1.6x |
| 雇佣人数 | 1,578 万 | ~2,200 万 | 美国 1.4x |
| 老龄化 65+ | 15.6%（2.2 亿） | ~17.5%（6,000 万） | 中国 65+ 是美国 **3.7x 体量** |

**两条 takeaway**：
1. **绝对值差距巨大，相对增速中国领先**（中国 7-9% vs 美国 5-6%）。但 GDP 占比看，中国仍有充分增长空间（OECD 平均 ~10%，美国 17.8% 是天花板异类）
2. **中国 65+ 老人是美国的 3.7 倍**，但 NHE 只是美国 1/4 → 单老人医疗资源严重稀缺 → 长期需求侧 β

### 为什么这么大 / 还会更大

- **老龄化加速**：60+ 已 3.1 亿（22.0%），65+ 2.2 亿（15.6%）；按 NDRC 测算 2050 → 60+ 占 38%
- **慢病基数巨大**：60+ 居民中 75.8% 患 ≥1 种慢病；高血压 58.3% / 糖尿病 19.4% / 高胆固醇 10.5%
- **医保覆盖完成 → 升级**：基本医保参保率 ~95%（13.3 亿人），但人均报销额仍低，是下一阶段抬升点
- **技术升级**：创新药谈判进医保 + 高值耗材集采 → 单位价格下降但用量上升
- **行政成本**远低于美国（美国 ~25% NHE 是行政；中国 <10%）→ 增长空间不在行政

### 体量分布（粗略，待 1.5 节细化）

- **医院**：3.87 万家（公立 ~1.2 万 + 民营 ~2.7 万），占 NHE 的 ~50%
- **基层医疗**：97 万家（村卫生室 + 社区中心 + 乡镇卫生院）
- **专业公卫**（CDC / 妇幼 / 急救 / 血站）：~1.2 万家
- **每千人执业医师 3.61，注册护士 4.16**（美国对比：医师 2.6，护士 11.7 — 中国医生密度高、护士密度低，是结构特征）

### Block 1.1 待办 / 不确定

- ⚠️ "增长率 7-9% 名义"是从公报趋势推断，未拿到官方 CAGR 表，1.2 节核实
- ⚠️ "全球市场份额 ~10-12%" 是按 NHE/USD 估算，需要 WHO Global Health Expenditure Database 交叉验证
- ⚠️ 折美元用 7.2 RMB/USD 即期汇率，PPP 口径会更高（PPP 下中国 NHE 约 $2.5T）
- ⚠️ 民营医院 2.7 万家是 NHC 公报推断，未明确分公私拆分，1.4 节核实

### 触发重读

- 2025 公报发布（预期 2026-12）→ 全部数字 refresh
- 卫生总费用占 GDP 突破 7.5%（结构性扩张信号）
- 商业健康险占 NHE 超过 5%（目前 ~2-3%）→ 付费方格局改变
- 医保基金出现年度赤字 → 整个支付侧 thesis 需重审
