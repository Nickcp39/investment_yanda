## 3.2 派生计算 — 真正的"未来计算"

### DTP CAGR 矛盾分析（`L3-DTP-CAGR-reconciliation.csv`）

```
2019 = 3,936 家 → 2023 = 7,132 家 → 实际 CAGR = 17%
2023 = 750 亿 → 2024 = 872 亿 → YoY = 16.3%
─────
行业宣称：2028 处方外流 7,200 亿（DTP 承接部分）
─────
2024 → 2028 隐含 CAGR = (7200/872)^(1/4) - 1 = 69.4%
─────
现实 base case（维持 17-18% 历史 CAGR）：2028 = 1,690 亿
现实 high case（30% CAGR）：2028 = 2,491 亿
极激进 case（50% CAGR）：2028 = 4,414 亿
```

**结论**：行业宣称 2028 = 7,200 亿是处方外流"总量"，不是 DTP 单一承接量。**真正 DTP 市场 2028 ≈ 1,700-2,500 亿（base-high）**，不是 5,000 亿。我之前 v1 的"5,000 亿"是误读。
