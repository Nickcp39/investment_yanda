# Layer 1: 监管 / 政策

> **本层一句话**：把"什么能卖 / 卖多少钱 / 钱怎么报 / 谁能开业 / 怎么检查"这些规则全部写出来并执行。中国 L1 的特点是**权力高度集中 + 编制制度二元（编内/编外天差地别）+ 地理高度集中在北京**。

## 1.0 L1 生态总览（先搞懂这几件事，后面才看得懂岗位）

### A. L1 由 6 大块组成（按权力大小）

```
国务院
│
├─ ① 国家卫生健康委 NHC（正部级）
│   主管：医院体系 + 公共卫生 + 医生执业 + 中医
│   下辖：国家中医药管理局 / 国家疾控局
│   特点：传统医疗系统的"大管家"
│
├─ ② 国家医保局 NHSA（副部级，2018 新设，独立于卫健委）⭐ 最有钱
│   主管：基本医保基金（~3 万亿）+ 集采 + 国谈 + DRG/DIP 改革 + 反欺诈
│   下辖：医保中心 / 医药价格和招标采购指导中心 / 大数据中心
│   特点：成立 8 年，编制还在扩张；技术岗位（数据/AI）新增最多
│
├─ ③ 国家药监局 NMPA（副部级，归市场监管总局）
│   主管：药品 + 器械 + 化妆品 准入审批
│   下辖：CDE（药审中心）/ CMDE（器审中心）/ CFDI（核查中心）/ NIFDC（中检院）
│   分中心：长三角（上海）/ 大湾区（深圳）/ 京津冀 / 华中 / 西南 — 5 个
│   特点：技术性最强；海归 PhD 最对口的国家局
│
├─ ④ 国家发改委 医药价格司
│   主管：产业政策 + 价格指导
│
├─ ⑤ 工信部 医药司
│   主管：医药制造业 + 应急储备
│
└─ ⑥ 各省 / 市 / 县三套并行
    省卫健委 + 省医保局 + 省药监局 各自延伸到地方
    总数：31 个省级 × 3 套 ≈ 90+ 省级单位
```

### B. **编制系统**：中国 L1 最重要的潜规则（决定薪酬 / 户口 / 退休 / 跳槽自由度）

权力 + 待遇排序：**行政编（公务员） > 参公事业编 > 全额事业编 > 聘用制 > 备案制 > 人事代理 > 合同制 = 劳务派遣**

| 编制类型 | 哪里有 | 薪酬 vs 编内 | 福利 | 跳槽 |
|---|---|---|---|---|
| **行政编**（公务员） | NHC / NHSA / NMPA / 发改委司局级 | 100% | 全套 + 解决户口 + 退休 | 调动可（系统内） |
| **参公事业编** | CDE / 部分研究中心 | ~95% | 几乎同公务员 | 调动可 |
| **全额事业编** | CMDE / 大多数直属事业单位 | ~80-90% | 五险一金 + 部分福利 | 调动较难 |
| **聘用制** | CDE 大部分新招 / 各局边缘岗 | ~70-80% | 五险一金 | 自由 |
| **合同制** | CMDE 新招主流 | ~50-70% | 五险一金 | 自由 |
| **劳务派遣** | 各局临时项目 | ~30-50% | 基础保险 | 完全自由 |

⚠️ 业界反映 CMDE 合同制"**前 3 年简单工作为主；应届硕士和工作 10 年的博士工作内容和待遇基本一样**"——意味着合同制天花板很低。

### C. 钱从哪儿来

- L1 整体：**100% 中央 / 地方财政拨款**，自己不创收
- NHSA 例外：管 ~3 万亿医保基金，但基金不计入 NHSA 自身经费
- 私营生态（咨询 / 律所 / 智库）：钱来自药企 / 械企 / VC

### D. L1 内部三个 cross-cut 维度（每层都有）

| 维度 | L1 切分 |
|---|---|
| **公 vs 私** | 体制内（部委 + 直属单位 + 智库）vs 体制外（CRO 注册部 + RA 咨询 + 律所 + 商业智库） |
| **央 vs 地** | 中央（北京 + 5 个分中心）vs 地方（31 省级 + 数百市县） |
| **决策 vs 执行** | 决策（司局长 + 政策研究员）vs 执行（审评员 + 检查员 + 招采员） |

## 1.1 工作场景（具体在做什么）

按 L1 内部分工分 7 类：

| 场景 | 谁做 | 做什么 |
|---|---|---|
| **药/械审评** | CDE / CMDE 审评员 | 看药企/械企交的注册申请，决定通过/补充资料/驳回 |
| **GMP/GCP 检查** | CFDI / NMPA 检查员 | 飞工厂/临床现场，查质量体系 |
| **国谈 / 集采** | NHSA 招采司 + 各省医保局 | 跟药企砍价；组织带量采购 |
| **DRG/DIP 改革** | NHSA 医保中心 + 大数据中心 | 给医院制定病组打包付费规则 + 监控 |
| **反欺诈监测** | NHSA 基金监管司 + 大数据中心 | AI 算法识别异常报销 |
| **政策研究** | NHC/NHSA/NMPA 研究中心 + 智库 | 起草政策草案 + 评估实施效果 |
| **执法 / 行政处罚** | 各省局执法队 | 处罚违规药店/医院/广告 |

**最新热点（决定哪些岗位扩张）**：
- DRG/DIP 2.0 版分组方案 2024-07 发布 → 各省医保局数据团队扩招
- CDE 2024-2025 持续扩聘用制 → 药品审评积压
- AI 辅助审评试点（CDE 智能审评系统）→ 算法岗位新增
- 创新药械"快速通道"扩容 → 审评员需求增

## 1.2 工作地点（中国 L1 比 US 更集中）

| 城市 | 占比 | 主要雇主 |
|---|---|---|
| **北京** | **~60%** | NHC / NHSA / NMPA 总部 + CDE + CMDE + CFDI + NIFDC + 发改委 + 工信部 + 中央研究院 |
| 上海 | ~10% | NMPA 长三角分中心 + 上海市三局 + 部分央企研究院 |
| 深圳 | ~5% | NMPA 大湾区分中心 + 深圳市三局 |
| 武汉 / 成都 / 沈阳 | ~3% 各 | NMPA 华中/西南/东北分中心 |
| 31 省会 | ~20% | 各省卫健委 + 医保局 + 药监局 |
| 其他市县 | ~2% | 市县级三局（人数多但岗位枯燥） |

**关键观察**：跟 US 联邦监管 80% 集中在 DC 周边类似——中国 L1 也是**北京一城独大**。想离家近又做 L1，要么北京、要么大湾区/长三角分中心。

## 1.5 进入门槛（描述事实，不评判匹配）

### 编制内（最难）
- **国考**：35 岁卡死；要求党员（多数岗位）；名校优先；政审通过；学历对口；海归 PhD 在 2 年内可算"应届"延伸
- **应届事业编**：35 岁卡死（博士），30 岁卡死（硕士）；接受海归
- **京户**：CDE 编内要求北京户口或解决户口；CMDE 合同制不解决
- **党员**：核心岗多要求党员或群众身份审查通过
- **政审**：海归常通过；少数涉密 / 涉外岗位受限

### 编制外（中等难）
- **聘用制**：年龄相对宽松（35-40 岁可）；学历要求保留
- **合同制**：年龄最宽松；天花板低（业内反映"博士和硕士待遇没差")

### 体制外（最开放）
- CRO / RA 咨询：年龄 / 户口 / 党员均不限制
- 律所：JD/JM + 行业关系
- 智库：海归 PhD + 论文产出
- MNC RA：英语 + 学历 + GMP/GCP 经验
- in-house RA：5+ 年经验 + 产品成功

### 反向"曲线进 L1"通道
- 从 L4/L5 私营做出名 → 跳到智库 → 顾问位
- 从 MNC RA Senior → 国家局外聘专家委员会
- 学术界（PhD + 论文）→ 各局专家咨询委员会
- 这条路对海归 PhD 实际更通

## 1.7 L1 如何运转 + 如何增长

### 运转机制
- **财政**：100% 中央/地方拨款（NHSA 例外管基金但基金不计自身经费）
- **编制**：5 年一轮编制改革，扩缩按机构改革批
- **流程**：政策草案 → 司局会签 → 部委联签 → 国务院发布 → 各省执行
- **决策权分布**：副部级以下不对外发文；司局长以下不主导政策
- **信息透明**：年度统计公报 / 招聘公告 / 部分财政预算公开

### 增长信号（什么会让 L1 编制扩张）
| 信号 | 影响 |

### 衰减信号（什么会让 L1 编制压缩）
- 医改进入收尾期（短期不太可能）
- 财政紧 → 编制总量控制（已发生，编制内增长慢，编制外增长快）
- AI 替代审评（中期可能压缩纯审评岗，但增加 AI 治理岗）

### 历史轨迹（过去 10 年）
- 2013：NMPA 前身 CFDA 独立成局

# Layer 2: 付费方

> **本层一句话**：钱从口袋到医院/药店/药企的"管道"。中国 L2 的特点是**基本医保一家独大（3.5 万亿）+ 商保停滞（5 年没爆发，2024 增速跌到 8.2%）+ DRG/DIP 改革重塑全行业**。L2 跟 L1 的区别：L1 制定规则；L2 是规则的**执行者 + 资金调度方**。
>
> ⚠️ **重要**：阅读本层时先看 [2.10 修订记录](#210-修订记录我之前写错的-3-处)，第一版写过 3 处与现实不符的内容，已修正。

## 2.0 L2 生态总览

### A. 付费方的 5 块（2024 年规模）

```
中国医疗付费总盘 ~¥9 万亿（=NHE 9.09 万亿）
│
├─ ① 基本医保基金             ¥3.49 万亿/年（基金收入）   ⭐ 最大单一付费方
│   ├─ 职工医保  ¥2.37 万亿收入  3.79 亿参保
│   └─ 居民医保  ¥1.12 万亿收入  9.47 亿参保
│   合计参保 13.27 亿人（95% 覆盖率）
│   累计结存 3.86 万亿（基金"水库")
│
├─ ② 商业健康险                ¥9000-10000 亿/年（估算）
│   头部：人保健康 487 亿 / 平安健康 168 亿 / 各寿险公司健康险事业部
│   占 NHE ~2-3%（远低于美国 ~30%）
│
├─ ③ 政府财政（非医保）          ¥2.26 万亿（NHE 政府支出口径）
│   公卫 + 扶贫医疗 + 基建 + 应急
│
├─ ④ 个人自费 OOP              ¥2.50 万亿（NHE 个人支出口径）
│   不报销部分 + 起付线下 + 超封顶线
│
└─ ⑤ 企业团险 / 福利            ¥几百亿
    雇主补充健康保险 + 体检套餐
```

### B. **付费方力量对比**（决定谁有定价权）

| 付费方 | 体量 | 定价权 | 增速 |
|---|---|---|---|
| 基本医保（NHSA 系统）| 3.5 万亿 | **绝对垄断**（集采 + 国谈 + DRG 都是他定）| 7-8% |
| 商业健康险 | ~1 万亿 | 弱（只能跟着医保走，议价能力低）| 10-15% |
| 政府财政 | 2.26 万亿 | 中（公卫项目可定向）| 5-7% |
| 个人自费 | 2.5 万亿 | 无（patient-as-price-taker）| 持平 |
| 企业团险 | ~几百亿 | 弱 | 8-10% |

**核心事实**：**NHSA 系统是中国医疗事实上的"独家买方"**（monopsony）。商保只是 add-on。医院、药企、械企的全部商业决策都围绕"医保怎么报"展开。

### C. 钱怎么进 NHSA 基金

```
雇主缴费（约工资 8%）─┐
雇员缴费（约工资 2%）─┼→ 职工医保基金 ── 各省统筹（部分地市统筹）
财政补贴            ─┘                  ↓
                                       医院/药店报销
财政补贴 + 个人缴费   ─→ 居民医保基金 ── 县/市统筹
                                       ↓
                                       医院/药店报销
```

⚠️ **统筹层次问题**：基本医保还是"市/地级统筹"为主（少数省级统筹），意味着各省各市规则、报销目录、价格都不一样。**这造就了 L2 大量的地方岗位**。

### D. L2 内部 cross-cut 维度

| 维度 | L2 切分 |
|---|---|
| **公 vs 私** | 体制内（NHSA 系统 + 医院医保办）vs 体制外（保险公司 + TPA 第三方管理 + 健康险 SaaS） |
| **央 vs 地** | 中央（NHSA 总部）vs 地方（31 省级 + 数百市县医保局）— 地方岗位**远多于**中央 |
| **筹资 vs 支付 vs 监管** | 筹资（基金征收）vs 支付（报销审核 + DRG/集采）vs 监管（反欺诈 + 飞行检查） |

## 2.1 工作场景（L2 在做什么）

| 场景 | 谁做 | 做什么 |
|---|---|---|
| **基金征收** | 各市医保局 + 税务总局（2018 后医保费由税务征） | 收职工/居民医保费 |
| **报销审核** | 各市医保经办中心 + 医院医保办 | 审核医院上报的费用是否合规 |
| **集采执行** | 各省医保局招采中心 + NHSA 招采司 | 组织省级集采 + 监督 NHSA 集采落地 |
| **国谈测算** | NHSA + 药物经济学专家组 + 25 名医保谈判专家（来自各省）| 给国谈药品算"医保能承受的价格" |
| **DRG/DIP 推进** | 各省医保局 + 各医院医保办 + 病案室 | 把医院费用按病种打包付费 |
| **反欺诈飞检** | NHSA 基金监管司 + 各省医保局飞检队 | 突击查医院虚假报销 |
| **目录调整** | NHSA 医保中心 + 专家库 | 每年 11 月调整国家医保药品目录 |
| **商保产品设计** | 保险公司健康险事业部 + 精算 | 设计百万医疗 / 重疾险 / 中高端医疗 |
| **TPA 第三方管理** | 思派 / 镁信健康 / 圆心惠保 | 药企-保险-医院之间撮合 + 创新支付 |

**2024-2025 最热点**（决定哪类岗位扩张）：
- **DRG/DIP 2.0 版**（2024-07 发布）→ 各省医保局 + 全国 4000+ 三甲医院的医保办都在扩 DRG 团队
- **第十批集采**（2024-12 开标，累计 435 药品）→ NHSA 招采系统持续招人
- **2024 国谈现场谈判**（2024-11 结束，新医保目录 11 月底公布）→ 谈判专家库扩
- **创新支付**（CAR-T、CGT、罕见病药）→ TPA + 商保公司新设事业部
- **惠民保 → 普惠健康险升级**（各城市跟政府合作的城市定制险）→ 保险公司 + 医保局合作岗

### 医院端 — 医院医保办（payer 在医院的延伸）

- **三甲医院医保办**：每家 10-30 人；全国 ~1600 家三甲 → 约 2-5 万岗位
- **二级医院医保办**：每家 3-10 人；全国 ~9000 家二级 → 约 4-7 万岗位
- **职业方向**：DRG/DIP 数据治理 / **病案编码（专业技术岗，扩张）** / 医保对接 / 飞检应对
- ⚠️ **行政岗位实际在压缩**（修订）：2026 年公立医院清理非核心冗余岗位 **12 万+**，**行政编整体压缩 10%+**（医院运营管理网 / 网易 2026-02 报道）。医保办的**行政部分在缩**；**病案编码、DRG 质控、医疗信息化**才是真正扩张的专业技术岗。第一版写"政策刚需扩张"含糊不准——准确版本是"技术岗扩 + 行政岗缩"。
- **病案编码地位反转**：DRG/DIP 落地后，病案编码员从幕后到台前（影响病案质量 / 等级评审 / 绩效考核 / DRG 支付）。三甲编码员月工作量 200-400 份病案，是 DRG 时代医院最稀缺的技术岗之一。

## 2.5 进入门槛（描述事实，不评判）

### 编制内（最难）
- **NHSA 中央**：国考 ≤35 + 党员 + 名校 + 京户加分
- **省级医保局**：省考 ≤35 + 当地户口或解决
- **市/县级医保局**：当地考试 + 当地户口（强制 in 多数地市）
- **海归 PhD**：可走"高层次人才引进"（罕见），或省级以上"数据/AI 紧缺人才"特招

### 编制外（中等难）
- 医院医保办合同制：相对开放，年龄宽松
- NHSA 大数据中心社招：开放，要算法 / 数据背景

### 体制外（最开放）
- 商业健康险公司：年龄 / 户口不限；要保险或医疗背景
- TPA / 创新支付：互联网 + 医疗跨界背景受欢迎
- 咨询：MBA + 行业经验 + 英语

### 反向"曲线进 L2"通道
- 保险公司健康险事业部做出名 → 政府"专家咨询委员会"
- TPA 公司做创新支付 → 政府医保创新支付试点项目顾问
- 大学卫生经济学 → 各省医保局智库专家
- IQVIA / 麦肯锡 健康险咨询 → 政府改革咨询

## 2.7 L2 如何运转 + 如何增长

### 运转机制
- **筹资**：职工医保 = 雇主 8% + 雇员 2%（工资基数）；居民医保 = 财政补贴 + 个人缴费
- **统筹层次**：地市级统筹为主，少数省级统筹（推动统筹层次抬升是改革方向）
- **基金运行**：基金收入 - 支出 = 年度结存（2024 = 4,639 亿）；累计结存 3.86 万亿（"水库")
- **支付流程**：医院垫付 → 医保结算 → DRG/DIP 打包 → 月度/季度结算回款
- **集采流程**：NHSA 招采司确定品种 → 各省/联盟报量 → 企业报价 → 拟中选 → 落地执行
- **国谈流程**：药企申报 → 形式审查 → 专家评审 → 测算 → 现场谈判（25 名医保专家分 5 组）→ 公布目录

### 增长信号
| 信号 | 影响 |
|---|---|
| DRG/DIP 2.0 全国推广 | 各省医保局 + 医院医保办大扩 |
| 商业健康险渗透率提升（目前 ~3%）| 保险公司健康险事业部扩 |
| 普惠健康险（惠民保）扩城市 | TPA + 城市定制险公司扩 |
| 创新药械医保准入加速 | 国谈测算 + 创新支付 PM 扩 |
| AI 反欺诈强化 | NHSA 大数据中心 + 各省扩 |
| 罕见病 / CGT / CAR-T 支付创新 | 商保 + TPA 新岗位 |
| 老龄化 → 长护险扩面 | 各省长护险经办 |
| 医保统筹层次提省 / 全国 | 中央 NHSA 扩 / 地方收缩重组 |

### 衰减信号
- 基本医保基金压力（老龄化 + 经济周期 + 缴费率上调难）→ 控费倒逼集采扩品种 → 药企/械企裁员压力（不是 L2 自身）
- 互联网医院冷却（部分平台已收缩）→ 平安好医生 / 微医等 TPA 端裁员
- **商保渗透 5 年没爆发**（修订加深）：2024 健康险增速从 2020 的 15.7% 跌到 **8.2%**；综合赔付仅占 NHE 的 **3.3%**；**惠民保对重疾险的"挤出效应"**（惠民保 5 年达全国 10% 参保率 → 抢了重疾险/百万医疗的市场）
- **公立医院行政岗位压缩**（修订加深）：2026 行政编 -10%，医保办行政部分缩，专业技术（病案 / DRG 质控）扩 → 医保办不是单边扩张

### 历史轨迹
- 2016：DRG 上海试点
- 2018：NHSA 成立（人社部医保司 + 卫健委定价权 + 发改委药价权 三合一）
- 2018-2019：医保税务征缴（征收效率大涨）
- 2019：4+7 城市药品集采首批（影响行业格局至今）
- 2020-2023：10 批集采 + 5 次国谈，435 药品 + 创新药入医保加速
- 2021-2024：DRG/DIP 三年行动计划
- 2024：DRG/DIP 2.0 版 + 第十批集采 + 国谈新版目录

## 2.8 业务实图（L2 的"血液系统"到底怎么流）

文字描述不够清楚，画 4 个闭环。

### A. DRG 报销闭环（一笔医保钱怎么从医院流到回款）

```
[患者出院]
   │
   ↓ 24h
[临床医生填病案首页] ─────────→ [病案编码员编码]
   │                                  │
   │                                  ↓ 72h
   │                            [科室 DRG 质控员审核]
   │                                  │
   │                                  ↓
   │                            [病案室上报医保结算清单]
   │                                  │
   │                                  ↓ 月度
   │                            [医院医保办收齐月度费用]
   │                                  │
   │                                  ↓
   │                            [医保经办中心审核（费率+合规）]
   │                                  │
   │                                  ↓
[患者已自付部分]               [生成月度结算总表]
                                      │
                                      ↓
                               [医保基金按 DRG 浮动费率结算给医院]
                                      │
                                      ↓ 年终
                               [年度核算+扣除违规（错编/不规范）+ 调整次年费率]
```

**关键岗位插入点**：
- 临床医生（L4 层）→ 写诊断和操作（错一个字 = DRG 组错 = 钱少）
- **病案编码员**（L2 ⭐ 关键稀缺岗）→ 把诊断翻译成 ICD-10/9-CM 编码
- **DRG 质控员**（L2 ⭐ 新岗）→ 看哪些病例 DRG 分组不合理
- 医院医保办对接专员（L2）→ 跟医保经办沟通
- 医保经办审核员（L2 各市医保中心）→ 审清单
- NHSA 大数据中心（L2 中央）→ 跨省抽样监测异常

**业务核心矛盾**：医院想多报费用，医保局想控费 → 这个矛盾是 L2 半数岗位存在的理由。

### B. 集采闭环（一个药品如何从厂家到医院架子）

```
[NHSA 招采司确定品种]
   │
   ↓
[各省/采购联盟向医院"报量"]
   │
   ↓
[联合采购办公室公布采购规则]
   │
   ↓
[企业报价（封闭式）]
   │
   ↓
[拟中选 → 企业选择供应地区（不得弃权）]
   │
   ↓
[签备忘录 → 公告中选结果]
   │
   ↓ 1-3 月
[各省医保部门组织医院与中选企业签采购协议]
   │
   ↓
[药品通过流通公司送达医院]
   │
   ↓ 落地第 3 月起
[NHSA + 各省排查 "进院情况"（防医院偷偷不采集采品种）]
   │
   ↓ 持续
[全国统一医保信息平台 招采子系统 动态监测]
```

**关键岗位插入点**：
- NHSA 招采司 + 招采指导中心（L2 中央）
- 各省医保局招采处（L2 地方）
- 联合采购办公室（轮值，由某省牵头）
- 流通公司 BD（L3）—— 跟医院签合同 + 配送
- 医院药剂科（L4）—— 实际下采购订单
- 药企集采策略部（L5）—— 算报价 + 选择供应地区

**业务核心矛盾**：药企想保利润 vs 医保局想压价 vs 医院想用熟悉的非集采产品 → 三方博弈。

### C. 国谈"灵魂砍价"闭环（一个创新药如何进医保目录）

```
[药企申报] (每年 7-9 月)
   │
   ↓
[形式审查 + 临床专家组评审（含院士级专家）]
   │
   ↓
[药物经济学组 + 医保测算组并行测算]
   │            │
   │            ↓
   │      [医保底价（"神秘信封"）— 谈判前药企不知道]
   │
   ↓
[NHSA 现场谈判 (10-11 月，4-5 天)]
   │
   ├── 25 名医保谈判专家（来自全国各省，10-20 年医保经验）
   ├── 分 5 组，每组 5 人
   ├── 1 v 1 跟药企谈
   │
   ↓
[药企报价 ≤ 医保底价 → 进医保（最多两次报价机会）]
[药企报价 > 医保底价 → 出局]
   │
   ↓ 11 月底
[公布新版国家医保药品目录]
   │
   ↓ 次年 1 月
[新目录全国执行]
```

**2024 国谈数据**（NHSA 公开）：
- 涉及 127 家企业、162 种药品
- 谈判成功 119 种（成功率 73.46%）
- **平均降价 50.64%**
- 谈判周期：4-5 天，每个药品 30-60 分钟

**典型故事**（验证机制存在）：阿斯利康"金牌谈判手"累计参与 26 场国谈，2026-03 离职引发行业讨论 → 印证"国谈代表"是真实的稀缺岗位。

**业务核心矛盾**：药企想用专利期赚回研发成本 vs 医保局想让更多人吃到药 → 通过"以价换量"达成（降价 50% 但销量增 5-10 倍）。

### D. 反欺诈闭环（医保局怎么抓医院 / 药店骗保）

```
[全国统一医保信息平台 反欺诈数据监测专区]
   │
   ├── "虚假住院" 模型
   ├── "医保药品倒卖" 模型
   ├── "医保电子凭证套现" 模型
   ├── "重点药品监测分析" 模型
   │
   ↓
[异常线索（如：某医生开药金额 = 全国平均 200x）]
   │
   ↓
[NHSA + 公安部 数据研判 (已发现 2000+ 涉案线索)]
   │
   ↓
[省/市医保局组飞行检查队]
   │
   ↓
[突袭检查（不通知）]
   ├── 查病历是否真实
   ├── 查 CT/MRI 报告是否伪造
   ├── 查服务项目是否虚构
   ├── 查住院床位是否空挂
   │
   ↓
[处罚（追回 + 罚款 + 终止协议 + 移送司法）]
```

**2018-2023 累计数据**（NHSA 公开）：
- 检查 357.9 万家次
- 处理 170.3 万家次
- 追回 **835 亿**
- 一年追回 200+ 亿

**典型案例**：沈阳林济中医院（伪造病历 / 伪造 CT 报告 / 虚构医药服务，涉及 166.2 万）

**算法核心**（来自公开技术文章）：
- 关联特征建立"患者-患者 / 患者-机构"同构异构关系网络
- 稠密子集检测算法识别异常群体
- 这是典型的图神经网络 (GNN) + 异常检测任务

**业务核心矛盾**：基金有限 vs 医院 / 药店 / 医生有人性 → 永恒博弈，反欺诈是 L2 内 AI 应用最成熟的方向。

## 2.9 Insider 视角（来自知乎 / 业内媒体 / 真实从业者声音）

每条都标了出处，目的是让你**看到教科书 vs 现实的 gap**。

### 路径 A: 海归 PhD → NHSA 大数据中心

- **现实**：NHSA 大数据中心 2018 才成立，编制扩张快，**真在用 AI（GNN + 异常检测）**，是 L2 内最对口海归 PhD 的岗位。
- **薪资真相**：编内 25-40 万 + 隐性福利（北京户口 / 公积金高 / 退休保障）→ 比商保 / TPA 同年龄段低 50%，但稳定性 / 社会地位高
- **天花板**：处级 / 副司级（同部委内晋升），跳出 NHSA 难度高（专业性太垂直）
- **入局现实**：国考 35 岁 + 党员 + 北京户口加分。海归博士走"高层次人才引进"是稀有路径（往往要省级以上层面推荐）

### 路径 B: 工业 → 商保 AI

- **现实**：商保整体增速 8.2%（从 2020 的 15.7% 跌下来）→ 增长红利已过
- **行业麦肯锡报告原话**："保险公司核心能力欠缺（深度整合医疗服务体系 / 精细化风险分层 / 精准客户需求识别），难以持续推出有效产品"
- **真相**：商保数据科学家被招进来后，**最大问题是没数据**（医院 / 医保的数据拿不到，自己保单的数据样本太薄）→ 大量"AI 岗"实际在做 BI 仪表盘
- **薪资真相**：平安 / 众安 / 蚂蚁保 大厂数据科学家 40-100 万，**但裁员风险高于互联网**（保险行业整体收缩）
- **天花板**：到精算总监 / 健康险事业部总经理 100-500 万，但要熬 15+ 年

### 路径 C: 创新支付 PM（镁信 / 思派 / 圆心）

- **必须知道的现实**（前面已修订）：这些不是真正的创新支付公司，**90%+ 营收来自特药药房**
- **加入后真实工作**：80% 时间做药企 BD / 患者服务 / 药房运营，20% 做"保险产品"（多数是被监管点名的"通道业务"）
- **监管风险**：2023-11 NHSA 联合金融监管局点名"药转保"，整个赛道处于合规收紧期 → 短期内裁员风险
- **圆心科技状况**：5 次递表港交所未上 / 销售费用近 29 亿 / 持续亏损 → 模式可持续性存疑
- **判断**：想做"创新支付"的人，**反而应该看政府主导的惠民保 / CGT 分期付费等真正的创新支付场景**，不是这 3 家公司

### 路径 D: 三甲医院医保办 / 病案室

⚠️ **必须知道的现实**（前面已修订）：
- 三甲医院 2026 已清理非核心冗余岗位 12 万+，行政编 -10%
- **医保办的行政部分在缩**，**病案编码 + DRG 质控才在扩**
- 从知乎（《三甲医院编码员一个月工作量多少合适》），编码员月工作量 200-400 份，时间紧 + 责任大
- **跳槽风险**（来自医院运营管理网 2026-02）："离开三甲后想回原单位，降级都回不去" / "私营医院不建议去" → 公立医院是单向门
- **入编竞争**：入编比合同工有显著优势（退休时尤其明显），但排队进编要数年
- **真实从业者认知**：DRG 改革让病案科"从幕后到台前"，行业地位上升，但工作量也同步上升 → 短期红利期

### 通用 insider 观察（跨路径）

- **"医保系统是中国版的 NHS（英国国民医疗）"** — 体量、权力、运作逻辑很像，但中国是后发优势 + 数字化超前（全国一张网比英国先进）
- **NHSA 8 年走完了美国 CMS 30 年的路**（2018 成立 → 2024 已实现集采 + 国谈 + DRG/DIP + 全国一张网）→ 编制扩张红利期还没结束
- **三甲医保办的"病案编码 + DRG 质控"** 是被 L2 各角色暗暗惦记的 stepping stone（学 DRG 内部规则 + 攒人脉，3-5 年后跳到 NHSA / 商保 / TPA）

| A7 咨询 / 智库 | ~5,000-10,000 | 健康险专项 |
| **L2 总计** | **~50-100 万人** | 跟 2.0 总览的 80-120 万一致 |

**对比 L1（30-50 万）**：L2 体量是 L1 的 2-3 倍，因为 L2 涉及全国 4000+ 医院 + 30 家商保 + 13 亿参保人的服务。

---

### D. 关键观察（这一节给框架的 4 个 takeaway）

1. **L2 内"事业编工资 25-40 万 + 户口 + 退休"** vs **"商保 / TPA 50-200 万 + 不稳"** 是两条平行轨道，几乎不交叉。选哪条 = 风险偏好选择。

2. **AI / 数据岗位 spread**：编内（NHSA 大数据中心 25-40 万）→ 厂商（久远 / 卫宁 25-50 万）→ 商保互联网（蚂蚁保 / 众安 40-100 万）。同样的算法 skill，在不同 sub-layer 工资可以差 3-5 倍。

3. **A6 医保 IT 厂商** 是被前 4 个版本框架忽略的"中间地带"——既能用 AI / 数据 skill，又不需要编制，工资居中，跟着政策走稳定增长。**对海归 PhD 是最容易直接 land 的 sub-layer**。

4. **A4 商保 vs A5 TPA** 的求职逻辑完全不同：商保看精算 / 风控 / 销售 / 产品；TPA 看 BD / 药企关系 / 创新支付模式设计。两者人才几乎不互通。

# Layer 3: 流通与零售

> **本层一句话**：药 / 器械 / 健康品 从工厂到医院 / 药店 / C 端的"血管系统"。中国 L3 的特点是**寡头 + 持续被挤压**——CR5 占 51.3% 但行业利润率从 2010 年 2.2% 跌到 2023 年 1.6%；集采 / 两票制 / 处方外流 / DTP 双通道四股力量同时改造这一层。L3 跟 L2 的区别：L2 给钱（payer），L3 搬货（middleman）。L3 与 L5 的区别：L5 造药 / 器械，L3 把它送出去。
>
> ⚠️ L3 是中国医疗行业**就业体量最大的层之一**（~200-300 万人），但 **AI / 高薪岗位密度远低于 L2 / L5**。绝大多数岗位是销售 / 药师 / 物流。

### B. **L3 是"持续被挤压"的层**（关键事实）

| 年份 | 行业平均利润率 |
|---|---|
| 2010 | 2.2% |
| 2023 | **1.6%** |

挤压来自 4 个方向：
1. **集采**："以价换量"，药品出厂价下降 90%+ → 流通商加价空间被压缩
2. **两票制**（2017 起）：从生产到医院最多 2 道发票，砍掉 3-4 层经销商 → 中小流通商死亡
3. **处方外流**：医院 → 药店转移，流通商被迫从 B2B 转 B2C/DTP（不会做 C 端的就死）
4. **互联网医药**（京东健康 / 阿里健康 / 拼多多）：传统药店再次被分流

### C. **L3 的 4 大未来增长引擎**（被挤压中的新空间）

| 引擎 | 规模 | 状态 |
|---|---|---|
| **DTP 专业药房** | 872 亿（2024）/ 预 2028 5,000 亿 | 高速扩张（4 年内门店翻倍）|
| **双通道药房** | 国药已 1,211 家 / 全国 7,000+ | 2021 政策落地后扩张 |
| **互联网医药** | 京东健康 582 亿 / 阿里健康 306 亿 | 增速放缓但仍 +8-13% |
| **智慧物流 / 智慧供应链** | 国药 600+ 物流中心数智化 / 九州通 12 亿北京枢纽 | AI 应用最快增长点 |

### D. L3 内部 cross-cut 维度

| 维度 | L3 切分 |
|---|---|
| **B2B vs B2C** | 批发流通（B2B）vs 零售药店（B2C）vs DTP（B2C 但专业药）|
| **传统 vs 数字** | 传统流通商（国药 / 上药）vs 互联网（京东健康 / 阿里健康）vs 智慧物流（国药数智 / 九州通 AI）|

**2024-2025 最热点**（决定哪类岗位扩张）：
- **DTP 双通道扩张**（4 年门店翻倍）→ 各流通商 + 连锁药店 + 创新支付公司都在抢 DTP 药师 / BD

- **处方外流落地**（医保支付改革推）→ 医院流向药店的处方数量持续增加

## 3.5 进入门槛

## 3.7 L3 如何运转 + 如何增长

### 运转机制
- **现金流模式**：高周转低毛利（毛利率 6-8%，净利率 1-2%）；账期 60-180 天是行业常态
- **集采落地执行**：流通商负责"最后一公里"——把集采品种从厂运到医院 + 货款回收
- **两票制**（2017 起）：生产到医院最多 2 道发票，砍掉 3-4 层经销商，行业出清
- **处方外流**：医院 → 药店 / DTP 转移，流通商 + 零售 + DTP 重新分蛋糕
- **三医联动改革**：医保 + 医疗 + 医药联动 → 改变 L3 的整个收入结构

### 增长信号
| 信号 | 影响 |
|---|---|
| DTP 双通道扩 | 流通商 DTP + 思派镁信圆心 + 连锁 DTP 全部受益 |
| 智慧供应链投资 | 国药数智 + 九州通 AI 仓储 + 京东物流医药扩 |
| 处方外流政策落地 | 互联网医药 + 连锁药店 + DTP 全受益 |
| 创新药入医保（国谈）| DTP 药房承接关键 |
| 慢病管理人群扩 | 连锁药店 + DTP + 互联网医药 |
| 老龄化 + 居家医疗 | 上门配药 + 末端配送扩 |
| 出海（一带一路医药商业）| 部分流通商出海试点 |

### 衰减信号
| 信号 | 影响 |
|---|---|
| 集采品种持续扩 | 流通商毛利持续被压缩 |
| 关店潮（一年关近 4 万家）| 中小药店出清 |
| 利润率从 2.2% 跌到 1.6% | 行业整体被挤压 |
| 拼多多医药模式（极致效率）| 传统平台 + 连锁被冲击 |
| 处方外流执行不及预期（部分省）| DTP 增速不达预期 |

### 历史轨迹
- 2017：两票制全国推开（行业重大转折）
- 2017-2020：中小流通商大量出清，CR5 集中度抬升
- 2018：4+7 城市集采首批（开启集采时代）
- 2019-2024：10 批集采，行业利润率从 2.2% → 1.6%
- 2020-2025：互联网医药崛起（京东健康 / 阿里健康 + 拼多多）
- 2021：双通道政策落地（DTP 黄金窗口）
- 2024：国药智慧供应链全面 AI 化 / 九州通北京 12 亿枢纽
- 2024-2025：连锁药店关店潮（行业洗牌）

## 3.8 业务实图（4 个闭环）

### A. 药品从厂到医院（两票制下流通闭环）

```
[药企生产]
   │
   ↓ 第 1 票（药企 → 流通商）
[流通商（如国药 / 上药 / 华润）]
   │
   ↓ 第 2 票（流通商 → 医院 / 药店）
[医院药剂科 / 零售药店]
   │
   ↓
[患者]
```

**关键点**：两票制砍掉 3-4 层中间商，流通商加价空间约 6-8% 毛利。集采品种这个加价空间被进一步压缩到 2-4%。

### B. 集采品种流通改造闭环

```
[NHSA 集采中选 → 报量到各省（L2）]
   │
   ↓
[流通商（中标的指定配送商）]
   │
   ↓ 货款 60-180 天账期
[医院药剂科]
   │
   ↓
[患者用药]
   │
   ↓ 货款回收（流通商压力最大）
[流通商承担：库存 + 配送 + 账期 + 政策响应]
```

**为什么流通商被挤压**：集采品种降价后，药企本身利润压缩 → 流通商加价空间从 6-8% 压到 2-4% → 货款账期还是 60-180 天 → 现金流压力巨大。

### C. DTP 处方流转闭环（连接 L4 医院 → L3 DTP）

```
[医院医生开特药处方（如肿瘤靶向药）]
   │
   ↓
[患者持处方到 DTP 药房（医院附近 / 互联网）]
   │
   ↓ 双通道：医保直接报销
[DTP 执业药师审方 + 配药 + 患者教育]
   │
   ↓
[冷链配送（部分药）]
   │
   ↓
[用药随访（药师 + 数字化平台）]
   │
   ↓
[复购 / 续方提醒]
```

**关键点**：DTP 不只卖药，是"专业药师服务 + 患者管理 + 医保结算 + 配送" 一站式。这就是 DTP 毛利 15-30%（远高于普通药店 8-10%）的原因。

### D. 互联网医药交易闭环（京东健康为例）

```
[用户在京东 APP / 京东健康 APP 下单]
   │
   ↓ 处方药 vs OTC
[处方药：在线问诊 → 处方流转 → 执业药师审方]
[OTC：直接下单]
   │
   ↓
[京东健康自营仓库（85% 占比）or 第三方平台商]
   │
   ↓ 京东物流配送（C 端 1-2 日达）
[用户收药 + 用药随访（部分慢病有）]
```

**为什么京东健康赚钱**（净利 47.9 亿 / +93.9%）：高自营占比（85%）+ 京东物流自有 + 处方药占比抬升（高毛利）+ 健康产品多元化。

## 3.9 Insider 视角

### 路径 A: 国药 / 上药 / 华润 / 九州通 体系

- **真实薪酬**：年薪 20-30 万常见，优秀 30-50 万（来自知乎《中国医药集团员工薪资被泄露》）
- **校招规模**：国药 2026 校招 **2,472 + 社招 1,010 = 3,482 人**，覆盖 150 家子公司
- **岗位类型**：医药代表 / 销售专员 / 采购管理 / 医药电商运营 / 医疗大数据分析师
- **福利**：周末双休 + 五险一金 + 内部晋升 + 租房补贴 + 司龄补贴 / 餐饮补贴 / 体检
- **晋升路径**：销售 → 区域经理 → 省分总经理 → 集团事业部
- **稳定性**：央企体系，稳定但创新慢；上升通道清晰但慢

### 路径 B: 连锁药店店长 / 督导

- **真实薪酬**：店长年薪 6-15 万（含店长岗位补贴 + 业绩提成）
- **业绩压力**：底薪 + 提成结构，业绩差 = 收入低
- **加班强度**：知乎从业者反馈"996 都不是事"，零售线长期需要应对客流
- **督导岗最累**：负责区域多店，每天到处跑，"早上出门忙到天黑"
- **执业药师溢价**：考取执业药师证后，月工资多 800-2000 元
- **天花板**：店长 → 督导 → 区域总（年薪 20-50 万）→ 集团事业部
- **转行情况**：很多人离开后做药品培训 / 转药企，收入翻倍

### 路径 C: DTP 药师 / DTP BD ⭐（增长最快的 sub-track）

- **DTP 市场规模 2019 = 345 亿 → 2024 = 872 亿** → 4-5 年内还能再翻几倍（预 2028 = 5,000 亿）
- **DTP 药师** 比普通药师工资高 50-100%（专业化高 + 处方药 + 患者管理）
- **DTP BD** 跟药企 / 医院 / 保险公司 / TPA 多方对接，是 L3 内薪资上限最高的非管理岗（30-100 万）
- **真实工作**：从知乎 / 业内分享，DTP 药师工作内容 = 30% 配药 + 30% 患者教育 + 20% 用药随访 + 20% 医保政策对接
- **困境**：DTP 800 亿市场，但盈利能力差（高房租 + 高人力 + 集采压价）→ 行业增收不增利

### 路径 D: 京东健康 / 阿里健康 算法 / 产品

- **真实薪酬**：互联网大厂标准（数据科学家 40-100 万，PM 30-80 万）
- **业务结构**：京东健康 85% 自营 → 实际是"医药电商 + 物流"为底层，"互联网医院 + AI"为增长故事
- **真实工作**：数据科学家做"商品推荐 + 用户分层 + 履约优化"，不是"医疗 AI" — 跟互联网电商更像
- **晋升 + 跳槽**：内部晋升 + 跳到字节 / 阿里 / 拼多多医药事业部，互联网圈内流动性高
- **风险**：互联网医药增速放缓（京东健康 +8.6%），裁员风险高于互联网平均

### 路径 E: 国药物流 / 九州通 智慧供应链算法工程师 ⭐（被低估的 sub-track）

- **背景**：国药 600+ 物流中心 + 上海中心数字孪生货位优化（拣货 -40%）；九州通北京 12 亿枢纽 + AMR + RGV 立库 150 万件
- **真实工作**：典型 OR + ML 任务（库存优化 / 路径优化 / 需求预测 / 自动化运维）→ **跟工业 4.0 / 智能制造接近，跟医疗 AI 距离远**
- **优势**：稳定（央企 / 大型民企）+ 工资居中（25-60 万）+ 不需要 leetcode（工业岗）+ 政策稳健（医药物流是刚需）
- **劣势**：技术 stack 偏传统（OR + Python，少前沿 LLM）+ 行业封闭（跳出难）

### 通用 insider 观察（跨路径）

- **L3 是中国医疗行业里"勤劳致富"的代名词**：高就业体量 / 低薪资中位 / 低门槛 / 长时间 / 拼业绩。年轻人收入弹性大，但天花板远低于 L2 商保 / L5 创新药企
- **L3 集采冲击不可逆**：行业利润率 2.2% → 1.6% 是结构性趋势，不会反转。流通商被迫 DTP / 互联网 / 物流转型
- **执业药师证是 L3 唯一的"中国第一证"**：含金量高于英语 / 计算机证书，跨流通商 / 零售 / DTP / 互联网医药都通用
- **互联网医药 vs 拼多多医药的反差**：京东健康 3,118 人做 582 亿；**拼多多医药 10 余人做 600 亿 GMV** → 极致效率冲击传统模式，未来 L3 互联网线还有结构性洗牌

- B1 销售 + B2 药学 + B3 物流 = **~260 万**
- B4-B6 合计 ~13 万
- 总计 **~270 万** → 跟 3.0 总览的 200-300 万范围一致
- **是 L2（50-100 万）的 3-5 倍**，是 L1（30-50 万）的 6-8 倍

### D. 关键观察（这一节给框架的 4 个 takeaway）

1. **L3 是就业体量最大的层之一**（~270 万），但**绝大多数是销售 + 药师 + 物流**，AI / 高薪岗位密度远低于 L2 / L5
2. **L3 内 AI 高薪岗位主要集中在三块**：①智慧供应链算法（九州通 / 国药物流 / 京东物流医药）②互联网医药算法 / PM（京东健康 / 阿里健康）③DTP 患者管理 SaaS（思派 / 镁信 / 圆心）
3. **L3 是"勤劳致富"的层**：低毛利 + 低门槛 + 高强度 + 高弹性。年轻人收入弹性大，但**天花板低于 L2 商保 / L5 创新药企**
4. **集采冲击不可逆**：行业利润率从 2010 的 2.2% 跌到 2023 的 1.6%。流通商被迫向 **DTP / 互联网 / 智慧物流** 转型，这三条路径决定 L3 未来 5-10 年的就业增量
