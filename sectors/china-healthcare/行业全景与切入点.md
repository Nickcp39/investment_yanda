## 3. 行业结构图

### 3.1 A 层 监管/政策（设规则的人）

```
国务院
│
├── 国家卫健委 NHC（正部级）
│   ├── 国家中医药管理局
│   ├── 国家疾控局（CDC）
│   └── 各省/市/县卫健委 + 公立医院系统
│       职能：医院 / 医生 / 公卫 最高主管
│
├── 国家医保局 NHSA（副部级，独立）
│   ├── 医保中心（DRG/DIP 主推）
│   ├── 价格招采司（国谈 / 集采）
│   └── 各省/市/县医保局
│       职能：报销 + 价格 + 集采
│       核心：最有钱的部委（~3 万亿基金）
│
├── 国家市场监管总局
│   └── 国家药监局 NMPA（副部级）
│       ├── 药品审评中心 CDE（北京）
│       │   └── 长三角 / 大湾区 分中心
│       ├── 医疗器械技术审评中心 CMDE / 器审中心（北京）
│       │   └── 长三角 / 大湾区 / 京津冀 / 华中 / 西南 分中心
│       ├── 食品药品审核查验中心 CFDI（北京）
│       ├── 中国食品药品检定研究院 NIFDC / 中检院
│       └── 各省药监局
│           职能：药品 / 器械 / 化妆品 准入审批
│
├── 国家发改委
│   职能：产业政策 + 价格指导
│
└── 工信部医药司
    职能：制造业支持 + 应急储备
```

**核心二元论：药监局 vs 医保局**
- 药监局：**能不能卖**（准入）
- 医保局：**卖多少钱 / 能不能进医保**（定价 + 报销）
- 两个互相不搭，但药企/械企必须两边都搞定

### 3.5 钱怎么流

```
医保基金 + 商保 + 个人自费 + 财政
                │
                ▼
   医院 / 零售药店 / 互联网平台
                │
                ▼
   药企 / 械企 / 医疗服务商
                │
                ▼
   CRO / CMO / IT / 上游元器件
                │
                ▼
       高校 / 研究所 / 上游材料
```

**监管在每一层都设关卡**：
- 注册关 → 药监局
- 招标关 → 卫健委 + 医院
- 报销关 → 医保局
- 价格关 → 医保局
- 监管关 → 各级药监 + 卫监

### 3.6 决定行业冷热的政策杠杆

| 杠杆 | 谁拍板 | 影响 |
|---|---|---|
| 国谈 | 医保局 | 创新药能不能进医保（销量天花板） |
| 集采 | 医保局 + 医保中心 | 仿制药 / 耗材 / IVD 价格腰斩 |
| DRG / DIP | 医保局 | 医院按病种付费 → 医院控费动力暴增 |
| **AI 医疗器械审评指南** | NMPA / CMDE | AI 软件能不能作三类器械上市 ⭐ 你方向 |
| 真实世界研究指南 | NMPA + 卫健委 | 上市后数据要求 |
| 互联网医院政策 | 卫健委 | 线上问诊 / 处方流转 |
| **AI+医疗实施意见**（2025-11） | 卫健委 | 顶层文件给医疗 AI 开绿灯 |

## 4. 三城落户对比（北上深）

| 维度 | 北京 | 上海（2025-12-01 新政） | 深圳 |
|---|---|---|---|
| 你能走的通道 | 央属单位 + 教育部留服 | PhD + TOP 500 直落 | 博士 ≤ 45 岁直落 |
| UB 排名是否够 | ❌ QS 三年没进 TOP 300 | ✅ QS #410 在 TOP 500 内 | ✅ 不看排名 |
| 社保门槛 | 缴 3 个月 + 单位指标 | 1 个月，无基数门槛 | 无社保要求 |
| 年龄红线 | 45 | 无（PhD 通道） | 45 |
| 单位资质要求 | 必须央属企事业 | 在沪正规单位即可 | 在深正规单位即可 |
| 现金补贴 | 无 | 无 | 博士 3-10 万 |
| 你的实际落户概率 | 50-60%（看单位指标） | 90%+ | 95%+ |

**结论**：落户难度 北京 >> 上海 ≈ 深圳。落户不是真问题，瓶颈在"找到合适的事做"。

## 5. 我能切的 5 个口子

**坐标系**：C 层产业 × 我的技能（医学影像 AI + LLM + 物联网创业）。按匹配度排序。

| # | 切口 | 行业位置 | 干什么 | 匹配度 | 收入估算 |
|---|---|---|---|---|---|
| 🎯 1 | **AI 医疗器械注册咨询** | C5 配套 | 给做 AI 影像/CADx 的初创公司提供：注册路径设计 + 临床方案 + 与器审中心沟通 + 应对审评意见 | ★★★★★ 你是内行 | 单项目 20-100w / 顾问 5-15w/月 |
| 🎯 2 | **LLM 工具卖给 CRO/RA/医院** | C4 + C5 | 把你的 LLM 报告 QA 能力做成 SaaS，卖给 CRO 写申报材料、药企 RA 起草监管资料、医院 GCP QA | ★★★★ | SaaS 订阅 / 按文档计费 |
| 🎯 3 | **医院医工科项目外包** | C3 + C4 | 接三甲医院的"AI 影像辅助诊断"项目；帮医院申请国自然 / 北京自然基金课题 | ★★★★ | 单项目 30-150w，3-12 个月 |
| 🎯 4 | **真实世界研究（RWS）数据服务** | C5 | 帮 AI 医疗器械厂商收集 / 清洗 / 分析上市后数据，写 RWS 报告交监管 | ★★★ | 项目制 |
| 🎯 5 | **AI 医疗投资 DD 顾问** | 跨层 | VC/PE 投 AI 医疗器械时缺懂技术+懂监管的 DD 顾问 | ★★★★ 用你的投资 mindset | 单次 5-30w，3-7 天工作量 |

**为什么这五个口子甜**：
- 传统 CRO（药明、泰格）不懂 AI
- AI 公司不懂 NMPA 监管语言
- 你混血背景（医学影像 PhD + LLM + 创业）正好填这个空白
- AI 医疗器械是**新行业**，没有头部固化

## 7. 下一步要回答的未决问题

按优先级排：

1. **【最近就要答】先做哪 1-2 个口子试水？**
   - 候选：口子 1（注册咨询）+ 口子 5（VC DD）—— 都是项目制、低投入、能快速试错

3. **【2026 Q3 答】具体定居哪个城市？**
