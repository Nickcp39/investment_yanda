# Wall Street News

Use this folder for Wall Street news inputs if the source is article/RSS/news
based rather than a YouTube channel.

Recommended files:

- `manifests/` - dated URL and metadata lists.
- `articles/` - raw article captures or source excerpts where allowed.
- `runs/` - scraper/import run reports.

If the source is a YouTube channel, put it under
`sources/channels/wall-street-news/` instead and use
`scripts/youtube_channel_queue.py`.
