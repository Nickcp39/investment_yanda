# NOVT Value Chain Map — as_of 2026-07-10

```
[Raw optics / magnets / semis] -> [NOVANTA: precision sub-components] -> [OEM capital-equipment makers] -> [end users: hospitals, factories, fabs]
```

- **Position:** Tier-2/critical-component supplier. Novanta sells sub-systems (encoders, servo drives, lasers, beam-steering, end-of-arm tooling, insufflators, light engines) to **OEMs** who integrate them into surgical robots, industrial/warehouse robots, semiconductor tools, and medical devices.
- **Upstream:** commoditized inputs (optics, magnets, electronics) — Novanta adds value via design/precision, so input cost is not the constraint.
- **Downstream OEMs (the customers):** surgical-robotics & minimally-invasive OEMs, industrial-robot/cobot makers, warehouse-automation integrators, semiconductor-equipment makers. Low single-customer concentration (arms-dealer model).
- **Where the profit pool sits:** in the *designed-in* precision spec. Once Novanta's part is qualified into an OEM platform (multi-year cycle, regulatory in medical), it is sticky and hard to swap — that is the switching-cost moat.
- **Robotics-chain placement:** Novanta is a **picks-and-shovels** supplier to robotics, NOT the robot/AI-brain owner. It benefits if robots proliferate but captures a component-level slice, not the system margin. Contrast NVDA (owns the compute chokepoint) — Novanta owns *a* precision niche, not *the* bottleneck.
