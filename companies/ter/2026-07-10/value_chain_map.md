# TER Value-Chain Map — as_of 2026-07-10

## Semiconductor Test (86.6% of Q1'26)
```
AI chip designers (NVDA, AMD, Broadcom, hyperscaler custom ASIC, memory makers)
        │  design more complex silicon (AI accelerators, HBM, networking) -> more test per die
        ▼
Foundry / OSAT / IDM  ──buy──▶  [ TERADYNE ATE ]  ◀──duopoly──  Advantest
        │                         testers + software + service
        ▼
Tested chips -> systems -> AI data centers
```
- **TER's position**: a **toll booth on test intensity**. As AI silicon complexity rises, test time/coverage per device rises → structural tailwind on top of the unit cycle.
- **Customers**: concentrated among leading-edge logic + memory makers; large, sophisticated buyers.
- **Suppliers**: instrument/component vendors; not a bottleneck for TER.
- **Substitutes**: in-house test (rare at leading edge), Advantest (the other duopolist). Cobalt point: SoC test is TER-led; memory/SoC split contested with Advantest.

## Robotics (7.1% of Q1'26)
```
UR cobots / MiR AMRs ──via distributors/integrators──▶ factories, e-commerce/logistics, electronics, semi fabs
```
- Fragmented, competitive downstream; UR pioneered easy-to-program cobots (UR+ ecosystem) and MiR does AMRs.
- **Commoditizing**: Fanuc, ABB, KUKA, Doosan, Techman, JAKA (cobots); many AMR players. Pricing pressure + industrial-automation cyclicality.
- Physical-AI angle (NVIDIA GTC demo with "Generalist") is upstream-narrative, **not yet a value-chain lock-in**.

## Where the profit pool sits
Overwhelmingly in **Semi Test** (duopoly economics, 58–61% gross margin). Robotics is a **negative** contributor today. Product Test is small/steady.
