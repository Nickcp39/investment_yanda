# GOOGL Industry / Value Chain

Last updated: 2026-06-15

Status: skeleton.

## Value Chains To Map

Google is not one value chain. The research needs separate maps:

1. Search advertising
2. YouTube / creator video advertising
3. Mobile ecosystem: Android, Chrome, Play, distribution
4. Cloud infrastructure and enterprise software
5. AI compute stack: data centers, TPU, power, networking, models, applications

## Preliminary Map

```text
Users / creators / developers
  -> Google distribution surfaces
  -> Advertisers / enterprises / subscribers
  -> Google monetization engine
  -> Infrastructure suppliers: chips, networking, data centers, power
```

## Profit-Pool Questions

- In search, how much value is captured by Google vs advertisers vs publishers?
- In Cloud, does value accrue to hyperscalers, chip suppliers, model labs, or enterprise software layers?
- In AI, is Google a bottleneck owner or mostly a capex funder?
- What part of the chain can competitors attack without matching Google's full distribution?
