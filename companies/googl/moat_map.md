# GOOGL Moat Map

Last updated: 2026-06-15

Status: skeleton. Do not treat this as conclusion.

## Moat Candidates

| Moat source | Possible evidence | Metric to verify | Attack surface |
|---|---|---|---|
| Search habit and distribution | Default search, Chrome, Android, brand habit | Search revenue, query growth, TAC, market share | AI-native search, regulation, default restrictions |
| Advertiser data / auction liquidity | Large advertiser base and conversion data | Ad revenue, CPC, ROI commentary | Amazon/TikTok/Meta/AI agents |
| YouTube network effects | Creator supply, audience scale, recommendation data | Watch time, ad revenue, subscription growth | Short video, streaming competition |
| Cloud scale and AI infra | Data centers, TPU, enterprise accounts | Cloud revenue, margin, backlog/RPO | AWS/Azure, specialized AI clouds |
| Ecosystem distribution | Android, Chrome, Play, Gmail, Maps | User scale, platform revenue | Antitrust remedies, mobile platform shifts |
| Technical talent / models | Gemini, DeepMind, TPU stack | Product adoption, cost/performance | OpenAI, Anthropic, open-source models |

## Key Moat Test

Can Alphabet sustain high returns on incremental capital after AI capex rises?

## Falsification

- [ ] Search revenue growth decelerates materially while AI usage rises.
- [ ] Cloud growth requires structurally low returns on capital.
- [ ] Antitrust remedies break distribution economics.
- [ ] AI assistants move commercial intent away from Google surfaces.
