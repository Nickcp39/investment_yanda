## Upstream → BSX → Downstream

**Upstream inputs**:
- Specialty catheter materials, electronic components (mapping/navigation systems), battery/implant-grade materials (Watchman, CRM devices); component tariff exposure is real (Section 301/232 layered China-sourced-component tariffs can push some effective device-component duty rates >40% per general medtech-sector context, not BSX-specific this run — facts.md E32/O6).
- Contract manufacturing for select disposable components; BSX guides to fully offsetting ~$200M of 2026 tariff costs via pricing/cost actions (search-derived, O-item, needs primary confirmation).

**BSX internal value-add**:
- R&D / device engineering (Farapulse non-thermal PFA technology, ~20 patents assigned from the Farapulse acquisition; OPAL HDx mapping hardware/software; Watchman FLX/FLX Pro generational improvements)
- Clinical trial infrastructure (AVANT GUARD, CHAMPION-AF, SIMPLAAFY — ongoing trials generating the evidence base that supports both regulatory expansion and physician adoption)
- Physician training/certification programs (Watchman implant training — didactic, imaging, patient selection, proctoring phases)
- Active, well-resourced M&A engine (Silk Road, Axonics, Bolt Medical, SoniVie, Elutia, Nalu, pending Penumbra) — BSX systematically acquires adjacent/complementary technology rather than relying purely on internal R&D

**Downstream customers**:
- Hospitals / cath labs / EP labs (capital equipment + procedural device buyers)
- Interventional cardiologists / electrophysiologists (the direct clinical decision-makers — physician relationships and training investments are a genuine, if soft, moat)
- Payers/reimbursement systems (procedure reimbursement rates directly gate volume growth — not quantified this run)
- Emerging Markets / China (bundled, not separately disclosed — O3)

## Where the Profit Pools Sit

| Value chain link | Profit pool depth | Notes |
|---|---|---|
| Farapulse PFA catheters (disposable) | **Deep but increasingly contested** | High-margin disposable, but the SHARE of this profit pool captured by BSX vs. Medtronic/J&J/Abbott has fallen sharply (100%→41% since 2023) — the pool itself is growing (market +20%/yr unit volume) but BSX's SLICE is shrinking |
| OPAL HDx mapping + FARAVIEW software | **Emerging, genuine lock-in** | Only 1-in-3 Farapulse accounts use the integrated system yet — real moat-in-formation, not yet a majority-adopted, fully-proven ecosystem lock-in |
| Watchman (concomitant) | **Deep, sticky** | Bundled into existing surgical relationships — likely the most durable sub-segment of the LAAC franchise |
| Watchman (standalone) | **Weakening** | Declining since Feb 2026; more exposed to new-patient acquisition, referral patterns, and hospital capital-spending cycles |
| Endoscopy | **Stable, moderate** | Steady mid-single-digit organic growth, least contested this run |
| Urology | **Weak currently** | Near-zero growth (+0.5% Q1 2026), one of three named guidance-cut drivers, root cause not disambiguated this run |
| Neuromodulation | **Growing, smallest** | M&A-augmented (Nalu), fastest MedSurg growth |

## Bottleneck Candidates (developed further in bottleneck_map.md)

1. **Physician training/certification (Watchman)** — genuine, unquantified switching-cost moat; real but evidently insufficient alone to prevent standalone-procedure deceleration once demand/competitive conditions shift.
2. **OPAL HDx / FARAVIEW software ecosystem lock-in** — the closest thing to a durable, non-price-based EP moat, but only ~33% penetrated within BSX's own Farapulse installed base — not yet a company-saving bottleneck.
3. **Regulatory approval cycles (FDA)** — genuine time-based moat (SEISMIQ IVL launch, Watchman FLX Pro trials, Farapulse generational approvals) but a MOVING target shared by all four PFA competitors, not a permanent BSX-specific barrier.
4. **Clinical evidence base (AVANT GUARD, CHAMPION-AF trial data)** — genuine differentiator if BSX's specific device shows superior long-term outcomes data (facts.md mentions FARAPULSE "greater long-term AF treatment success" and lower repeat-ablation rates at 4 years vs thermal ablation) — this is a real, evidence-based moat candidate that competitors' newer PFA entrants (Medtronic PulseSelect Jan 2024, J&J Varipulse Dec 2024, Abbott Volt Jan 2026) have not yet had time to generate comparable long-term data for, which could matter over a multi-year horizon even if near-term SHARE has already shifted.
