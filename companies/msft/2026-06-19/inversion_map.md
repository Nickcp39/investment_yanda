|---|---|---|---|---|

## 永久减值测试(资产负债表能否存活?)
- **能,稳健**:净现金 ~$47B、债务下降、OCF $170B TTM、持续 dividend。即便 bear(全 capex 当维护、Azure 减速),IRR ~breakeven 而非永久亏损。**无硬 veto**。

## 硬 veto 检查(5 铁律)
- thesis 耐久性破裂?**否** — 三引擎护城河 + RPO 确认需求。
- 资产负债表无法存活?**否** — 净现金。
- → **无硬 veto;F1(capex ROI)是前瞻不确定性 = size haircut 不是 veto**(这是修好的老病,不复发)。

## M5 信号:**−1(警告,非 veto)**
- 真实可存活风险:F1 capex ROI(主)+ F4 OpenAI 信息缺口。
- 比 GOOGL 的 M5 更轻:MSFT 有一股一票纠错杠杆 + 净现金 + RPO 需求确认 + 起始估值已 de-rate。

## Kill 标准(搬入 monitor.md)
- K1 [F1]: capex 持续升 + FCF 连续两年下降 + 无 incremental ROIC 回升证据 + 管理层仍无 ROI 框架。
- K2 [F3]: Azure 增速连续两季 ≤ +25% 且管理层归因需求(非汇率/产能)。
- K3 [F4]: OpenAI 关系/经济学出现对 Azure 重大负面口径变化或减值。
- K4 [F2]: AI run-rate 增速塌 + Copilot 续费/attach 明显走弱。
- K5 [估值]: 价格隐含 10y IRR < 8%(≈ 价格 > ~$436 base)。
