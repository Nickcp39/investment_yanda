# META Value Chain Map (M3) — as_of 2026-06-19

## 数字广告价值链中的位置
```
用户注意力(3.56B DAP) → 第一方数据 + AI 排序/推荐 → 广告拍卖/定价 → 广告主 ROI
        ↑META 拥有全链                                          ↑META 直接货币化
```
- **上游（用户/内容）**: META 自有平台（FB/IG/WhatsApp/Messenger/Threads），不依赖第三方分发；UGC + creator 生态供给内容。
- **核心（数据 + AI 排序）**: 第一方行为数据 + AI 推荐（Advantage+、Reels/视频排序）是定价权来源；AI capex 主要砸在这里。
- **下游（广告主）**: 数百万长尾 + 品牌广告主，自助拍卖，转化导向（DR 广告占比高）。

## 利润池归属
- META 捕获数字广告利润池的**最大份额之一**（与 GOOGL 双寡头）。Q1 单价 +12% 说明它在拍卖端有定价权、不是被中间商分润。
- **价值链脆弱点**: ①流量入口若被生成式 AI / agent 改写（用户不再刷 feed）→ 广告位价值下降；②设备层（Apple/Google OS）掌握隐私开关（ATT 先例）→ META 不控制最关键的归因数据通道，这是结构性依赖。

## AI capex 在价值链的作用（核心未决 O4）
- 管理层逻辑：capex → 更强 GPU 集群 → 更好排序/推荐模型 → 更高广告相关性与单价 → 增量营收。Q1 单价 +12% / 量 +19% 与该逻辑**方向一致**但**无一手量化**证明 $135B capex 的边际 ROI。
- 反面：capex 也可能投向 RL/"superintelligence" 等十年期权（FY25 RL 损失 ~$19B），与广告价值链复利无直接关系。
