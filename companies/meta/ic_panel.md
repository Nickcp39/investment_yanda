## 1. 五灵魂初评

### 1.1 Warren Buffett — 十年好生意？owner earnings？够便宜？
- **Verdict: `WATCH`**
- **理由**: 广告引擎是一门我能理解的好生意（双寡头、定价权 +12% 单价、48% 分部利润率）。但 owner earnings 这关出问题了：报表净利 $26.8B 被 $8.03B 一次性税收放大，而真正落袋的 FCF 只有 ~$49.6B 年化——差额被翻了 3.5 倍的 capex 吃掉。我付 ~20x forward，买的是"capex 全是复利"的假设，可这假设无一手量化证据。FCF yield 3.4% 对我不够便宜。
- **真实可考立场**: Buffett 1986 股东信定义 owner earnings（净利 + 折旧摊销 − 维护性 capex），并反复强调"增长 capex 若不产生超额回报就是价值毁灭"；要合理价的伟大公司，不为成长本身付溢价。
- **应用**: 这是"伟大引擎 + 不确定的重资本下注 + 不便宜的价格"。在 capex ROI 没证明、FCF yield 3.4% 的情况下，我等更便宜（~$480 有 MOS）→ WATCH 不建仓。

### 1.2 Charlie Munger — 败在哪？激励有毒？愚蠢风险？
- **Verdict: `WATCH`**
- **理由**: 反过来想——杀死这笔投资的最可能路径是 **capex 黑洞（F1）**：FY26 $135B capex 持续上调但营业利润不随之复利，ROIC 跌破资本成本。叠加**激励错配**: Zuckerberg 控制 ~61% 投票却只有 ~14% 经济权益——他承担 capex 错配的成本远小于外部股东，却完全主导决策，且 2026 股东提案全否、停回购全押 capex，**外部股东没有踩刹车的杠杆**。RL 烧了 ~$19B/年至今无兑现就是先例。
- **真实可考立场**: "Show me the incentive and I'll show you the outcome"、"Invert, always invert"、长期警告控制人无约束的资本错配与"为活动而活动"。
- **应用**: 这里激励**确实有毒倾向**（控制权 ≫ 经济权益 + 无外部约束 + 重资本期）。这不是说一定败，而是说**别在没有 MOS 时承担这个错配风险**。19.8x forward 没给我犯错空间 → WATCH。

### 1.3 段永平 — 真懂生意？用户价值？管理层？贵不贵？（叠不为清单 + 100% 私有化测试）
- **Verdict（初评，主审 §3 另出 final）: `WATCH`**
- **理由**: 生意我懂——广告 + 社交网络效应，用户价值真实（3.56B DAP 离不开）。管理层 Zuckerberg 是产品天才，2022 也证明过顶住华尔街压力做对事。但"贵不贵"这关过不了: base IRR +4% 不到 8% 门槛，FCF yield 3.4%——**现价没给安全边际**。而且 capex 这件事我不确定是复利还是黑洞（O4），不懂的部分恰好是决定 base 成败的部分。
- **真实可考立场**: "买股票就是买公司""不懂不做""平常心""不为清单（不做不懂的、不预测短期、不在无安全边际时追高）"。
- **应用**（叠两测试）:
  - **不为清单**: ①不预测 AI capex 短期能否兑现（属看不准的事）；②不因"AI/广告满血、大家都买"从众；③**不在无安全边际时付溢价**——而现价**没有**安全边际（base +4% < 门槛）→ **这条直接触发否决/等待**。
  - **100% 私有化测试**（用 $1.48T 买下、十年不卖，愿意吗？）——**能确信**: (a) 广告引擎现金创造真实、(b) 网络效应 + 定价权强、(c) 资产负债表死不了。**不能确信**: (d) $135B/年 capex 十年能否赚回资本成本（O4）、(e) 控制人在重资本无约束期会不会持续错配、(f) 这个价格（FCF yield 3.4%）私有化第一年现金回报够不够看。**多数生意质量能确信，但价格 + capex ROI + 控制人三项不能确信 → 私有化"愿意，但要更低的价"→ WATCH，等回撤到 ~$480。**

### 1.4 Howard Marks — 共识？周期/情绪位置？赔率？
- **Verdict: `WATCH`**
- **理由**: 共识是温和偏多——现价距高点只 −27%，市场基本接受了"AI capex 会赚回来"的叙事（否则不会给 20x forward）。这不是 NVDA 那种"市场已对周期见顶定了价"的偏空位置。赔率: bull +15% / base +4% / bear −10.5% → **赔率近乎对称偏下**（base 不过门槛），不是我要的不对称向上。
- **真实可考立场**: second-level thinking、"it's not what you buy, it's what you pay"、风险=永久损失、周期与心理决定赔率。
- **应用**: 在市场已接受乐观叙事 + base 不过门槛的位置，赔率没补偿不确定性。第二层思考: 大家都看广告满血，少有人盯 FCF 被 capex 吃光 + base IRR 不过门槛——这恰是该谨慎的位置 → WATCH，等更好赔率（回撤）。

### 1.5 Seth Klarman — 下行保护？安全边际真实吗？
- **Verdict: `WATCH`**
- **理由**: 下行保护一半好一半弱: 好的是资产负债表（净现金 ~$22.4B、OCF $32B/季）→ bear −42% 是 de-rate 非破产、可存活。弱的是**安全边际为负/无**: base IRR +4% < 8%、现价高出保守公允价 ~21%、FCF yield 仅 3.4% → 我没有"50 美分买 1 美元"的折扣，反而在为乐观情形付溢价。
- **真实可考立场**: margin of safety 第一、先看下行、cash as optionality、拒绝为充分定价资产 FOMO。
- **应用**: 资产负债表能护底（无破产风险），但**安全边际不存在**——这是充分定价（偏贵）的优质资产。我的纪律是不为它 FOMO → WATCH，cash 等回撤到有真实 MOS（~$480）。

## 2. 一轮 Critique

**① Marks → critique Buffett（反方：会不会太苛刻，错过伟大生意？）**:
"你盯 FCF yield 3.4% 说不便宜，可这是史上最强广告引擎之一，+33% 增长、单价 +12%。当年你也因为'看不懂'错过早期科技。capex 高是因为在抓 AI 这个时代机会——会不会你又在 too_small_missed_asymmetry？"
> **Buffett 回应（→ 维持 WATCH）**: 公允质疑，这正是框架要防的另一面（SNDK 教训）。所以我没说 REJECT，我说 WATCH——生意我认，是好生意。但区别在价格: NVDA 我给了 STARTER 因为 forward 19x + owner earnings 干净 + 价格站买方这边；META 同样 ~20x 但 owner earnings 被 capex 吃掉、base IRR +4% 不过门槛。我不是回避伟大生意，我是**等这门伟大生意回到有 MOS 的价**（~$480）。等待不是错过，是纪律。**→**

**② Buffett → critique 段永平（capex 到底是不是黑洞？）**:
"你把 capex ROI 当不确定（O4），可 Q1 单价 +12%、量 +19% 不就是 capex 在起作用的证据吗？也许 owner earnings 该用经营 NI $75B 锚、base IRR 其实 +10%，那现价就有 modest MOS 了。"
> **段永平回应（→ 维持 WATCH）**: 方向上 capex 确实在帮排序（单价涨是旁证），所以我不给 REJECT。但"方向有帮助"≠"$135B 边际 capex 赚回资本成本"——这是量级问题，没有一手量化（O4）。在能用保守锚也能用乐观锚、结论从 +4% 跳到 +10% 的情况下，**不懂的部分恰是胜负手 → 按平常心，宁可等量化证据 + 更低价，不为乐观假设付现价**。这跟 NVDA 不同：NVDA owner earnings = FCF 没有这个锚的二义性。**→**

**③ 段永平 → critique all（先问治理：控制人风险该多重？）**:
"你们都把双层股权 + 停回购全押 capex 当 haircut。可 brief 提醒过：2022 这个控制人结构验证为价值-正，他顶住华尔街做对了。是不是不该太重罚治理？"
> **Munger（→ WATCH）**: 同意不自动重罚——2022 确实证明他对。但 2022 是"砍成本"（纪律为正），现在是"全押 capex + 停回购"（纪律方向相反）。同一个无约束控制权，顺周期可以很好、逆周期也可能很糟。我不因治理 veto，我因**治理 + 无 MOS 叠加**而等待。**→**
> **Klarman（→ WATCH）**: 控制人风险翻译成"要求更大下行保护"——而现价没有提供（base +4%）。治理不是 veto，是"更需要 MOS"的理由，现价没给。**→**
> **Marks（→ WATCH）**: 我不重罚治理，我罚价格。2022 那次治理对了 + 价格 10.6x（深价值），赔率不对称向上；现在治理同样但价格 20x（无 MOS），赔率向下。差别在价格不在治理。**→**
> **Buffett（→ WATCH）**: within-band 处理，治理计 haircut 不 veto——但 haircut + 无 MOS = 等待。**→**

## 3. 段永平主审综合（Chair Synthesis）

### 3.1 不为清单 4 项
1. **不做不懂的**: capex ROI（$135B/年是复利还是黑洞）我看不到底（O4），且它是 base 成败手——故不重仓。
2. **不预测短期市场/不择时**: 不预测 META 几个月涨跌；只认价格——现价无 MOS（base +4% < 门槛）。
3. **不因从众买入**: 不因"广告满血 + AI 叙事 + 大家都买"建仓。
4. **不在无安全边际时付溢价**: 现价**没有**安全边际（保守锚高出公允价 ~21%）→ **这条直接触发"等"**。

### 3.2 100% 私有化测试结论
能确信: 广告引擎现金创造真实、网络效应 + 定价权强、资产负债表死不了。不能确信: $135B/年 capex 十年赚不赚回资本成本、控制人重资本无约束期的纪律、现价（FCF yield 3.4%）的第一年现金回报。**→ 用 $1.48T 私有化十年不卖，我愿意——但要更低的价（~$480 有 MOS 时），不是现价 → WATCH。**

### 3.3 Final Verdict

# **WATCH（新钱）· initial 0% · buy-below ~$480 · 等回撤 / 等 capex ROI 量化**

**一句话**: META 的广告引擎是一门满血的好生意（+33% 营收、单价 +12%、48% 分部利润率、网络效应 3.56B DAP）——生意质量从来不是问题。问题是 **(a) 现价没有安全边际**（base 5y IRR +4% < 8% 门槛、FCF yield 仅 3.4%、保守公允价 ~$480 现价高出 ~21%），**(b) owner earnings 被翻 3.5 倍的 capex 吃掉而 capex ROI 无一手量化**（O4，决定锚选 FCF 还是经营 NI、直接决定 base 过不过门槛），**(c) 控制人 ~61% 投票 / ~14% 经济 + 停回购全押 capex + 股东提案全否**（重资本无约束期的激励错配 haircut）。完整度 ~62% 封 ceiling 在 STARTER，价格进一步把现价实际 verdict 压到 **WATCH**。

- **个人 vs panel**: 五票一致 WATCH + 主审 WATCH。initial 0%；回撤到 ~$480（base 8% 门槛）或 capex ROI 量化兑现（解 O4、可上修锚）后再评 STARTER。
- **与 NVDA 对照**: 同一 panel、同一主审，NVDA → STARTER（价格友好 + owner earnings 干净），META → WATCH（价格偏贵 + owner earnings 被 capex 吃 + 控制人激励错配）。差别全在客观事实，不在叙事——证明框架没被"广告满血 + AI"叙事带跑。
- **与 GOOGL 对照**: 两家同型（数字广告双寡头 + capex 黑洞 + 价格封顶 → WATCH）。META base IRR +4% 略好于 GOOGL base −3%，广告引擎更强，但同样无 MOS → 同为 WATCH。

### 3.4 Kill / 升级 Criteria（主审精选，各给当前读数）
1. **K1 [价格触发，最直接]**: 回撤到 ~$480（base 8% 门槛）→ 可评 STARTER 建仓。当前 $577，距触发 −17%。
2. **K2 [capex ROI，决策胜负手 O4]**: 若一手披露/连续季度证明 capex 转化为营业利润复利（ROIC > 资本成本）→ 可上修 owner-earnings 锚到经营 NI → base IRR → ~+10% → 解封顶可评 STARTER。当前无量化。
3. **K3 [广告引擎质量证伪]**: 营收增速跌到个位数 **且** 单价转负 → thesis 核心受损 → 降到 REJECT 候选。当前 +33% / 单价 +12%，离触发远。
4. **K4 [capex 黑洞坐实]**: capex 继续上调但营业利润不增 → F1 成真 → 永久降耐久性，即便价格更低也先确认未结构破坏。
5. **K5 [流量入口/监管，唯一可升 veto]**: 生成式 AI 结构性改写流量入口（DAP/时长持续下滑）或反垄断拆分 → F2/F4 → thesis 耐久性破裂（exit 候选）。当前无证据。

### 3.5 监控触发器
- **价格触发**: 回撤到 ~$480–520 → 评估建仓 STARTER；> ~$640 → 进一步无 MOS，no-chase。
- **证据触发（可上修 verdict → 解封顶）**: ①capex ROI 量化兑现（解 O4）；②10-K 逐行 + proxy 补齐（完整度 →>80%，配合价格可评更高 verdict）。
- **证伪触发**: K3/K4/K5 任一 → 即便价格更低也先确认 thesis 未被结构性破坏。

## 4. 分歧说明（主审 vs 多数票）
