# Raw Source Notes

This folder holds original-source extracts and neutral notes.

Rules:

- Use one file per source or source group.
- Preserve exact source URL, date accessed, and source ID.
- Separate quoted/extracted facts from analyst comments.
- Do not write investment conclusions here.

Suggested file names:

```text
10k-YYYY.md
10q-YYYY-QX.md
proxy-YYYY.md
earnings-call-YYYY-QX.md
competitor-<ticker>-10k-YYYY.md
```
