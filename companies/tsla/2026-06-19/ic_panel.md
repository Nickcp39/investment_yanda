## 1. 五灵魂初评

### 1.1 Warren Buffett — 十年好生意？owner earnings？够便宜？
- **Verdict: `REJECT`**（能力圈外 + owner earnings 薄 + 价格离谱三连）
- **理由**: 三关全过不了。①**owner earnings**: DELIVERED owner earnings ~$5B 对 $1.41T 市值，起始 yield 0.35%——这是我见过最贵的之一；而且核心汽车利润率从 16.8% 塌到 4.6%，owner earnings 在**变薄**不是变厚。②**价格**: P/E 371x，付的全是对尚未存在的 robotaxi/Optimus 利润的预付款。③**能力圈**: 汽车我懂（资本密集、周期、竞争激烈、历来是糟糕生意），robotaxi/Optimus 的十年终局在我能力圈外。
- **真实可考立场**: Buffett 1986 股东信定义 owner earnings；长期评价汽车业为艰难生意（资本密集、竞争惨烈）；"price is what you pay, value is what you get"；宁等好球、不勉强挥棒（能力圈 + 安全边际）。
- **应用**: owner earnings 薄 + 0.35% 起始 yield + 汽车是难生意 + AI 终局看不懂 → 这是"既不够便宜、又不够懂"的双重出局。REJECT。

### 1.2 Charlie Munger — 败在哪？激励有毒？愚蠢风险？
- **Verdict: `REJECT`**（最大愚蠢风险 = 为故事付 371x、把未兑现期权当已实现）
- **理由**: 反过来想——杀死今天 $400 买入的不是公司破产（资产负债表 fortress），而是**为 NARRATIVE 付的溢价的永久性蒸发**: robotaxi/Optimus 落空或延后 + 估值向 DELIVERED 基本面回归 = 腰斩级 de-rate（base −17%、bear −27%）。激励上，Musk $1T 包**把目标锚在极端乐观市值**——这正是"为活动/愿景买单"而非为已兑现现金买单的诱惑。而汽车利润率在塌是硬数据。
- **真实可考立场**: "Show me the incentive..."、"Invert, always invert"、长期警告为故事/愿景支付高倍数、警惕把可能性当确定性。
- **应用**: 在 371x 为未兑现期权付钱 = 教科书级愚蠢风险。$1T 包激励虽对齐上行但把估值钉在 moonshot。REJECT——不是做空，是不碰。

### 1.3 段永平 — 真懂生意？用户价值？管理层？贵不贵？（叠不为清单 + 100% 私有化测试）
- **Verdict（初评，主审 §3 另出 final）: `REJECT`**
- **理由**: 用户价值这关 Tesla 部分过得去（车好、超充网络真实、能源储能有用）。**但"贵不贵"这关彻底过不去**——P/E 371x、起始 yield 0.35%、base IRR −17%，这是**完全没有安全边际**的价格。而且管理层（Musk）虽强但**分心 + 把公司估值人格化绑定在自己身上 + $1T 包 12% 稀释**。我反复讲"买股票就是买公司""贵不贵很重要"——这个价格我没法当成在合理价买一门好公司。
- **真实可考立场**: "买股票就是买公司""不懂不做""平常心""不为清单（不做空、不做不懂的、不预测短期、不在无安全边际时追）"；好公司也要算长期生意价值、价格很重要。
- **应用**（叠两测试）:
  - **不为清单**: ①**不做不懂的**——robotaxi/Optimus 十年终局我不懂；②**不预测短期**——不赌 narrative 何时兑现；③**不因从众/故事买入**——Tesla 是最强的故事股之一，恰恰要警惕；④**不在无安全边际时付溢价**——现价**无任何 MOS**，**这条直接否决**。
  - **100% 私有化测试**（用 $1.41T 买下、十年不卖，愿意吗？）——**几乎全是不能确信**: 不能确信(a) 汽车利润率能否止跌、(b) robotaxi/Optimus 能否成真利润、(c) 能源是不是真利润腿、(d) Musk 会不会继续分心、(e) 故事溢价会不会蒸发。**能确信的只有: 公司死不了（净现金 $35.5B）。** 但"死不了"不是"用 $1.41T 私有化划算"。**→ 用 $1.41T 私有化十年不卖？不愿意。这与 NVDA（多数能确信 + 价格合理 → 愿意小比例）形成最鲜明对照。** REJECT。

### 1.4 Howard Marks — 共识？周期/情绪位置？赔率？
- **Verdict: `REJECT`**（共识极度乐观、情绪在中上部、赔率向下不对称）
- **理由**: 第二层思考的核心问题: 价格里**已经定价了什么**？$400、距高点仅 −20%、P/E 371x——市场已经把 robotaxi 大规模 + Optimus 千万级 + FSD 软件经济**当成大概率事件定价了**。这不是悲观/便宜的位置，是**乐观/昂贵的位置**。赔率: 下行 −27% / 基准 −17% / 乐观上行也才 +5% → **向下不对称**。这正是 second-level thinking 要回避的——人人相信故事时，赔率最差。
- **真实可考立场**: second-level thinking、"it's not what you buy, it's what you pay"、风险=永久损失（为故事付溢价后的减值就是永久损失）、周期/心理决定赔率、最危险的是"everyone knows"的共识。
- **应用**: 共识 = "Tesla 是 AI/robotaxi 赢家"，已充分定价；赔率向下不对称。REJECT/WATCH——等市场对故事祛魅、价格回归 DELIVERED 基本面再说。

### 1.5 Seth Klarman — 下行保护？安全边际真实吗？
- **Verdict: `REJECT`**（无安全边际——这是反面教材）
- **理由**: 安全边际是我的第一原则，而**这里根本没有**: 起始 OE yield 0.35%、base IRR −17%、只有纯 NARRATIVE 的 moonshot 过门槛。下行保护？公司层面有（净现金 $35.5B 死不了），**但估值层面没有**——为 371x 的故事付钱，故事一旦打折，下行 −27% 是真实的永久损失，不是波动。这正是我警告的"为充分定价的优质/性感资产 FOMO"。
- **真实可考立场**: margin of safety 第一、先看下行、cash as optionality、拒绝为充分/过度定价的资产 FOMO、风险=永久损失非波动。
- **应用**: 公司死不了 ≠ 投资有安全边际。估值无 MOS + 赔率向下 → 这是教科书级"无安全边际"案例。REJECT。

## 2. 一轮 Critique

**① Marks → critique 全体（魔鬼代言：会不会 over-veto 真增长？）**:
"你们都 REJECT，可 robotaxi 真上线了、FSD 1.3M 订阅、能源 46.7GWh 翻倍——这些是 DELIVERED 不是纯故事。会不会重复 SNDK 老病，因为'前瞻不确定'就把一门真在转型的生意 over-veto 了？"
> **段永平回应（→ 维持 REJECT）**: 区别在**价格**。SNDK 老病是"因前瞻不确定而对**价格友好**的真瓶颈 over-veto"（NVDA 就被正确地没 over-veto）。这里前瞻不确定**叠加了无任何安全边际的价格**——robotaxi 上线但规模/单位经济未披露、Optimus 还产前，而我要付 371x。修好的规则是"前瞻不确定 = haircut 不 veto"——我没 veto（生意没破裂），我是**让价格封顶**。haircut 后的 DELIVERED 基本面对 $400 仍 base IRR −17%。这不是 over-veto，是价格纪律。**→**

**② Munger → critique Buffett（能力圈是不是逃避？）**:
"你说能力圈外，可你后来重仓 Apple。Tesla 也可以理解成'AI/自动化平台'，是不是该花功夫看懂而不是一句能力圈带过？"
> **Buffett 回应（→ 维持 REJECT）**: 即便我**完全看懂**robotaxi/Optimus 会成，按 bull（OE 增 8 倍到 $40B + 维持 45x）现价 5y IRR 也才 +5%，不到门槛。所以这次**不是懂不懂的问题，是价格的问题**——价格已经把"看懂且成功"都定进去了还不够 8%。Apple 我买在合理价（13x），Tesla 现价 371x，性质不同。REJECT 不靠能力圈这条腿也成立。**→**

**③ Klarman → critique Marks（赔率会不会算得太死？）**:
"你说赔率向下，可故事股的右尾很厚——moonshot +21%。完全不参与会不会错过非线性上行？"
> **Marks 回应（→ 维持 REJECT）**: 右尾厚我承认（所以记了 moonshot）。但 moonshot **无任何当前一手数据支撑**（Optimus 还产前、robotaxi 单位经济空白），它是"如果一切完美"。我的工作是认赔率不是认右尾——为 21% 的尾部承担 −27% 的基准/下行，且付着 371x，赔率不对称偏坏。**不参与不等于做空**——是 WATCH，等价格补偿这个右尾（≈ 价格 < $200 才有讨论空间）。**→**

## 3. 段永平主审综合（Chair Synthesis）

### 3.1 不为清单 4 项
1. **不做不懂的**: robotaxi/Optimus 十年终局我不懂；汽车我懂——是难生意且利润率在塌。
2. **不预测短期市场/不择时**: 不预测 Tesla 几个月涨跌、不赌 narrative 何时兑现。
3. **不因从众/故事买入**: Tesla 是当下最强的故事股之一——这恰恰是我最该警惕从众的地方。
4. **不在无安全边际时付溢价**: 现价 **base IRR −17%、起始 yield 0.35%、只有 moonshot 过门槛 → 无任何安全边际 → 这条直接否决。**

### 3.2 100% 私有化测试结论
能确信: 公司死不了（净现金 $35.5B）。**不能确信**: 汽车利润率止跌、robotaxi/Optimus 成真利润、能源真利润率、Musk 不分心、故事溢价不蒸发。**→ 用 $1.41T 私有化十年不卖，我不愿意。** 这与 NVDA（多数能确信 + 价格合理 → 愿意小比例参与）形成最鲜明对照。

### 3.3 Final Verdict

# **WATCH（新钱）· 0% · business_verdict = uncertain · existing_position_verdict = TRIM/HOLD（视成本）**

**一句话**: Tesla 是一门**核心汽车在收缩、利润率连续三年腰斩（16.8%→4.6%）、由尚未兑现的 robotaxi/Optimus NARRATIVE 撑起 $1.41T 极端估值（P/E 371x、起始 yield 0.35%）**的故事股——和 NVDA"价格友好的真瓶颈"恰好相反。资产负债表 fortress（净现金 $35.5B）让它**死不了，但死不了 ≠ 该买**。现价对 DELIVERED 基本面无任何安全边际（base 5y IRR −17%，只有纯 NARRATIVE 的 moonshot 过 8% 门槛）；叠加 Musk key-man + $1T 包 12% 稀释。按铁律**不让故事驱动 size → 新钱 0%**。这**不是硬 veto**（生意没破裂、按修好的规则前瞻不确定只 haircut 不否决），而是**价格 + 利润池收缩主导的 verdict 封顶**。

- **个人 vs panel**: 五票 REJECT + 主审 WATCH（panel 的 REJECT 是"现价不碰"，主审取 WATCH = "0% 但放监控、等价格回归 DELIVERED 基本面或 narrative 兑现再评"，措辞更精确）。
- **与 NVDA 对照**: NVDA 五票 STARTER-开仓（价格友好、owner earnings 干净）；TSLA 五票 REJECT / 主审 WATCH-0%（价格极端、owner earnings 薄且收缩、故事驱动）。同一框架、同一五灵魂、同一主审，结论相反——差别全在客观事实（估值起点 371x vs 19x、利润率塌 vs 干净 OE），证明框架既未被 robotaxi/AI 叙事带跑、也未对真增长（能源/robotaxi 已上线）over-veto。

### 3.4 Kill / 解封顶 Criteria（主审精选，各给当前读数）
1. **K1 [NARRATIVE 兑现/证伪]**: robotaxi 披露**正向单位经济 + 规模化运营** 或 Optimus 进入真规模生产并产生收入 → 可上修（NARRATIVE → DELIVERED）。当前: robotaxi 小、Optimus 产前 → 未兑现。
2. **K2 [核心利润池]**: 汽车营业利润率继续降（<4%）或交付继续两位数下滑 → 进一步证实收缩，封顶更紧。当前: 利润率 4.6%、交付 −9%。
3. **K3 [Musk key-man]**: Musk 离任/失能/重大声誉事件 → narrative 溢价崩（estimate 最快减值路径）。
4. **K4 [稀释]**: $1T 包里程碑解锁触发大额稀释 → 每股价值摊薄。
5. **K5 [价格解封顶]**: 价格回到对 DELIVERED 基本面有 MOS 的区间（base IRR 转正 ≈ < ~$160–200，深度 MOS < ~$130）→ 才进入 WATCH→STARTER 的讨论。当前 $400.49 远高于此。
6. **K6 [完整度]**: SEC 10-K/10-Q 逐行（分部利润）+ proxy + robotaxi 单位经济补齐 → 完整度升，但**只要价格无 MOS，仍封顶**。

### 3.5 监控触发器
- **价格触发**: 唯有显著回落（< ~$200，向 52 周低 $288 以下并继续）才进入"是否有 DELIVERED MOS"的讨论；现价不构成任何建仓理由。
- **证据触发（可上修 verdict → 解封顶）**: robotaxi 正向单位经济 + 规模化（K1）/ 能源分部高利润率被一手坐实 / 汽车利润率止跌回升。
- **证伪触发（封顶更紧/EXIT 现有仓）**: K2 利润率/交付继续塌 / K3 Musk 事件 / robotaxi/Optimus 重大延后。
