# TSLA Value Chain Map (M3) — as_of 2026-06-19

## 三条价值链中的位置
Tesla 横跨三条不同的价值链，各自捕获位/利润池差异极大：

### 1. 电动车价值链（DELIVERED，主体，收缩中）
```
[电池: 自产 4680 + 宁德/松下/LG] → [Tesla: 整车设计+制造+软件+超充网络] → [直营销售] → [车主 + FSD 订阅]
```
- **捕获位**: 垂直整合（自产电池/软件/直营/超充）曾是利润优势；但 **2022 后利润率从 16.8% 塌到 4.6%** → 议价/定价权在中国 EV（比亚迪等）价格战 + 传统车企电动化下被侵蚀。
- **超充网络**: 真实资产（NACS 成北美标准、对手付费接入）→ 是为数不多在加宽的护城河，但利润贡献小。

### 2. 能源价值链（DELIVERED，增长腿）
```
[电芯/系统] → [Tesla: Megapack/Powerwall 制造 + 软件(Autobidder)] → [公用事业/商业/住宅]
```
- **捕获位**: 储能系统集成 + 软件。FY25 储能 46.7GWh（翻倍+）→ 真增长。**但分部真实利润率 OPEN（O5）**——是真利润腿还是低毛利量增，无一手分部数据，不能假设高利润。

### 3. AI/自动化价值链（NARRATIVE，未兑现）
```
[FSD 数据飞轮 + Dojo/AI 芯片] → [robotaxi 网络 + Optimus 机器人] → [出行即服务 + 劳动力替代]
```
- **捕获位（声称）**: 自有数据 + 垂直整合 → 自动驾驶/机器人平台。**现实**: robotaxi 已上线但规模小、单位经济未披露；Optimus 产前。利润池**尚不存在于报表**。

## 利润池归属（DELIVERED）
- 汽车: 利润池在**萎缩**（营业利润 $4.36B FY25，利润率 4.6%）。
- 能源: 量在涨，利润率 OPEN。
- 监管积分: $380M/季并**结构性消退**（利润补贴在消失）。
- → **DELIVERED 总利润池薄（owner earnings ~$5B），对不起 $1.41T 市值。**

## 价值链风险
- **下游需求走弱**: 产>交、库存 27 天 → 终端需求软。
- **竞争位移**: 中国 EV + 传统车企电动化挤压份额与定价权（利润率塌的直接来源）。
- **AI 链的执行风险**: robotaxi 落后 Waymo 运营规模；Optimus 时间表反复推迟。价值链第 3 段是估值依据但**最不确定**。
