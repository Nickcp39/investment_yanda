# AMZN Value Chain Map (M3) — as_of 2026-06-19

## 三条价值链
### 1. 零售/物流（NA + International）
- 上游: 第一方采购 + 第三方卖家（marketplace）。
- 中游: AMZN 物流网络（仓储/配送/最后一公里）、Prime 会员、广告位。
- 下游: 消费者。AMZN 同时是平台 + 自营 + 物流 + 广告媒体——**多重抽佣**（佣金 + FBA 物流费 + 广告费 + Prime 订阅）。
- 利润迁移: 低毛利自营零售 → 高毛利第三方服务费 + 广告。FY25 NA OI $29.6B（利润率回升）。

### 2. AWS 云（利润核心）
- 上游: 芯片（NVDA GPU + 自研 Trainium/Inferentia + Annapurna）、服务器/数据中心建设、电力。
- 中游: AWS 计算/存储/数据库/AI 平台（Bedrock）。
- 下游: 企业/初创/政府客户。**AWS 捕获西方第二大云利润池（FY25 OI $45.6B）**。
- AI 层: 自研 Trainium（降低对 NVDA 依赖 + 成本优势）+ Anthropic 战略投资（模型层卡位）。

### 3. 广告（嵌入零售段，高利润）
- AMZN 是零售媒体网络龙头；广告嵌在 NA/Intl OI 里，是零售利润率回升的高利驱动（PR 未单列，OPEN）。

## 利润池在哪里
- **AWS 是利润中枢**: FY25 AWS OI $45.6B / 总 OI $80.0B = **57%**（Q1'26 59%）。零售段贡献营收规模 + 现金流，利润率薄但回升。广告是嵌入式高利层。
- 价值链最稀缺环节: **AWS 的规模 + 转换成本 + 全栈（含自研芯片）**，以及零售的**配送网络密度 + Prime 飞轮**。
- **当前 capex 流向**: ~$200B/年（主要 AWS 数据中心 + AI 芯片）——AMZN 正用整条价值链的现金流（+ 加杠杆）押注 AI 工作负载的未来利润池。
