## 1. 五灵魂初评

### 1.1 Warren Buffett — 十年好生意？owner earnings？够便宜？
- **Verdict: `WATCH`**（好生意，但 owner earnings 看不清 + 价格不便宜）
- **理由**: AMZN 是好生意——AWS 利润池耐久、零售利润率创纪录。但我估值看的是 **owner earnings，而这门生意现在 owner earnings 看不清**: FCF 近零（$1.2B TTM），全被 ~$200B capex 吞掉。我得问"这 capex 是维护性还是会创造价值"——若是又一次 AWS 式逆向投资，正常化 owner earnings 远高；若赚不回资本成本，是价值毁灭。我**算不清这个桥**（O4）。即便给足信用按 NOPAT 算，起始收益率 2.5%、base IRR 负——**对一门我看不清 owner earnings 的生意付 34x，我不干**。
- **真实可考立场**: 1986 股东信定义 owner earnings；强调宁要合理价的伟大公司、能力圈、对看不清现金的生意保持距离。
- **应用**: 生意过关，**owner earnings 这关因 capex 桥不闭合 + 价格这关因起始收益率太低**，两关都没过 → WATCH，等价格或等 capex ROI 兑现。

### 1.2 Charlie Munger — 败在哪？激励有毒？愚蠢风险？
- **Verdict: `WATCH`**（治理清洁，但最大愚蠢风险 = 在 capex ROI 未证时按完美定价付钱）
- **理由**: 反过来想——杀死这笔投资的不是生意打输，而是 **①$200B/年 capex ROIC 低于资本成本（F1，价值毁灭）+ ②在这个不确定上按 34x / 起始收益率 2.5% 付完美价**。治理是干净的（one-share-one-vote、创始人在场、长期复利记录强），激励不毒。真正的愚蠢是**在重投资期 owner earnings 模糊时还付高价**——这正是 Mag7 叙事最容易让人犯的错。
- **真实可考立场**: "Show me the incentive..."、"Invert, always invert"、警告为高质量资产付过高价、警告资本错配。
- **应用**: 激励/治理无毒，但"在 capex ROI 未证 + 价格无 MOS 时买入"是可避免的愚蠢 → WATCH。等 capex 开始释放 FCF 或价格回落。

### 1.3 段永平 — 真懂生意？用户价值？管理层？贵不贵？（叠不为清单 + 100% 私有化测试）
- **Verdict（初评，主审 §3 另出 final）: `WATCH`**
- **理由**: 用户价值极清楚（电商 + 云刚需）、管理层靠谱（Jassy + Bezos、长期主义）。但"贵不贵"这关——现价处 52 周区间中上部、起始收益率 2.1–2.5%、base IRR 负——**价格不站在我这边**（与 NVDA 相反）。而且这次有个额外问题: **生意正在重投资期，owner earnings 我算不清**——$200B capex 是豪赌，赌赢是又一个 AWS，赌输是烧钱。**我不懂的部分 + 价格无边际 → 平常心，先看不动。**
- **真实可考立场**: "买股票就是买公司""不懂不做""平常心""不为清单（不做空、不做不懂的、不预测短期）"；好公司也要好价格。
- **应用**（叠两测试）:
  - **不为清单**: ①不预测 AI capex 周期/AWS 短期增速；②不因"Mag7/AI 赢家大家都买"从众；③**不在无安全边际时付溢价——现价无边际（base IRR 负），触发这条否决性观望**。
  - **100% 私有化测试**（用 $2.66T 买下、十年不卖，愿意吗？）——**能确信(a) 生意十年还在、(b) AWS 利润池耐久、(c) 管理层靠谱；不能确信(d) $200B/年 capex 十年累计能否赚回资本成本、(e) 用 $2.66T 买下、起始第一年现金回报仅 2% 出头、且这点现金还在被 capex 吞——划不来**。私有化测试**不通过**（不是生意不好，是这个价 + 这个 capex 不确定下，私有化不划算）→ WATCH。

### 1.4 Howard Marks — 共识？周期/情绪位置？赔率？
- **Verdict: `WATCH`**（共识偏多、估值在区间中上、赔率向下不对称）
- **理由**: 现价距 52 周高仅 −11%、处区间中上部 [S004]——**市场已给 AWS 重新加速 + AI 故事相当乐观的定价**（与 NVDA/GOOGL 贴 52 周低、情绪偏空相反）。第二层思考: 当共识为"AMZN 的 AI capex 必将兑现"时，赔率 bull +7.7% / base −1.5% / bear −13.5% 是**向下不对称**——上行勉强够门槛，下行实在。这不是补偿风险的位置。
- **真实可考立场**: second-level thinking、"it's not what you buy, it's what you pay"、风险=永久损失、周期与心理决定赔率。
- **应用**: 在情绪偏多 + 估值区间中上的位置，赔率不补偿 capex ROI 的不确定 → WATCH，等更好赔率（价格回落或证据兑现）。

### 1.5 Seth Klarman — 下行保护？安全边际真实吗？
- **Verdict: `WATCH`**（资产负债表可存活，但安全边际为负 + 加杠杆削弱缓冲）
- **理由**: 下行保护看三处: ①资产负债表能存活（OCF $148.5B、净现金（金融债）仍正 ~+$24B）——这关过；②**但安全边际为负**: 起始收益率 2.1–2.5%、base IRR 负，现价是溢价不是折扣，**不是 50 美分买 1 美元，是 1.3 美元买 1 美元**；③加杠杆（长债季增 $53B）+ capex 吞 FCF **正在削弱下行缓冲**。下行能存活 ≠ 有安全边际。
- **真实可考立场**: 先看下行、margin of safety 第一、cash as optionality、拒绝为充分/过度定价的优质资产 FOMO。
- **应用**: 这是**充分到过度定价**的优质资产，安全边际为负 → 明确 WATCH，等价格给出真实折扣（~$175 以下）。

## 2. 一轮 Critique

**① Marks → critique Buffett**:
"你说 owner earnings 看不清就 WATCH，可如果这 $200B capex 真是又一个 AWS（多年后释放巨额 FCF），你现在 WATCH 会不会重蹈'看不懂科技而错过'的覆辙？这是不是 too_small_missed_asymmetry？"
> **Buffett 回应（→ 维持 WATCH）**: 公允质疑，这正是修好的老病（别因前瞻不确定就 over-veto）。所以我给 WATCH 不给 REJECT——我不否定生意，我**等**。但区别在: NVDA 是"价格友好 + 看不清份额"，我用 starter 参与；AMZN 是"价格无边际 + 看不清 owner earnings"，付高价参与一个我算不清的桥不是不对称的好赌注，是付钱赌。**等价格或等 capex ROI 兑现，再升 STARTER**。**→**

**② Klarman → critique 段永平**:
"你说价格无边际所以观望，但 AMZN 历史上每次重投资期都'看起来贵'，事后都证明便宜。你的 WATCH 会不会是用静态估值否定一个动态复利机器？"
> **段永平 回应（→ 维持 WATCH）**: 我认这个生意是复利机器——所以我没说 REJECT，没说做空。但"平常心"是: 我不预测这次 capex 一定像 AWS 那样赢，也不在起始收益率 2% 出头时为这个可能性付满价。**懂的部分我给生意高分，不懂的部分（capex ROI）+ 价格无边际 → 我等**。这不是否定，是纪律。**→**

**③ 段永平 → critique all（先问懂不懂）**:
"你们都说生意好——可有谁能说清 $200B/年 capex 十年累计的 ROIC？AWS 份额五年后被 Azure(OpenAI)/GCP/客户自研削成什么样？说不清就别假装算得出 owner earnings，更别在区间中上部的价格上假装有边际。"
> **Buffett（→ WATCH）**: 同意，owner earnings 桥不闭合是我 WATCH 的核心。**→**
> **Munger（→ WATCH）**: 不懂十年 capex ROI + 付完美价 = 可避免的愚蠢。**→**
> **Klarman（→ WATCH）**: "说不清"翻译成"要求安全边际"——而现价安全边际为负，所以 WATCH 而非 STARTER。**→**
> **Marks（→ WATCH）**: 我不需要懂终局，我需要价格补偿"不懂"——区间中上部的价格不补偿。**→**

## 3. 段永平主审综合（Chair Synthesis）

### 3.1 不为清单 4 项
1. **不做不懂的**: $200B/年 capex 十年 ROIC + AWS 五年份额我看不到底——生意底层（电商 + 云刚需）我懂，但 owner earnings 桥我算不清，故不在高价建仓。
2. **不预测短期市场/不择时**: 不预测 AMZN 几个月涨跌；只认价格——现价无边际就先不动。
3. **不因从众买入**: 不因"Mag7/AI 赢家"从众。
4. **不在无安全边际时付溢价**: 现价**无**边际（起始收益率 2.1–2.5%、base IRR 负）→ **触发否决性观望**（与 NVDA 相反，与 GOOGL 一致）。

### 3.2 100% 私有化测试结论
能确信: 生意十年还在、AWS 利润池耐久、管理层靠谱。不能确信: $200B/年 capex 累计 ROIC、AWS 五年份额、当前 owner earnings 真实水平。**→ 用 $2.66T 私有化十年不卖、起始第一年现金回报仅 2% 出头且还在被 capex 吞、capex ROI 未证——我不愿意。私有化测试不通过 → WATCH。**

### 3.3 Final Verdict

# **WATCH（新钱）· size 0% · buy-below ~$175–180**

**一句话**: AMZN 是一门好生意——AWS 重新加速 +28%、零售利润率创纪录 13.1%、利润池耐久——但**和 NVDA"价格友好"恰好相反，这次价格不站在买方这边**: 现价 $244.39 处 52 周区间中上部、起始正常化收益率仅 2.1–2.5%、即便给足 $200B capex 信用的 NOPAT 口径 base 5y IRR 也约 −1.5%（bull 才 +7.7%，<8% 门槛）。叠加 owner earnings 桥因 $200B/年 AI capex 不闭合（FCF 近零、维护/增长拆分缺一手 O4）、GAAP 净利被 Anthropic 收益失真、长债季增 $53B。**这是"好生意 + 错价格 + 重投资期 owner earnings 模糊"，与 GOOGL 同构。** 按规则价格封顶 WATCH（非 REJECT——生意没破裂、资产负债表能存活），留 STARTER 升档路径。

- **个人 vs panel**: 五票 WATCH + 主审 WATCH。size 0%；价格跌破 ~$175（base IRR 回 8% 门槛）或 capex 开始释放 FCF + ROIC 兑现 → 解封顶、可评 STARTER。
- **与 NVDA 对照**: NVDA 完整度封顶 STARTER-但开仓（价格友好）；AMZN 价格封顶 WATCH-0%。差别是估值起点（起始收益率 5.1% vs 2.1–2.5%）与 owner earnings 质量（干净 FCF≈OE vs FCF 近零、桥不闭合）。
- **与 GOOGL 对照**: 同为价格封顶 WATCH-0% + capex 黑洞——AMZN 的 FCF 更极端（近零）、价格相对自身区间更贵（中上部 vs GOOGL 贴低）。

### 3.4 Kill / 升档 Criteria（主审精选，各给当前读数）
1. **K1 [capex ROI，最强单一信号]**: 连续多季 capex 大增但 AWS 增量营收/利润不匹配（ROIC 恶化）→ F1 价值毁灭。当前 AWS +28% 重新加速，暂未恶化但 capex 强度极高。
2. **K2 [AWS 份额]**: AWS 增速结构性回落到 <15% 且份额走弱 → F2。当前 +28%（15 季最快），离触发远。
3. **K3 [零售利润率]**: 总营业利润率从 13.1% 显著回吐（<9%）→ F3。当前创纪录高。
4. **K4 [资产负债表弹性]**: 净债务恶化 + OCF 增长停滞 → F4。当前 OCF $148.5B、净现金（金融债）仍正。
5. **K5 [估值升档/no-chase]**: 价格跌破 **~$175**（base IRR 回 8%）→ **解 WATCH 封顶、可评 STARTER**；价格上行（base IRR 更负）→ 维持 no-chase。现价 $244.39 远高于 $175。
6. **K6 [GAAP 失真]**: Anthropic 公允价值反向波动致 GAAP 净利大跌 → 估值口径事件（非经营，监控）。

### 3.5 监控触发器
- **价格触发**: 跌破 ~$175 → 升档 STARTER 候选；区间中上部/上行 → no-chase。
- **证据触发（可上修 verdict → 解 WATCH 封顶）**: ①capex 开始释放 FCF（capex 强度回落 + FCF 回升）证明投资期接近兑现；②10-K 逐行披露 capex 维护/增长拆分 + 增量 ROIC（解 O4）；③AWS 连续兑现 + backlog 转化为营收。
- **证伪触发**: K1/K2/K3 任一 → 重估 thesis 耐久性。
