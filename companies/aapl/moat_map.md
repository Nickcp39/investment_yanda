# AAPL Moat Map (Stage 4D / M3) — as_of 2026-06-19

## 护城河来源（按强度）
1. **生态切换成本 + 网络效应（最强、最耐久）**: iMessage/FaceTime/iCloud/App 库/Watch-iPhone 配对/家庭共享 → 用户深锁。20 亿+ 装机量，留存极高（公司称"所有品类创装机量新高"，S001）。
2. **品牌与定价权**: 季度 GM 49.3%（S001）证明定价权未流失；高端价位带几乎垄断。
3. **自研 SoC（A/M 系列）**: 性能/能效领先 → 产品差异化 + 成本控制，竞品难复制。
4. **App Store 双边平台**: 开发者必须来，用户离不开 → 抽租权（30%/15%），但受监管侵蚀（DMA/反垄断）。
5. **规模采购**: 把供应链利润压薄。

## 耐久性评级: **HIGH**
- 装机量年金 + 切换成本是结构性、多年坐实的（不是故事）。Services +13.5% FY25 / +16.3% 季度 = 抽租机器仍在加速。
- 唯一**正在被侵蚀**的边: App Store 抽租受全球反垄断/DMA 压力 + Google TAC 受美国搜索反垄断救济威胁（O1）。这是**利润率边际侵蚀**，不是装机量护城河崩坏。

## 与 NVDA 对照
- NVDA = 瓶颈型护城河（CUDA + 全栈），周期性强，size 封 Confirmed。
- AAPL = **年金型护城河（装机量 + 生态锁定）**，周期性弱（消费但黏性高），耐久性更适合 Core 级 size 天花板 —— 但**完整度 + 价格**会另行封顶 verdict 与 size。
