# AAPL Value Chain Map (Stage 4C / M3) — as_of 2026-06-19

## 价值链位置
Apple 是**集成商 + 生态平台**，把上游组件/代工的利润压到最薄，把下游用户关系/服务抽租做到最厚。

```
[组件] Nvidia/TSMC(自研 A/M 芯片代工)·三星/LG/京东方(屏)·SK海力士/美光(存储)·高通/博通(基带/射频)
   ↓ Apple 设计自研 SoC + 强势采购（压价 + 多源）
[代工] 富士康/和硕/立讯 — 利润薄、Apple 强势
   ↓
[Apple 集成 + 品牌 + iOS + App Store] ← 利润池在这里
   ↓
[2B+ 装机量用户] → Services 抽租（App Store 30%/15%、Google TAC、广告、iCloud、AppleCare、Pay）
```

## 利润捕获
- **上游**: Apple 用规模 + 自研 SoC 把组件商变成可替代供应商（高通除外，正逐步自研基带）。供应商利润薄。
- **中游集成**: Apple 拿走绝大部分硬件利润（GM 49% 季度）。
- **下游 Services**: ~70% 毛利的二次抽租，是利润质量的上行来源。

## 关键依赖（链上风险）
- **TSMC 制程**: 先进芯片单一关键依赖（地缘 + 产能）。
- **中国制造集群**: 仍是主产地；关税/迁移是前瞻成本风险（OPEN O6）。
- **Google TAC**: Services 中一块高毛利无成本收入（~$20B/yr）依赖 Google 默认搜索协议 → 反垄断救济直接威胁（OPEN O1）。
- **Nvidia/Google 云（AI）**: WWDC 显示 Apple 前沿 AI 模型部分跑在 Google 云的 Nvidia GPU 上 → AI 层 Apple 不是独立链主（解读）。
