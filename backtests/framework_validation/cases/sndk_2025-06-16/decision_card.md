## 4. Main Reason To Own

SNDK was priced like a weak, recently spun-off memory cyclical while pre-as-of evidence already showed supply discipline, price increases, and AI / enterprise SSD demand that could improve normalized earnings power.

## 5. Main Reason Not To Own

NAND is cyclical, competitive, and capital-intensive. Current financials had not yet proven the turn. Flash Ventures / Kioxia obligations and fixed costs can turn a false cycle recovery into real downside.

## 6. Binding Constraint

This is not a quality compounder core position. The binding constraint is evidence confirmation: gross margin, enterprise/cloud demand, and inventory / utilization must improve before size increases.

## 7. Add / Trim / Exit Rules

- Add if Q4 FY2025 revenue is at or above guide midpoint and non-GAAP gross margin reaches the guided range.
- Add if management confirms continued price increases and supply discipline.
- Add if Cloud / enterprise revenue growth becomes both sequential and year-over-year positive.
- Do not add if price rises sharply without evidence.
- Trim if position exceeds risk budget before margin confirmation.
- Exit if Q4 guide fails, margin reverses, or balance-sheet / Flash Ventures obligations become the core story.
