# S3: Precision Medicine / Diagnostics

## 段级洞察先放前面

**这一段和其他段的根本区别：商业化已经彻底成立**

- Tempus AI 2025 营收 $1.3B (+83% YoY)，2026 guidance $1.59B (+25%)
- Guardant Health 2025 营收 $982M (+33% YoY)
- Caris 2024 营收 $412M (+35% YoY)，IPO 2025-06

**段头部 3 家合计年营收 > $2.5B**，比所有其他 healthcare AI 段（S1+S2+S4+S5+S6）加起来还多。原因：

1. **付费方明确** — 保险公司（Medicare CPT codes）+ pharma (data deals) 双付费源
2. **诊断本质是"必需消费"** — 癌症患者必须做基因分型决定用药，不是"nice to have"
3. **AI 是 enabling tech 不是 core product** — S3 公司不是"AI startup"，是"分子诊断公司 with AI 加速"，所以不被 AI 估值泡沫拖累，反而被实业现金流支撑
4. **regulatory + reimbursement moat 完整** — Foundation Medicine、Guardant360 都有 CPT code 和保险报销路径

**对 Nick 的核心含义:** S3 是**唯一一个 "AI 是 enabling 而非 substance" 的段**。这恰恰也是"中国医疗设备出海"的本质——设备是 substance，AI 是 enabling。S3 和 S5 (imaging) 在商业逻辑上比和其他 AI 医疗段更接近。**S3 + S5 应该作为 Nick 主线的并列候选评估，不只是 S5。**

## 段内深度分析

### 商业模式：三种 archetype

| 模式 | 例子 | 付费方 | 单测价格 (USD) | 报销状态 |
|------|------|--------|---------------|---------|
| **组织 NGS panel** | Foundation Medicine F1CDx, Caris, Tempus xT | 保险/Medicare | $3,000-5,000 | 全 CPT code，全报销 |
| **液体活检 ctDNA** | Guardant360, Tempus xF, Signatera (Natera) | 保险 | $5,000-8,000 | 大部分报销 |
| **多癌早筛 (MCED)** | Galleri (GRAIL), Shield (Guardant) | self-pay + payer pilots | $949 (Galleri), $1,495 (Shield Medicare) | 部分报销，扩展中 |

**关键洞察:** MCED 是这一段下一个 inflection point — 一旦 Shield (Guardant) 的 Medicare 报销扩展到所有 50-75 岁人群，TAM 是 $20B+。这也是 Guardant 2025 股价波动剧烈的原因。

### Tempus AI 商业模式 deep dive（值得作为 case study）

Tempus 不是传统的"AI 公司"，它是 **AI-enabled clinical diagnostics + healthcare data platform** 的组合：

**Revenue mix:**
- Diagnostics ($955M, 73%): 卖测试给医生/医院，按 test 收费
- Data & Apps ($316M, 24%): 卖 de-identified clinical + genomic data 给药企做临床试验招募 + 真实世界证据
- Hardware/Other (~$30M, 2%): 仪器和耗材

**为什么 AI 是 enabling 不是 product:**
- AI 用于：NGS panel 分析、bioinformatics pipeline、临床决策辅助、患者匹配试验
- 但**付费的不是 AI**，付费的是 test result + 数据
- 这种结构让 Tempus 在"AI 泡沫破灭"时仍然有 $1.3B 实业现金流支撑

**对 Nick 中国 thesis 的启示:**
- **同样的结构在中国对应公司里成立**：燃石医学、世和基因、华大智造的核心营收都是 test fee，AI 是 enabling
- 这种公司的**出海路径**：通过 CAP/CLIA 认证、与海外医院 partnership、获取海外 reimbursement 通道
- 燃石 2020 NASDAQ IPO（已退市美股 2023，回归 HK）→ 中国精准医疗公司的美股退市潮，但 HK 上市 + 东南亚出海是新模式

### Moat 分析

| Moat 类型 | 例子 | 强度 |
|----------|------|------|
| **CPT code + payer 网络** | Foundation Medicine, Guardant | **极强** (5+ 年建立) |
| **大型生物库 + ML model** | Tempus (clinical + genomic + RWE) | 强（数据规模 moat）|
| **配套设备 + 仪器** | Foundation (Roche 仪器), GRAIL (Illumina Galleri 仪器) | 中-强 |
| **临床试验 partnership** | Tempus + 200+ academic centers, Foundation + biopharma | 强 |
| **Brand + 医生认知** | Foundation (10+ 年累积), Guardant360 | 中-强 |
| **regulatory FDA** | Galleri 在 FDA 审批中 | 通过后强 |

### 风险信号

- **MCED （多癌早筛）的 false positive 问题** — Galleri 等公司被批评 sensitivity 过低，假阴性可能 delay 诊断。FDA 监管态度可能转严
- **AI-only 公司无法与 Tempus 这种垂直整合竞争** — 任何想"卖 AI 给 Tempus 用"的 startup 都会被 Tempus 内部建团队替代
- **中国 NGS 公司美股退市潮** — 燃石、华大基因、贝瑞和康都退市了，HK 上市重启路径但流动性差
- **Foundation Medicine 被 Roche 全收购后增长放缓** — 大药企控股可能让独立 partnership 变窄
- **Exact Sciences 这种"诊断公司里 AI 比例小"会被分类争议** — 它该不该被算作 "AI healthcare" 段？严格说不是 AI-native，是分子诊断 + AI 工具

### 中国可映射性: **极高**（且有出海雏形）

| 美国公司 | 中国对应 | 出海状态 |
|---------|----------|---------|
| Tempus AI | 燃石医学 (Burning Rock)、世和基因 (Genetron)、华大智造 (MGI) | MGI 已出海 (东南亚/欧洲)；燃石/世和探索中 |
| Guardant Health | 燃石 (液体活检) | 燃石美股退市 2023 回 HK；东南亚试点 |
| Foundation Medicine | 燃石 OncoScreen Plus、世和 LungPlus | 国内 NMPA 批，未出海 |
| GRAIL Galleri | 鹍远基因（多癌早筛）| 国内 pilot |
| Natera | 贝瑞和康 (产前)、华大基因 | 贝瑞美股退市；华大全球化 |
| Exact Sciences Cologuard | 诺辉健康 (Cologuard 中国分销 + 自研常卫清)| 诺辉国内主导，美股 IPO 失败 |

**关键发现:** 中国 precision medicine 公司**几乎全部经历了"美股 IPO → 退市 → 回 HK"的周期**（2020-2024）。这是地缘政治 + 中概股 audit 危机的副作用。**现在正在出现新模式:** HK 上市 + 东南亚 / 中东 / 拉美 partnerships（华大智造领跑），这正好和 Nick 的"中国医疗设备出海"叙事吻合。

**Nick entry paths (S3):**
1. **华大智造 (MGI) 海外业务** —— effort: 中。MGI 是国内最国际化的精准医疗设备厂，深圳 HQ + 多个海外办公室。你的双语 + 海外生活 + PhD 在 MGI 国际业务部门估值很高
2. **燃石医学海外 / 出海 BD 团队** —— effort: 中。燃石是国内 NGS 头部，需要海外推广人才
3. **联影智能 / 联影集团跨界 partnership 与精准医疗厂** —— effort: 中。联影做 imaging 但 partner Tempus-style data play 可能开新岗
4. **直接加入 Tempus / Guardant 中国办公室** —— effort: 高且不确定。美国精准医疗大厂在中国有商业团队但研发岗稀缺
5. **建议:** S3 应该作为 **S5 imaging 的并列主线候选**，特别是 **MGI 海外业务**。MGI 是"中国精准医疗设备出海"最成功标杆，和你 imaging 出海叙事高度同构

## Nick 视角：S3 段对你职业决策的含义

**简短结论:** S3 不在你原 candidate set 里，但应该被**升级为 S5 同级别的主线候选**，特别因为 **MGI 华大智造** 的存在。

理由：
- S3 是商业化最成立的段（远超其他 AI 医疗段）
- S3 的"AI 是 enabling"模式和 S5 imaging device 一致
- 中国精准医疗设备厂（MGI 领跑）已经走在出海前列，比 imaging AI 厂（联影/数坤）出海更早
- MGI 的国际业务部门对你 profile 估值很高（双语 + PhD + 海外）
- 上海 / 深圳办公室距离家族业务（北京）兼容

**建议给 Phase 4 elevator-map 的位置:**
- 主线选项 A: 联影智能 海外业务（S5 imaging）
- 主线选项 B: **华大智造 海外业务（S3 precision medicine）← 此次发现新增**
- 备选: GPS 中国 AI 部门（防守路径）
- Plan B: 数坤/推想/迈瑞 国内研发岗（保底）
