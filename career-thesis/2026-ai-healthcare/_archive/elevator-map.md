## 3. Stage 1 (29-32 岁)：入局 — 接下来 12-24 个月的具体行动

### 主线候选 (按优先级 + 推荐序)

#### ⭐⭐⭐ 主线 A: 联影智能 海外业务（S5 imaging）

**为什么 A:** 你的 PhD + PA imaging + DL 在这里 perfect match。联影智能刚拿 Series A $100亿估值（2025-06），处于扩张期。16 NMPA 三类证（行业第一）。上海 HQ 兼容北京家族业务。出海正在 planning 期，你可作为早期海外 BD/PM 介入。

**Entry path 候选 (engineer + PM 并列):**
- engineer/research scientist：算法工程师，专门做影像（你的方向：PA / multimodal / foundation model for imaging）
- clinical PM：与影像科医生 partner 做产品定义
- international BD：海外业务部门 — 你的双语 + 海外经验是稀缺资源

**Effort 估计:** 中。需要联影集团内部网络（Jun Xia lab 校友？PA imaging 圈？）做内推。

#### ⭐⭐⭐ 主线 A2: 联影医疗（母公司）国际业务 / 联影集团其他子公司

**为什么 A2:** 联影集团是中国唯一全栈 imaging device 厂（CT/MR/分子影像/超声）。A 股 $15B 市值，已 active 出海到 SEA/MEA/EU/LATAM。比联影智能更稳定 + 更国际化。

**Entry path:** international business 部门，可能岗位包括海外产品经理、海外区域销售、海外注册专员。

#### ⭐⭐⭐ 主线 B: MGI 华大智造 国际业务（S3 precision medicine）

**为什么 B:** MGI 是中国精准医疗设备厂出海最成功标杆 — 76 国 / 1080+ 客户 / 海外营收占 33%。和你"中国设备厂 + AI enabling + 出海"叙事完全 align。MGI Americas / MGI Europe / MGI LatAm 都在 active hiring。

**Risk signal:** 2025 H1 海外营收 YoY **-9%**（Trump 关税 + 中美脱钩），刚卖出 US CGI 子公司 $50M 给瑞士 Swiss Rockets。这是"中国设备厂出海" thesis 的 stress test — 但 MGI 的反应是技术留 + 风险出 + 转向 SEA/MEA/Africa 等非美市场。这反而印证了你 thesis 的弹性 valid。

**Entry path:** 深圳 HQ 国际业务部门 + 海外分公司 (Americas / Europe / LATAM)。你的 PhD + 双语 + 海外生活在 MGI 国际业务部门估值极高。

#### ⭐⭐ 主线 C: GPS 中国 AI 部门 (defensive path)

**为什么 C:** 外资稳定性 + 双合规经验 (FDA + NMPA) + 35+ 安全边际。但天花板被中国分公司的"VP of China R&D"封顶。

**Entry path:** GE Healthcare 上海 / Philips 苏州 / Siemens Healthineers 上海或深圳 的 AI 产品 / 算法 / 临床应用岗。

#### ⭐⭐ Plan B: 字节小荷AI医生 / 阿里通义医疗 (PM path)

**为什么 Plan B:** 如果主线 A/B/C 都不顺，PM path 是 fallback。你的 PhD + 双语 + 医学背景在中国 LLM 大厂医疗 vertical 是教科书匹配。但天花板偏 PM 路径（不是技术）。

**Entry path:** 北京/杭州的医疗产品经理 / 临床产品经理岗。

#### ⭐⭐ Plan C: 腾讯觅影 AI 影像 PM (PM path + S5 imaging)

**为什么 Plan C:** S5 影像段 + 腾讯生态，profile 双重对口。比 Plan B 更对你的影像背景。

#### Plan D: 数坤 / 推想 / 深睿 国内研发岗 (保底)

**慎选:** 红旗 1-2/6 (估值倒挂 + 长期没新轮)。但确实有岗位 + 北京家族业务直接同城。仅作 backup。

## 5. Stage 3 (36-40+ 岁)：选择

完成 Stage 1-2 大约 8 年后，你应该有：
- $300-500 万 RMB 累计税后净收入（含股票）
- 联影/MGI/GPS 内部高级职位 (Senior Director or VP)
- 中国医疗设备出海行业的 network 和 reputation
- 家族业务 (Hongyu) 仍在 + 投资仓位独立复利

### 4 个 Stage 3 options（按你 36 岁时 inputs 决定）

#### Option 1: 继续 ladder up，目标 CTO / VP

- 联影 VP of International R&D
- MGI Americas CEO
- GPS 中国 R&D VP

**典型年包 200-400 万 RMB。** 不创业，享受 senior leader 的资源 + 稳定性。

#### Option 2: 创业 — 中国医疗设备 + AI 出海

- 创办自己的 medical imaging AI 公司，专注新兴市场（SEA / MEA / LATAM）
- 可能 partner 联影/MGI 做 GTM
- 利用 Hongyu 现金流 + 你已 accumulated 的投资仓位作为初始 funding

**风险/回报:** 创业 5 年成功率 ~20%，但成功的话 5-10 年内可能 IPO + $50-200M+ 个人净资产

#### Option 3: 独立投资 / 全职管钱

- 你的投资仓位（BTC + 股权 + 现金）在 36 岁应该已经 $500K-1M USD 级别
- 加上未来 5 年职业收入 + 家族业务现金流，可能在 40 岁达到 $2-3M
- 全职做段永平式价值投资，每年 alpha 10-15%

**前置条件:** 投资能力已被 10 年验证 + 你的"高分析能力、慢启动"性格适合

#### Option 4: 升级家族业务

- 把 Hongyu 从 兼职 → 主业，做"水务 + AI + 出海"的升级版（M1500 替代 OTT Pluvio² 等）
- 利用你过去 10 年在 AI / 影像 / 出海 学到的所有 know-how 反哺 Hongyu
- 可能融资 + 招团队

**前置条件:** 父辈愿意 hand-off + Hongyu 业务规模允许全职投入

## 7. 这份 map 的 6 个核心信念（来自 Phase 1-3 数据）

1. **"中国医疗设备 + AI 出海"是 10-15 年级别 trend，不是 3 年风口。** β 信号强（联影/MGI/晶泰/Insilico 都已走通部分 path），但前 5 年薪资 ramp 比同期 AI 大厂同学慢 30-50%。你需要心理上接受这一点。

2. **S5 imaging AI 是国际化最高段。** 10+ 国家头部并存，不是 zero-sum 中美对决。Nick 出海叙事在 S5 落地最自然。

3. **S3 precision medicine 商业化最成立。** 美国 Tempus $1.3B 营收 vs S1 段总和 $600M。MGI 是中国 S3 最强出海者，是主线 B 应该选 MGI 而不是燃石/世和的原因。

4. **AI 是 enabling 不是 substance 才是健康模式。** HeartFlow / Tempus / Hims & Hers (公开市场低 multiple) 是赢家范式；Forward / Pear / Babylon / Olive (烧光 $1.6B+) 是输家范式。Nick 评估任何公司用这条 lens。

5. **PM 路径不是 fallback，是平行选择。** 第一轮 elevator map 把 "Nick 不擅长 leetcode" 误等同于 "不该进 S2 段"，违反了 "不靠成功率思考" 原则。PM path 让中国 LLM 大厂医疗 vertical 重新进入候选集。

6. **Insilico Medicine 是 Nick 出海叙事唯一已实证范本。** 中港双总部 + HK IPO + Phase 2 阳性 + multi-shore。其他公司都是 "正在走"，Insilico 是 "走通了"。Phase 3 后这一点更强：晶泰 + Insilico 双 HK IPO 形成模板。

## 10. 元教训（这份研究学到的关于"做研究"本身的事）

1. **写 plan md 后停下来确认** — 比一口气搜 8 个 block 更高效 + 可审计
2. **资格筛不能 gate on "Nick 难进"** — 那是 entry_paths 字段要描述的，不是筛选条件
3. **schema 要分 facts (CSV) vs analysis (md)** — 让 fresh agent 6 个月后能 cold-start
4. **数字必须带日期** — Nabla 估值被 aifundingtracker 错报 $5.3B vs StatNews 实际 $180M 是教训
5. **dead 公司要纳入样本** — Olive / Forward / Pear / Babylon 共 $1.95B 烧光 是 calibration 红旗清单的来源
6. **不靠成功率思考** — 用"路径 + effort"替代"yes/no/概率"，让候选集保持开放
