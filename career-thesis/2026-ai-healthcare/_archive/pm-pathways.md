# PM 路径专题分析

## 1. 框架修正：entry path 不是 1 维是 2 维

初版分析的盲点：把 entry path 简化成"工程师 path 一个通道"，导致整段 S2 (Medical LLM) 被错误剔除。修正后的框架：

|  | engineer path | PM path | clinical app path | BD path |
|---|---|---|---|---|
| **门槛** | leetcode + system design | 产品 sense + 利益相关方 + domain | 临床经验 + 沟通 | 客户关系 + 销售技巧 |
| **PhD 价值** | 几乎不计入 | **高**（医疗 PM 优先 MD/PhD）| 高 | 中 |
| **双语价值** | 中 | **极高**（出海岗）| 中 | 高 |
| **35+ 风险** | 高 | **低**（domain expertise 越老越值钱）| 低 | 低 |
| **跨行业流动性** | 低 | **高** | 中 | 高 |

**Nick 的具体含义:** 工程师 path 的 binding constraint 是 leetcode；PM path 的 binding constraint 是"是否有 1 段相关产品经验"——这是可补的（联影/MGI 海外 BD 1-2 年即可补）。

## 2. 中国 LLM 大厂医疗 vertical PM 岗位地图

这是 Nick 应该重新认真评估的赛道。2024-2026 是中国 LLM 大厂全面建医疗 vertical 的窗口期，PM 招聘活跃。

### 字节跳动（豆包 + 小荷医疗）
- **PM 岗位类型:**
  - 医疗产品经理（豆包医疗 vertical）
  - 临床产品经理（针对医生端 workflow）
  - 健康内容产品经理（C 端患者侧）
  - **小荷医疗 BD/产品总监**（小荷已独立运营，有 PM + BD 团队）
- **Profile match:** 字节医疗 vertical 直接对应你的 PhD + 双语 + 海外，是教科书匹配
- **薪资:** P6 ~ 60-90 万 / P7 ~ 100-150 万 / P8+ 200 万+（含股票）
- **Entry path:** LinkedIn / 脉脉 / 字节官网；最有效是字节内部员工内推（你 Nebius 准备时和字节有过联系？）
- **风险:** 字节医疗 vertical 内部组织变动频繁（小荷曾经独立又被并回），需要确认目标部门稳定性

### 阿里巴巴（通义 + 阿里健康）
- **PM 岗位类型:**
  - 阿里健康 AI 产品经理（处方药 + 互联网医院 + 医保支付）
  - 通义医疗大模型应用 PM
  - 阿里云 + 行业大模型 - 医疗方向产品经理
  - **阿里健康海外业务 PM**（阿里全球化战略一部分）
- **Profile match:** 阿里更看重商业 sense + 流量运营，PhD 帮助但不是核心。**海外业务岗** 更对 Nick
- **薪资:** P6 ~ 50-80 万 / P7 ~ 90-140 万
- **Entry path:** 阿里官方招聘 + 脉脉

### 腾讯（医典 + 觅影 + 微信医疗）
- **PM 岗位类型:**
  - 腾讯医典内容产品经理
  - 腾讯觅影 AI 影像产品经理（**这个对你最对口 — 在 S5 影像 + 腾讯大平台**）
  - 微信医疗 H5 / 小程序 PM
- **特别说明:** 腾讯觅影是腾讯的 AI 医疗影像团队，做的事情和数坤/推想类似，但有腾讯生态资源。**这是 S5 imaging 但属于互联网大厂内部产品**，profile match 极高
- **薪资:** T9 ~ 50-80 万 / T10 ~ 90-150 万 / T11 ~ 150-250 万

### DeepSeek / 月之暗面 / 智谱 AI / MiniMax / 阶跃星辰
- **PM 岗位类型:** 大模型应用 vertical PM（医疗方向）
- **特别说明:** 这一档是 2024 年新冒出来的 LLM 公司，多数还没单独建立医疗 vertical 但都在招"行业应用 PM"。**风险高但上升空间也最大**。
- **薪资:** 大模型 startup 普遍现金少股票多，整体包 60-150 万但波动大
- **Entry path:** 主要靠 LinkedIn outreach 或这些公司投资方 (HongShan, Capital Today 等) 的网络

### 京东健康 / 平安好医生 / 微医
- **PM 岗位类型:** 互联网医院 PM、AI 医生助手 PM、健康管理 PM
- **Profile match:** 中等。这是 consumer 医疗，PhD 不是 differentiator
- **薪资:** 比 LLM 大厂略低，P6 40-70 万

## 3. 美国 PM 路径（受 visa 限制）

虽然你有 NIW 绿卡进度，但**美国医疗 AI 公司的 PM 岗在中国候选人池里没有 China 招聘那么对你倾斜**：

### OpenEvidence
- **PM 类型:** Clinical Product Manager (要求 MD 或 medical PhD)，Pharma Partnership PM
- **Profile match:** 你的 PhD 加分，但他们更倾向 Harvard/Stanford MD 校友圈
- **薪资:** Senior PM ~ $200-300k base + equity，total ~$400-500k

### Hippocratic AI
- **PM 类型:** Clinical Application PM (定义 agentic 工作流)，Health System BD
- **Profile match:** 类似 OpenEvidence

### Tempus AI / Guardant / Caris
- **PM 类型:** Oncology Clinical PM、Pharma Data Partnership PM
- **Profile match:** 你的 PhD + cancer adjacent (Roswell Park) 直接对口
- **薪资:** Senior PM $200-280k base

### Subtle Medical / Aidoc / Cleerly
- **PM 类型:** Imaging Product Manager
- **Profile match:** **最对你的影像 PhD** ；Subtle Medical 特别因为双总部，中国 dev team + 美国 PM team
- **薪资:** Senior PM $180-250k base

**美国 PM 路径的 binding constraint:** 不是技能（你具备），是签证 + 网络。NIW 绿卡解决长期身份但不解决"美国 PM 招聘还是更倾向校友/本土候选人"这个问题。

## 4. 中国 PM vs 美国 PM 的 ROI 对比

| 维度 | 中国 LLM 大厂医疗 PM | 美国医疗 AI PM |
|------|---------------------|---------------|
| Entry 摩擦 | 低（招聘活跃 + PhD 加分 + 双语稀缺）| 中-高（visa + 网络） |
| 5 年 total comp | 100-200 万 RMB | $400-600k USD（PPP 调整后 ~ 200-300 万 RMB） |
| 35+ 风险 | 低 | 低（PM 比 engineer 抗龄） |
| 家族业务兼容 | **极高**（地理一致）| 低 |
| 出海叙事一致 | **极高**（中国大厂 + 出海岗一致）| 中（出海概念是反向） |
| 5 年后职业 option | 跳到联影/MGI 出海 BD director / 自己创业 | 跳到 startup C-level / 创业 |

**中性判断:** 对 Nick，**中国 LLM 大厂医疗 PM 是 PM 路径最优解**——薪酬接近美国 PPP-adjusted、entry 摩擦低、和你已经收敛的"中国 + 出海"主线完全 align。

## 6. PM 路径具体准备清单（如果 Nick 决定走这条路）

### 短期（5 月答辩后 2-3 个月）
1. **重写简历的 PM 版本**：突出 HydroSense (B2G + 多国部署) 实业经验、PA imaging 临床转化、SGLang 开源协作
2. **LinkedIn 升级**: title 改成 "Healthcare AI Product Strategist" 或类似（不是 "PhD Candidate"）
3. **拉名单**: 字节豆包医疗 + 阿里健康 + 腾讯觅影 + DeepSeek + 阶跃星辰的产品负责人，识别 Cornell/UKY/Buffalo 校友
4. **冷邮件 / 脉脉私信**: 不是求职，是"我做 X，你们做 Y，想交流"
5. **PM 面试准备书单（中文）**: 《俞军产品方法论》《启示录》《精益创业》

### 中期（6-12 个月）
6. **拿 1 段相关 PM 经验**: HydroSense 内部正式定义为"product lead"，写出 PRD / 路线图，作为简历背书
7. **写技术博客 / 知乎专栏**: 选 2-3 个医疗 AI 话题深度写（如"为什么 OpenEvidence 是 healthcare AI 史上最快增长"），建立 thought leadership
8. **目标公司面试**: 至少投 3-5 家中国 LLM 大厂的医疗 PM 岗位
9. **同时跑工程师 path**: 联影智能 / MGI 也投，**两条路径并行 8-12 周，看 offer 落地情况再决定**

### 红旗清单应用（PM 视角）
评估具体 PM offer 时，问：
1. 这个产品的付费方明确吗？（Olive 模式：付费方不明确 → 死）
2. 这个产品有 1 个北极星指标吗？（多模块平台无明确 NSM → 危险）
3. 这个团队过去 12 月发了多少版本？（迭代节奏 < 月度 → 危险）
4. 我能从这个 PM 岗位 3 年后跳去哪里？（lateral mobility 是 PM 核心价值）
5. 老板是 MD/PhD 还是 MBA/工程师？（PM 老板背景决定你能学到什么）
